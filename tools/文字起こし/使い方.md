# 文字起こしの使い方

## このフォルダのツール（YouTube用）

**`fetch_transcript.py`** に YouTube のURL（または動画ID）を渡すと、動画内の字幕をまとめて取得できます。

- 手動字幕・自動生成字幕の両方に対応
- APIキー不要（[youtube-transcript-api](https://github.com/jdepoix/youtube-transcript-api) 使用）
- 詳細は `README.md` を参照

```bash
pip install -r requirements.txt
python fetch_transcript.py "https://www.youtube.com/watch?v=xxxxx" -o 書き起こし.txt
```

## 運用の流れ

1. 動画のYouTube URLを用意する
2. 上記ツールで書き起こしを取得し、`-o` で保存
3. 結果を **該当チャンネルの** 決めたフォルダに保存する（例: 忍び経済学で競合動画なら `channels/忍び経済学/08_競合文字起こし/`）
4. 必要なら競合分析・自チャン分析・台本執筆の材料として参照する

## その他の方法（例）

- YouTube標準の「字幕」から書き起こしをコピー
- 外部サービス（Googleドキュメントの音声入力など）
- 字幕がない動画は Python + Whisper など別ツールを検討

APIキーや認証情報を使う場合は `.env` に書き、リポジトリには含めないでください（ルートの `.gitignore` で除外済み）。
