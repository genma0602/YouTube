# YouTube運用プロジェクト

チャンネル運営単位で、台本分析・執筆・プロンプト・文字起こしを一括管理するための構成です。

## フォルダ構成

```
01_YouTube/
├── README.md                 # このファイル
├── tools/                    # 全チャンネル共通ツール
│   └── 文字起こし/           # 文字起こしツール・設定
├── prompts-shared/           # 共通で使うプロンプト雛形
└── channels/                 # チャンネル別フォルダ（1チャンネル = 1フォルダ）
    └── [チャンネル名]/
        ├── 01_競合台本分析/
        ├── 02_自チャン台本分析/
        ├── 03_台本執筆/
        ├── 04_プロンプト/
        └── 05_文字起こし/
```

## 使い方

1. **新チャンネルを追加する**  
   `channels/_template/` をコピーし、`channels/あなたのチャンネル名/` のようにリネームする。

2. **競合・自チャン分析**  
   該当チャンネルフォルダ内の `01_競合台本分析` と `02_自チャン台本分析` に台本・メモを保存する。

3. **台本執筆**  
   `03_台本執筆` に本番用・下書き用台本を保存する。

4. **プロンプト**  
   チャンネル固有のプロンプトは `04_プロンプト`、共通の雛形は `prompts-shared/` を使う。

5. **文字起こし**  
   共通ツールは `tools/文字起こし/`。チャンネル別の書き起こしデータは各チャンネルの `05_文字起こし/` に保存する。

## 運用のコツ

- 動画タイトルや日付をファイル名に含めるとあとから検索しやすい（例: `20250213_〇〇の話.md`）。
- 競合分析は「チャンネル名_動画タイトル.md」のように整理すると比較しやすい。
