# 共通プロンプト

複数チャンネルで使い回すプロンプトの雛形を置くフォルダです。

## 一覧

| ファイル | 用途 | 深さ |
|----------|------|------|
| [台本分析_伸びる理由・感情曲線・構成.md](台本分析_伸びる理由・感情曲線・構成.md) | 伸びている台本の理由・導入・感情曲線・構成を分析する | 簡易版 |
| [台本深掘り分析_完全版.md](台本深掘り分析_完全版.md) | 一文単位で心理テクニック・感情曲線・語り手構造・数字戦略・対比・リズムまで網羅的に分析する | **完全版** |
| [新企画プロット作成_競合分析ベース.md](新企画プロット作成_競合分析ベース.md) | 競合の分析結果と文字起こしから、自チャン用の新企画の情報収集項目とプロットを作成する | — |

このほか、タイトル・説明文生成用など、共通で使うプロンプトを追加できます。

チャンネル固有のプロンプトは各チャンネルフォルダ内の `04_プロンプト/` に保存してください。
