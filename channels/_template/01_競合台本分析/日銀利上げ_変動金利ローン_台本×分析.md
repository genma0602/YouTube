# STEP 5：技法の深掘り（横断分析）

## 5-1. 冒頭フックの心理設計

**日常疑問→条件付き恐怖→逆説CQ→心理的リアクタンス** の4段構成。

1. 「ぐっすり眠れましたか」 — **日常の質問に見せた不穏の導入**
2. 「変動金利で…なら、ゆっくり眠れるのは今のうち」 — **ターゲットだけを狙い撃ち**
3. 「返済額が変わらない＝破産の罠」 — **常識の逆転＝CQ生成**
4. 「覚悟がある方だけ」 — **リアクタンスで離脱を防止**

→ 全9作の冒頭は「一人称キャラクターの自己紹介」から始まるものが多かった（マルコの「ちゃお」、ハンスの「ハロー」、エマの「私はエマ」）。今作は**キャラクター不在の冒頭**。語り手が「あなた」に直接語りかける**二人称攻撃型フック**。

## 5-2. 語り手の構造

|--------|------|---------|

| **佐藤さん（シミュレーションの主人公）** | 恐怖の体現 | Block 6-9 |

| **視聴者自身（「あなた」）** | 恐怖のターゲット | Block 1, 2, 4, 5, 10, 12 |

→ **前9作と最も異なる点**: 一人称キャラクターが**不在**。マルコもハンスもエマもミゲルもいない。代わりに**「あなた」が主人公**。佐藤さんは架空のモデルケースであり、語り手は「あなた」に向かって直接シミュレーションを見せている。

**語り手の切り替えパターン**:
- 解説者→佐藤さんシミュレーション→解説者→佐藤さん→…の繰り返し。
- 「あなた」への語りかけは台本全体に散在。**12回以上「あなた」が登場**。

## 6-2. 構成上の強み

**① 「変わらない＝罠」の逆説CQ**
全10作中で最もシンプルで最も強いCQ。「毎月の返済額が変わらない」は安心材料。それが「破産の罠」だと言われた瞬間、全ての住宅ローン保持者の注意が固定される。1行で完結するCQ。

**② 佐藤さんの4段階シミュレーション**
0.475%→1%→2%→3%の段階的エスカレーション。各段階で「同じ12万9000円」の内訳を分解する反復構造。**同じフォーマットの繰り返し＋数字だけが変化**する設計が、映画のタイムラプスのように恐怖を加速させる。

**③ リコースローン＋人口減少の二重封鎖**
「売ればいい」の逃げ道を2段階で潰す。Block 4（法的に逃げられない）→Block 10（市場的にも逃げられない）。視聴者が「でも〇〇すれば…」と思うたびに先回りして否定。**全ての出口を塞いでから対策を提示**する構成。

**④ 「今、そう思いませんでしたか」の心理的先読み**
視聴者の内心を正確に言い当て、それが「銀行の策略にはまっている」証拠だと指摘する。**自分の思考パターンが罠だった**と気づく恐怖は、外部の恐怖（ベネズエラの崩壊等）より遥かに内臓に刺さる。

**⑤ 「絶望する必要はない。対策はある」の着地**
Block 1-10で恐怖を極限まで高めた後、Block 12で3つの対策＋「今すぐ償還予定表を出せ」の具体的行動指示。恐怖だけでは視聴者はフリーズする。**行動に変換する出口を最後に用意**することで、動画全体が「役に立った」体験になる。

**⑥ 冒頭「眠る」→結末「夢から覚める」のリング構造**
「ぐっすり眠れましたか」→「長すぎた異常な夢」→「夢から目を覚まされようとしている」。「眠り」のメタファーが台本全体を貫通。

## 6-3. 冗長・削れる可能性

- **Block 11（ライフステージ vs 金融市場のタイミング）** はやや観念的で、Block 9-10の恐怖のピークからトーンが変わる。「異常な夢」のメタファーは美しいが、サスペンス映画の勢いをやや止める可能性がある。ただしリング構造（「眠る」の回収）を成立させるためには必要。
- Block 12の対策3つは**実用的だが情報量は薄い**。固定金利への借り換えの具体的な数字（例：固定1.8%なら月いくら増えるが元金は確実に減る）があると説得力が増す可能性。
- 「5年ルール」の解説（Block 5）と佐藤さんのシミュレーション（Block 6-9）に一部重複あり。ただし「理論→実践」の二段構えは理解を深めるため、削ると逆に分かりにくくなる可能性が高い。
