### Block 5：「10ドルの為替マジック」

#### 台本

> ここで1つ大事な視点が出てきます。それは「日本企業は世界で稼ぎ、その結果が『円』という物差しで表示されているだけ」という事実です。例えばある日本企業を思い浮かべてみてください。アメリカやヨーロッパで商品を売り、現地通貨で利益を上げ、毎年大体同じだけのドル建ての利益を出している。仮にこの会社が毎年10ドルの純利益を安定して稼いでいるとしましょう。（中略）1ドルが100円の時、10ドルの利益は1000円です。1ドルが150円まで円安になった時、10ドルの利益は1500円になります。会社が稼いだのはどちらも同じ10ドル。変わったのは「円に換算した時の見え方」です。（中略）会社の実力も評価の物差しも何一つ変わっていません。それでも、円で表示された株価は1.5倍になります。ニュースはこう伝えるでしょう。「株価50%上昇、日経平均史上最高値」。でもその内訳を丁寧に分けてみると、「企業の本当の成長で押し上げられた部分」と、「円の価値が下がったことで高く見えているだけの部分」。この2つが混ざっているのです。

#### 分析

| 項目 | 内容 |
|------|------|
| 副題 | **「同じ10ドル。変わったのは円という物差し」— 台本の核心メカニズム** |
| 感情 | 集中↑↑ → 理解↑↑↑ → 衝撃↑↑（「1.5倍に見えるだけ」）→ 「目から鱗」↑↑↑ |
| 強度 | **最高** ↑↑↑ |
| 機能 | **台本全体の知的クライマックス**。円安と株高の関係を「10ドルの利益×為替レート」という**小学生でもわかる算数**に落とし込む。 |
| キーフレーズ | 「会社が稼いだのはどちらも同じ10ドル。変わったのは『円に換算した時の見え方』です」 |
| 技法 | **思考実験＋極限まで単純化した算数＋ニュースの「翻訳」**。10ドル→1000円 / 10ドル→1500円。この2行だけで為替と株高の関係が**完全に理解できる**。「ニュースはこう伝えるでしょう。『株価50%上昇』」とメディアの報じ方まで先回りし、「でもその内訳は…」と分解する。 |

| 「同じ10ユーロのパンが、昔は1300円に感じられた」 | 視覚＋触覚（パン） | Block 6 |

→ 五感描写は**Block 1-3（佐藤さんの日常）に集中**し、Block 4以降はほぼゼロ。前4作（特にベネズエラ）が台本全体に映像を散りばめていたのに対し、今作は**冒頭でしか映像を使わない**。以降は**論理の力だけ**で引っ張る。これは弱点でもあり、後述（6-3）で触れる。

## 5-4. 「なぜ？」の多重構造

```
大CQ：日経平均5万円なのに、なぜ自分の生活はカツカツなのか？
 ├─ 小Q1：バブルと今の株高は同じなのか？ → Block 4で回答（中身が違う）
 ├─ 小Q2：なぜ株価が上がっているのか？ → Block 5で回答（円安で見え方が変わっただけ）
 ├─ 小Q3：自分の生活にどう関係するのか？ → Block 6-7で回答（資産の有無で全く違う）
 ├─ 小Q4：なぜ企業の利益が給料に反映されないのか？ → Block 9で回答（循環の断絶）
 └─ 小Q5：ではどうすればいいのか？ → Block 11で部分回答（立ち止まって考えろ）
```

→ 小Q5つ。前4作の6-8個より少ない。これは台本が**1つの大きな疑問を段階的に深掘りする構造**だから。疑問を横に広げるのではなく、**縦に掘り下げる**。

## 5-5. 対比構造の分類

| 対比の種類 | 具体例 | 効果 |
|-----------|--------|------|
| **人物の対比** | 佐藤さん（給料のみ）vs 田中さん（投資あり） | 二極化の人物化 |
| **時間軸の対比** | バブル期の日経平均 vs 現在の日経平均 | 「同じ名前の別物」 |

| **構造の対比** | 昔の循環（利益→給料→消費）vs 今の断絶（利益→株主還元） | 「なぜ波及しないか」の核心 |

| **態度の対比** | ニュースをそのまま受け取る vs 立ち止まって考える | 結論の核心 |

→ 対比は**8つ**。特に**「暗黙の対比」（日経5万 vs 残り5万）**は明示されていないが、視聴者の無意識に刺さる設計。

## 5-7. 文のリズムとペース設計

**代表的な長短リズム**：

**Block 2（スーパーのシーン）**：
```
長文（スーパーの状況）→ 「牛乳、卵、食パン、冷凍食品を少し」(17字) → 長文（レジで）→ 「あれ、またちょっと高くなってない？」(14字)
```
→ 日常品の列挙と心内語が**短いアクセント**として挟まる。

**Block 5（為替マジック）**：
```
長文（仮定の説明）→ 「10ドルの利益は1000円です」(12字) → 「10ドルの利益は1500円になります」(14字) → 「会社が稼いだのはどちらも同じ10ドル」(16字)
```
→ **同じ構文の反復**（10ドル＝Xの繰り返し）がリズムを作る。反復のあとに「同じ10ドル」で締めることで、**変わったのは為替だけ**というメッセージが刷り込まれる。

**最も短い文（10字以下）**：

| 文 | 文字数 | 機能 |
|----|--------|------|
| 「今の主役は違います」 | 9 | **転換シグナル** |
| 「でも今は違います」 | 8 | **転換シグナル**（循環の断絶） |
| 「正解は1つではありません」 | 10 | **宣言** |

→ 短文が少ない。前4作と比べて**全体的に文が長い**。これはエッセイ型の特徴であり、「読み聞かせ」のリズムを持つ。

## 6-2. 構成上の強み

**① 「普通の朝」から始める革新性**
前4作が全て「衝撃」で始めたのに対し、今作は**「誰もが経験する平凡な朝」**で始める。差別化としても、共感の深さとしても非常に有効。「異常事態」ではなく「日常の違和感」をフックにできるのは、テーマが視聴者の生活に直結しているからこそ。

**② 「10ドルの算数」で経済を説明しきる知的設計**
円安と株高の関係を「10ドル＝1000円 vs 1500円」の1例だけで説明する。LDI戦略（トラスショック）や87地域分析（オランダ）が「データの量」で説得したのに対し、今作は**「算数の単純さ」で説得**。最も再現しやすく、最も記憶に残る教育手法。

**③ 「日経5万円 vs 残り5万円」の暗黙の対比**
明示されていないが、視聴者は「日経平均5万円」と「手取りから残る5万円」が同じ「5万円」であることに無意識に気づく。この**暗黙の皮肉**が台本全体のテーマ（マクロ指標と個人の生活の乖離）を数字1つで体現。

**④ 「影」のメタファー**
「日経平均5万円は、日本企業の強さと、日本円の弱さと、世界のマネーゲームが混ざり合ってできた影」。長い分析を**1語のメタファー**に圧縮する能力。SNSで引用されるフレーズ力がある。

**⑤ 押しつけない結論の知的誠実さ**
「投資しろ」でも「危ない」でもなく「立ち止まって考えろ」。3つの選択肢を並列し、どれも否定しない。前4作が「教訓」で締めたのに対し、今作は**「態度」の提案**で締める。知的な視聴者の信頼を獲得する設計。

**⑥ 佐藤×田中のミラーキャラクター**
「同じ会社・同じ給料」で変数を**投資の有無だけ**に限定する対照実験。2人の対比が台本のCQ（なぜ景色が違うのか）を人物化し、視聴者に「自分はどっちだ？」と問いかける。

## 6-3. 冗長・削れる可能性

- **Block 4以降の五感描写の欠如**: 佐藤さんの日常シーンが冒頭で終わり、以降は解説モードが長く続く。途中で**もう1つ佐藤さんの日常シーン**（例：「佐藤さんはその夜、証券口座の開設画面を開いて、そっと閉じた」等）を挟むと、感情のアンカーが強化される。
- **Block 5の為替計算がやや長い**: 10ドルの例→株価15年分の例→50%上昇の例、と3段階で説明しているが、**最初の10ドルの例だけで充分伝わる**可能性。2段目（株価計算）は削っても本筋に影響しない。
- **結末の重複**: 文字起こしの後半に同じ内容が重複している（「東京の株価を動かしているお金が…」以降が2回出現）。編集ミスと思われる。
- **田中さんの再登場がない**: 冒頭で登場した田中さんが、結末で再登場すれば**リング構造**が完成する。「佐藤さんはあの朝から、少し違う目でニュースを見るようになった」のような一文で締めると余韻が深まる。
