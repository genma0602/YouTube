### Block 3：「アービトラージ — なぜ物価は同じなのか」

#### 台本

> まずマルコが感じた疑問から片付けよう。なぜイタリアとドイツで給料が倍も違うのに、スーパーの物価はほとんど同じなのか。これ、おかしいと思わないかい。答えは自由市場で自然に作用する「アービトラージ」という仕組みにある。日本語で言えば裁定取引というんだ。（中略）例えばイタリアでパスタが1ユーロ、ドイツで3ユーロだとしよう。すると賢い業者がイタリアで買い占めて、ドイツで2ユーロで売る。結果、どこにいても値段は均等化していく。これを「一物一価の法則」と言うよ。

#### 分析

| 項目 | 内容 |
|------|------|
| 副題 | **「パスタ1ユーロの裁定取引」— 経済学をスーパーで説明する** |

### Block 4：「税負担47.9% — 手取りは半分」

#### 台本

> じゃあドイツ人は得じゃないか。給料高くて物は同じ値段なんだから。そう思うだろ。僕もそう思ってた。でも現実は違った。僕たちの財布を圧迫しているのは物じゃない。サービスと住居費だ。美容代、修理代、家賃。これらは輸入できない。ドイツの高い人件費がそのまま反映される。外食すればランチ25ユーロ。トイレの修理を頼めば、出張費と作業費で軽く100ユーロを超える。そして最大の問題がこれだ。ドイツの税負担率47.9%。OECD38カ国中ワースト2。僕の額面は4500ユーロ。でも手取りは2700ユーロ。半分近く。そこから家賃と光熱費を払ったら何が残る。高給のドイツ人なんて、僕から言わせれば幻想だよ。税引き前の数字に騙されてるだけだ。

## 5-4. 「なぜ？」の多重構造

```
大CQ：ドイツは本当に勝ち組なのか？ なぜ国を出ていくのか？
 ├─ 小Q1：なぜ物価が同じなのに給料が違う？ → Block 3で回答（アービトラージ）
 ├─ 小Q2：じゃあドイツ人は得じゃないのか？ → Block 4で回答（税で半分消える＋サービス高い）
 ├─ 小Q3：なぜ家が見つからないのか？ → Block 5で回答（人口急増＋供給不足）
 ├─ 小Q4：なぜ社会が分断しているのか？ → Block 6-7で回答（急激な変化＋タブー＋システムの限界）
 ├─ 小Q5：なぜ工場が逃げ出しているのか？ → Block 8で回答（エネルギーコスト）
 └─ 小Q6：なぜ人が出ていくのか？ → Block 10で回答（6つの問題の総和＝同僚の台詞）
```

→ 小Q6つ。各小Qが**前のQの上に積み上がる**エスカレーション構造。Q6（なぜ出ていくか）の回答はQ1-5の全ての**総和**。

## 5-7. 文のリズムとペース設計

**Block 1（数字マシンガン）のリズム**：
```
20字 → 17字 → 25字 → 14字 → 13字 → 10字 → 5字 → 11字
```
→ 全体的に短文。最短「冗談じゃない」（5字）が感情の爆発点。**ラップ的なBPMの高さ**。

**Block 10（同僚の台詞）のリズム**：
```
短文4連（税。家。分断。産業。）→「こんな国に誰が残りたい」→「答えは簡単だ」→「残らない」(4字)
```
→ 「残らない」の4字が台本中最短級。Block 1の「冗談じゃない」（5字）と対をなす。両方とも**怒りの爆発**。

**最も短い文（10字以下）**：

| 文 | 文字数 | 機能 |
|----|--------|------|
| 「冗談じゃない」 | 5 | **怒りの爆発** |
| 「残らない」 | 4 | **断言** |
| 「戦争だ」 | 3 | **メタファーの衝撃** |
| 「でも現実は違った」 | 8 | **ホープ＆クラッシュ** |
| 「これを空洞化と言う」 | 9 | **概念の命名** |

→ 短文が**全6作中最多**。ハンスの口語的な語り口が短文を生む。前作群（ナレーター的語り手）より**テンポが速い**。

## 6-2. 構成上の強み

**① 数字マシンガンの破壊力**
冒頭8文で「勝ち組ドイツ」の幻想を完全に粉砕する速度。前5作が「静かに引き込む」タイプだったのに対し、**「叩きつけて引き込む」**。テーマが「裏側暴露」の場合に最適。

**② 6層エスカレーションの構造**
税→住宅→社会分断→エネルギー→産業空洞化→頭脳流出。1層ごとに「まだ続くのか」の驚きが生まれ、全体が**崩壊のドミノ**として機能する。各層が独立しているようで前の層を前提にしており、カットすると全体が崩れる設計。

**③ 「マルコ↔ハンス」の対称構造**
EU経済の根本矛盾を2人の人物の相互憧れに圧縮する技法。論文のように語れば100ページかかる内容を**3文で詩的に表現**。SNSでの引用力が最も高いフレーズ。

**④ 同僚の送別会の台詞**
「眉間にシワを寄せて高給をもらうより、人間らしい生活を選ぶよ」。台本全体の6層のデータを**1人の人間の1つの台詞に凝縮**。データは忘れてもこの台詞は忘れない。

**⑤ 「普通に暮らす夢が高価になりすぎた」の普遍性**
ドイツの話をしているのに、日本・韓国・イギリス・アメリカ、どの先進国の視聴者にも刺さる結論。**国を限定しない普遍的フレーズ**で締めることで、動画の共有可能性が最大化。

**⑥ フロントキャラクター連携によるシリーズ化**
マルコ（イタリア）→ハンス（ドイツ）→次は？ 各国に語り手キャラクターがいて、彼らが互いに言及し合う。**MCU（マーベル）のクロスオーバー型**の動画設計。

## 6-3. 冗長・削れる可能性

- Block 3（アービトラージ解説）はやや教科書的。パスタの例は秀逸だが、「一物一価の法則」という用語紹介は**削ってもCQへの到達に影響しない**。ただしマルコの疑問の回答として配置されているためカットしにくい。
- Block 7（ウクライナ難民のバランス）は知的誠実さの担保に必須だが、Block 6（社会分断）との**境界がやや曖昧**。2つを統合しても良い。
- 台本全体を通して**数字が多い**（20個以上）。特にBlock 8（エネルギー+空洞化）は数字が密集しており、映像（テロップ/グラフ）なしでは処理が追いつかない可能性。
