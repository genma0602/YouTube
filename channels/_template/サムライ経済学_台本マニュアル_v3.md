# ═══ 第一部：3つの鉄則 ═══

台本の全てはこの3つに集約される。迷ったらここに戻る。

---

### 鉄則1：大きな数字は「1人の生活」に置き換える

何兆円、何百万人は伝わらない。「1人の人間の1日の財布」に変換して初めて伝わる。

```
❌「インフレ率130万%」
✅「1ヶ月の給料で卵12個しか買えない」

❌「移民受け入れの純コスト4000億ユーロ」
✅「60年かけて掘った天然ガスの総収入と同じ額が、25年で消えた」

❌「税負担率47.9%」
✅「額面4500ユーロ。手取り2700ユーロ。半分近く持っていかれる」
```

**変換の3段階**:
① まず数字そのものを出す → ② 身近なものと比較する → ③ 映像が浮かぶ表現にする

---

### 鉄則2：難しい言葉は同じ文で言い直す

「今のどういう意味？」と思った瞬間に離脱する。

```
❌「アービトラージが発生し…」
✅「安い国で買って高い国で売る仕組み、つまりアービトラージが…」

❌「リコースローンなので…」
✅「家を売っても借金が残る仕組み、リコースローンと言うんだけど…」
```

---

### 鉄則3：結論を押しつけず、問いを返す

「こうすべきだ」ではなく「あなたはどう思いますか？」で終わる。
視聴者に考えさせることで、コメント・共有が生まれる。

```
❌「だから投資をすべきです」
✅「1つの数字に自分の世界の見方を全部預けない。あなたはどんな物語として受け取っていますか？」

❌「移民政策は失敗だ」
✅「これは移民の責任じゃない。政策を作った私たちの責任です。…あなたの国はどうですか？」
```

# ═══ 第二部：ナレーターと主人公 ═══

台本には**ナレーター**と**主人公**の2つの声がある。
この2つを混同すると台本が壊れる。

---

## ナレーター（＝チャンネルの声）

**口調**：丁寧だが堅すぎない。「です・ます」調。

| ナレーターがやること | 具体例 |
|---|---|
| 制度・背景の解説 | 「イギリスではVATと呼ばれる付加価値税があります」 |
| 数字やデータの提示 | 「IFSの調査によると、恩恵の差は約3倍です」 |
| 数字の三段翻訳 | 「4000億ユーロ。日本円で約74兆円。天然ガス60年分と同じ」 |
| 話題の切り替え・ゲート宣言 | 「ここからは、誰も語りたがらない話に進みます」 |
| 日本との比較（日本ブリッジ） | 「日本でも2019年に軽減税率が導入されました」 |
| 予防線を張る | 「誤解しないでほしい。移民が悪いと言いたいわけじゃない」 |
| 共感の一言（交代時のみ） | 「同じ時代を生きる人間として胸が締めつけられます」 |

**ナレーターがやってはいけないこと**：
- 「僕は」「俺は」と一人称で体験を語る → 主人公の仕事
- 感情的に叫ぶ、怒る → 主人公の仕事
- 長い独り言を続ける → 主人公の物語を止めてしまう

---

## 主人公（＝その国に生きる1人の人間）

**口調**：友達に話しかけるような口語。「〜だ」「〜なんだ」「〜だよ」調。
**設定に必ず含めるもの**：名前・年齢・職業・**個人的な痛み**

| 主人公がやること | 具体例 |
|---|---|
| 自分の体験を語る | 「先月、病院に電話した。3週間待ちだと言われた」 |
| 自分の感情を出す | 「冗談じゃない」「怖かった」「恥ずかしいけど」 |
| 自分の数字を晒す | 「額面4500。手取り2700」「家賃2300。残り300ポンド」 |
| 身近な人のエピソードを話す | 「隣人は70代で、毛布を重ねて寝ていた」 |
| 冒頭で視聴者を引き込む | 「僕は32歳なのにまだ親と暮らしている。おかしいと思う？」 |
| 結末で余韻を残す | 「来月33歳になる。そろそろ実家を出たい」 |
| 過去の自分を裁く | 「地上の楽園だと思いました。地獄への入り口とも知らずに」 |

**主人公がやってはいけないこと**：
- 制度や政策を教科書のように説明する → ナレーターの仕事
- 統計データを淡々と読み上げる → ナレーターの仕事
- 途中で急に「です・ます」調になる → 口調が壊れる

---

## ★重要：ナレーターと主人公の「交代」の3つの型

16本の台本分析から、交代パターンは3つの型に分類できる。

---

### 型A：主人公主導型（主人公8割 / ナレーション2割）

**該当台本**: ドイツ（ハンス）、イタリア（マルコ）、ロンドン（ジョン）、トラスショック（ジョン）

主人公がほぼ全編を語る。主人公自身が構造をある程度理解しており、**体験しながら解説もできる人物**として設計されている。

```
ハンス →「ドイツの税負担率47.9%。僕の額面は4500。手取りは2700。半分近く」
ハンス →「家探しは戦争だ。20件内見して全部断られた友人がいる」
ハンス →「もうお金の問題だけならまだマシなんだ。本当に大変なのは…」
（ナレーションはほぼ登場しない）
```

**使いどころ**：主人公の知性が高く、自分の言葉で構造を語れる場合。「1人の証言者がすべてを語る」ドキュメンタリー感が最も強い型。

---

### 型B：交代型（主人公5割 / ナレーション5割）★最も汎用性が高い

**該当台本**: ベネズエラ3部作、オランダ前編・後編

主人公とナレーションが**明確に交代**する。交代のタイミングは「感情→構造」の切り替え地点。

```
ミゲル →「スーツを着てゴミ捨て場に向かう。7歳の娘が隣にいる」（感情）
    ↓ 交代
ナレーター →「胸が締めつけられます。なぜこうなったのか」（共感＋構造へ橋渡し）
ナレーター →「石油以外何もしなかった。オランダ病と呼ばれる構造です」（構造解説）
    ↓ 交代
ミゲル →「まさに地上の楽園だと思いました。地獄への入り口とも知らずに」（感情）
```

**交代のルール**：
- **主人公→ナレーター**：感情の嵐のあと、息継ぎさせてから構造を入れる
- **ナレーター→主人公**：構造を理解した視聴者に、もう一度感情で殴る
- ナレーターは交代時に必ず**共感の一言**を入れる（冷たい解説者ではなく「一緒に怒っている専門家」）
- **1.5〜2分に1回**切り替えるとテンポが良い

**使いどころ**：最も汎用性が高い。海外テーマ全般。「主人公の感情」と「ナレーターの知性」が相互に増幅し合う。

---

### 型C：ナレーション主導型（ナレーション8割 / 主人公2割）

**該当台本**: 日経平均（佐藤さん）、円安（佐藤さん+ファンドマネージャー）、変動金利（佐藤さん）

ナレーションが全体を主導し、主人公は**三人称で描写される「視聴者の分身」**。

```
ナレーター →「ある平日の朝、佐藤さんはテレビを見ていた」
ナレーター →「佐藤さんはふと思います。『景気がいいらしいのに、なんでカツカツなんだろう』」
ナレーター →「仕組みを整理しましょう。日経平均の中身はバブル期とは別物です」
（佐藤さんは自分では語らない。ナレーターが描写・代弁する）
```

**使いどころ**：日本国内テーマ。視聴者自身が当事者なので、主人公は「証言者」ではなく「鏡」。

---

### ★ 型の選び方

```
Q1. テーマは海外？日本？
  → 日本 → 型C
  → 海外 → Q2へ

Q2. 主人公は構造を自分で語れる知性がある？
  → YES（ドイツのハンス、イタリアのマルコのように）→ 型A
  → NO（ベネズエラのミゲルのように、体験は語れるが構造は語れない）→ 型B
  → 迷ったら → 型B（最も安全で汎用性が高い）
```

---

### ★絶対に守ること

**口調が混ざった瞬間に、視聴者は「誰が話してるの？」と混乱して離脱する。**

```
❌「僕はマイケル。IFSの調査によると恩恵の差は約3倍です」
  → 主人公の口調（僕は）でナレーターの仕事（データ提示）をしている

✅「僕はマイケル。スーパーで働いてる。先月フードバンクに並んだ」
  → 主人公は体験だけ語る

✅「IFSの調査によると、恩恵の差は約3倍です」
  → データの提示はナレーターの口調で
```

# ═══ 第四部：4つの台本パターン ═══

全ての台本は4つのパターンのどれかに当てはまる。

---

## パターンA：物語型（人が主役）

1人の人間の体験を通して国や経済の問題を描く。
**初回・シリーズ1本目に最適。**

**流れ**：
```
①つかみ → ②主人公の登場 → ③かつての平穏 → ④崩壊の始まり
→ ⑤崩壊の仕組み（解説を挟む）→ ⑥どん底 → ⑦転機 → ⑧教訓＋日本
```

**感情曲線**：
```
感情
 高│        ③平穏
   │       ╱    ╲
 中│  ①②╱      ╲ ⑤解説
   │   ╱          ╲╱╲            ⑧教訓
 低│  ╱              ╲    ⑦転機 ╱
   │╱                 ╲  ╱    ╱
   │                    ⑥どん底
   └──────────────────────────→ 時間
```

ポイント：
- **③平穏を丁寧に描くほど、⑥どん底の落差が大きくなる**
- ⑤解説パートで一度冷静にしてから⑥に落とす（二段落ち）
- ⑧は冒頭の場面を繰り返す（リング構造）

**代表例**: ベネズエラ崩壊（ミゲル）、イラン革命

**やること**：
- 主人公に名前・年齢・職業を必ず設定する
- 「昔は良かった」パートで匂い・音・手触りを入れる
- 崩壊は一気にではなく段階的に（食べ物→医療→命）
- 結末で冒頭の場面を繰り返す（リング構造）

---

## パターンB：謎解き型（問いが主役）

「なぜ？」「誰が得した？」を解き明かす。
**シリーズ2本目に最適。**

**流れ**：
```
①つかみ（結果を叩きつける）→ ②問い → ③事実の整理
→ ④「ここからが本題」の宣言 → ⑤答えを1つずつ開封（3つ）
→ ⑥歴史の事例 → ⑦得した人/損した人 → ⑧教訓＋日本
```

**感情曲線**：
```
感情（好奇心）
 高│                        ⑤答え③
   │               ⑤答え②╱     ╲
 中│        ⑤答え①╱     ╱  ⑥歴史╲ ⑧教訓
   │  ②問い╱    ╱     ╱    ╱    ╲╱
 低│  ╱  ╲╱   ╱     ╱    ╱
   │╱   ③事実  ④宣言
   └──────────────────────────→ 時間
```

ポイント：
- **④「ここからが本題」の宣言が最重要の転換点**（「ここまでが表。ここから裏」）
- 答えは3つに番号をつける（記憶に残り、人に伝えやすい）
- 好奇心が階段状に上がっていく設計

**代表例**: オランダ移民前編（74兆円の衝撃）、ベネズエラ続編

**やること**：
- タブーに触れる時は先に予防線を張る
- 「感情ではなく数字で語る」と宣言すると信頼が生まれる
- 答えは3つに番号をつけて出す

---

## パターンC：連鎖型（時間が主役）

出来事がドミノ倒しのように連鎖していく。

**流れ**：
```
①つかみ（最も衝撃的な結果）→ ②問い＋「時間を巻き戻す」
→ ③ドミノ① → ④ドミノ② → ⑤ドミノ③ → ⑥ドミノ④（最深部）
→ ⑦「その後」 → ⑧教訓＋日本
```

**感情曲線**：
```
感情
 高│                              ⑥ドミノ④
   │                     ⑤ドミノ③╱╲
 中│            ④ドミノ②╱      ╱  ╲  ⑧教訓
   │   ③ドミノ①╱     ╱      ╱    ╲╱
 低│  ╱       ╱     ╱      ╱ ⑦その後
   │╱①②   ╱     ╱      ╱
   └──────────────────────────→ 時間
```

ポイント：
- **「わずか◯日後」を繰り返すと加速感が出る**（トラスショック：17日→3日→5日→10日→38日→45日）
- ドミノの間に息継ぎを入れる
- 右肩上がりでどんどん深刻になる

**代表例**: トラスショック（45日の崩壊）、ドイツ（6層エスカレーション）

**やること**：
- 「わずか◯日後」「まだ終わりじゃない」のリフレインで加速
- 各ドミノの間に息継ぎを入れる
- 「その後」パートで他チャンネルとの差別化

---

## パターンD：解説型（仕組みが主役）

経済の仕組みを「なるほど」の連続で理解させる。
**日本国内テーマに最適。**

**流れ**：
```
①つかみ（日常の違和感）→ ②主人公＝視聴者の分身
→ ③問題を分解（3つの原因）→ ④原因①＋日常例
→ ⑤原因②＋具体的な計算 → ⑥原因③＋過去データ
→ ⑦全体像のまとめ → ⑧「あなたの家計では？」
```

**感情曲線**：
```
感情（理解度）
 高│                    ⑥原因③
   │           ⑤原因②╱    ╲ ⑦まとめ
 中│    ④原因①╱     ╱      ╲╱╲
   │  ╱     ╱     ╱           ╲ ⑧自分事
 低│╱①②③╱     ╱
   │
   └──────────────────────────→ 時間
```

ポイント：
- **「なるほど」が階段状に積み上がる**
- 衝撃・恐怖ではなく知的好奇心がエンジン
- 主人公＝視聴者の分身（三人称で描写）

**代表例**: 日経平均（佐藤さん）、円安160円、変動金利ローン

**やること**：
- 主人公は「佐藤さん」的な普通の人にする
- 仕組みは3つ以内に分解する
- 日常の行為に置き換える（「電気をつけるたびに円が海外に出ていく」）

---

## パターンの選び方

```
Q1. このテーマの「一番面白いところ」は何か？
  → 「1人の人生」が面白い    → A：物語型
  → 「裏側・真相」が面白い   → B：謎解き型
  → 「連鎖・加速」が面白い   → C：連鎖型
  → 「仕組み・構造」が面白い → D：解説型

Q2. 続編か？
  → YES → 前回と違うパターンを選ぶ
  → NO  → Q1の答えをそのまま使う
```

シリーズの場合：第1編＝物語型 → 第2編＝謎解き型 → 第3編＝物語型（サバイバル変形）

# ═══ 第五部：導入（冒頭）の書き方 ═══

導入＝台本全体の10%。5000字の台本なら**400〜500字、8〜12文**。

---

## 3ステップで書く

### Step 1：つかみ（1〜4文）

スクロールを止める。1文あたり**6〜20字**。短いほど衝撃が大きい。

| 方法 | やり方 | 例 |
|---|---|---|
| **衝撃の数字** | 前置きゼロで最も驚く数字を出す | 「毎年25万人のドイツ人がこの国を捨てて出ていく」 |
| **日常→破壊** | 何気ない日常を並べ、最後の1文で壊す | 「スーツを着てネクタイを締めていますが、向かうのはゴミ捨て場です」 |
| **結果先行** | 理由も経緯も省いて結果だけ叩きつける | 「住宅ローンの月々の返済額が来月から9万円上がる」 |
| **自分の体験** | 主人公自身の体験で始める | 「僕は32歳なのにまだ親と暮らしている。おかしいと思う？」 |

---

### Step 2：主人公の登場（3〜4文）

```
[名前] ＋ [年齢] ＋ [職業] ＋ [個人的な痛み]（1〜2文）
→ 「なぜこの話をするのか」の理由（1〜2文）
→ 前回の動画との接続（あれば、1文だけ）
```

**主人公に必要なもの**：
- **名前**（ミゲル、エマ、ハンス、マルコ、ジョン、佐藤さん）
- **年齢と職業**（銀行員、会計士、32歳、ごく普通の市民）
- **個人的な痛み**（ゴミ漁り、親同居、家賃で手取りが消える）
- **「自分の数字」の開示**（手取り2700ユーロ、残り300ポンド）

痛みのない主人公は視聴者が感情移入する理由がない。

---

### Step 3：問い＋約束（2〜3文）

**問いの公式**：
```
[みんなが信じていること] なのに、なぜ [現実は逆] なのか？
```

良い問いの条件：
1. 視聴者が感じていたけど言葉にできなかった疑問を代弁
2. 答えを知るまで離脱できない
3. **1行（15〜25字）**で言える

```
✅「食料品0%なのに、なぜ300万人が食べられない？」
✅「毎月払っているのに、なぜ借金が減らないのか？」
✅「勝ち組のはずのドイツ人が、なぜ国を出ていく？」
```

問いの直後に**約束**（「最後まで見ればわかる」）と**スタート宣言**（「では始めましょう」）を入れる。

# ═══ 第六部：感情曲線の設計 ═══

台本の感情設計は「何を感じさせるか」ではなく「どの順番で感じさせるか」。

---

## 感情の種類と使いどころ

| 感情 | どこで使うか | 担当 |
|---|---|---|
| **衝撃** | つかみ、ゲート宣言の直後、どん底 | 主人公 |
| **共感** | 主人公の自己紹介、生活描写 | 主人公 |
| **好奇心** | 問いの提示、「3つの理由」の予告 | ナレーター |
| **なるほど** | 仕組みの解説、うまい比喩 | ナレーター |
| **恐怖** | 問題がだんだん深刻になる場面 | 主人公 |
| **怒り** | 不正・搾取が明らかになる場面 | 主人公 |
| **悲しみ** | 家族の別れ、失ったもの | 主人公 |
| **危機感** | 日本との比較 | ナレーター |
| **余韻** | 締めの直前 | 主人公 |

---

## 感情曲線を描く手順

### 手順1：ピーク（最も感情が動く瞬間）を1つ決める

台本全体で**最も強い1文**を先に決める。これが感情のピーク。

```
例：
ベネズエラ →「地上の楽園だと思いました。地獄への入り口とも知らずに」
ドイツ →「ドイツで眉間にシワを寄せて高給をもらうより、人間らしい生活を選ぶよ」
オランダ →「60年かけて掘った天然ガスと同じ額が、25年で消えた」
ロンドン →「手取り2600。家賃2300。残り300ポンド」
```

### 手順2：ピークの位置を決める

ピークは台本の**60〜75%地点**に置く（終盤すぎると教訓・日本ブリッジの時間が足りない）。

```
                                  ★ピーク
                                 ╱╲
                               ╱    ╲
                             ╱        ╲
                           ╱            ╲
┌──────────┬──────────┬──────────┬──────────┐
0%        25%        50%       75%       100%
つかみ      背景       本題      ピーク    教訓＋締め
```

### 手順3：ピークに向かう「登り坂」を設計する

ピークは突然来てはいけない。**段階的にエスカレーション**する。

パターン別の登り坂：

**物語型・連鎖型**（恐怖・悲しみを段階的に積む）：
```
食べ物がない → 日用品の争奪 → 医療崩壊 → 遺体放置
   ↑軽い恐怖                        ↑最も重い恐怖

税金で半分 → 家探しが戦争 → 社会が分断 → 産業が逃げ → 人が逃げる
   ↑経済の問題                              ↑国の存亡の問題
```

**謎解き型**（好奇心を段階的に積む）：
```
食卓が凍りつく → 研究者が追い出される → 74兆円のデータ → 同じ移民で1億円の差
   ↑タブーの予感      ↑隠蔽の証拠         ↑数字の衝撃     ↑構造の核心
```

**解説型**（理解を段階的に積む）：
```
佐藤さんの違和感 → バブルと中身が違う → 10ドルの為替マジック → 手取り22万の家計
   ↑疑問              ↑視点の転換         ↑仕組みの理解        ↑自分事化
```

### 手順4：ピーク後の「着地」を設計する

ピークの後は必ず**感情を下げてから教訓に入る**。

```
             ★ピーク
            ╱╲
          ╱    ╲
        ╱      息継ぎ（ナレーターに交代、バランスデータ）
      ╱            ╲
    ╱                教訓＋日本ブリッジ
  ╱                      ╲
つかみ                      締め（問いかけ＋余韻）
```

### 手順5：「息継ぎ」の位置を決める

**重い感情（衝撃・恐怖・悲しみ）が3パート以上連続したら、必ず息継ぎを入れる。**

息継ぎの方法：
- ナレーターに切り替える
- バランスを取るデータを入れる（「全てが悪いわけではない」）
- ちょっとしたユーモアを入れる（トラスショックの「レタス」）
- 仕組みの解説パートに切り替える

---

## 感情曲線テンプレート

台本を書く前に、以下のようなメモを作る。

```
Block 1 [つかみ]      担当：主人公  感情：衝撃↑↑
Block 2 [自己紹介+CQ] 担当：主人公  感情：共感↑ → 好奇心↑↑
Block 3 [背景]        担当：ナレ    感情：理解↑
Block 4 [ゲート宣言]  担当：ナレ    感情：好奇心↑↑↑（「ここからが本題」）
Block 5 [本題①]       担当：ナレ    感情：驚き↑↑
Block 6 [本題②]       担当：主人公  感情：恐怖↑↑↑ ★ピーク
Block 7 [息継ぎ]      担当：ナレ    感情：理解↑（構造解説で冷静に）
Block 8 [日本ブリッジ] 担当：ナレ   感情：危機感↑↑
Block 9 [締め]        担当：主人公  感情：余韻（冒頭のリフレイン）
```

---

## リフレイン（繰り返し）の設計

同じ言葉を**意味を変えながら繰り返す**ことで、台本に背骨を通す。

| 台本 | リフレイン | 回数 | 意味の変化 |
|---|---|---|---|
| ドイツ | 「冗談じゃない」 | 3回 | 怒り → 呆れ → 諦め |
| オランダ | 「誤解しないでほしい」 | 3回 | 予防線 → 再確認 → 倫理的着地 |
| トラスショック | 「わずか◯日後」 | 6回 | 加速感が増していく |
| イタリア | 親同居の告白 | 2回 | 恥ずかしさ → 構造的必然の理解 |

設計方法：
1. 台本で最も象徴的なフレーズを1つ選ぶ
2. 序盤・中盤・終盤で**最低3回**使う
3. 回を追うごとに**意味が重くなる**ように設計する

# ═══ 第七部：テクニック集 ═══

★が多いほど重要。全部使う必要はない。

---

## 文章テクニック

| テクニック | 何をするか | ★ |
|---|---|---|
| **数字を生活に変換** | 大きな数字を「1人の1日の暮らし」に置き換える | ★★★ |
| **短い文で落とす** | 長い文の後に極端に短い文（3〜10字）でガツンと落とす | ★★★ |
| **体の感覚で描く** | 匂い・音・手触り・味・温度を言葉で描く | ★★★ |
| **日本との比較** | 海外の話を「日本で同じことが起きたら」に接続する | ★★★ |
| **対比で見せる** | 数字vs数字、昔vs今、Aさんの人生vsBさんの人生 | ★★★ |
| **本人の台詞を入れる** | 「彼はこう言った」で実際の言葉を入れる | ★★☆ |

---

## 構造テクニック

| テクニック | 何をするか | ★ |
|---|---|---|
| **ゲート宣言** | 前半と後半の切れ目で短く宣言する（「ここからが本題」「ここまでが表。ここから裏」） | ★★★ |
| **番号をつけて答える** | 答えを「3つの理由」のように番号つきで出す | ★★★ |
| **リング構造** | 冒頭の言葉を結末で「意味が変わった状態で」繰り返す | ★★★ |
| **段階的エスカレーション** | 食→住→医療→命のように、問題をだんだん深刻にする | ★★★ |
| **視聴者の疑問を先回り** | 視聴者が「なぜ？」と思う前に、その疑問を言葉にして回答する | ★★★ |
| **伏線→回収** | 前半に意味深な言葉→後半で解明（「おじいちゃんの言葉」） | ★★☆ |
| **主人公/ナレーター交代** | 感情担当（主人公）と情報担当（ナレーター）を交互に切り替える | ★★☆ |
| **希望→粉砕** | 「これで大丈夫かと思ったが…」→すぐに否定して緊張感維持 | ★★☆ |

---

## 心理テクニック

| テクニック | 何をするか |
|---|---|
| **結果だけ見せて原因を隠す** | 先に衝撃的な結果を見せて「なぜ？」で引っ張る |
| **「見ない方がいい」と言う** | 「閉じた方がいい」で逆に見たくさせる |
| **答えを言わず気づかせる** | 数字を並べて視聴者に自分で計算させる |
| **「聞いてくれませんか」と巻き込む** | 「聞く」と決めたら最後まで見る心理を使う |
| **ドラマティック・アイロニー** | 視聴者は結末を知っているのに、主人公はまだ信じている構造 |

# ═══ 第八部：文のリズム ═══

## 場面ごとの文字数

| 場面 | 文字数 |
|---|---|
| つかみ（冒頭1〜3文） | **6〜20字** |
| 通常の解説 | **20〜40字** |
| 感情が爆発する瞬間 | **3〜10字** |
| 動画全体の問い | **15〜25字** |
| 教訓・結論 | **25〜40字** |

---

## リズムの鉄則

**短→短→短→長→短（ドン！）** が基本パターン。

台本全体で**10字以下の極端に短い文を5〜10個**配置する。
それぞれが「転換」「感情の爆発」「断言」のどれかとして使う。

```
例（ドイツ冒頭）：
「フォルクスワーゲン3万5000人削減」（16字）
「メルセデス4万人」（8字）
「冗談じゃない」（6字）← ★ここが爆発
「僕たちは静かに沈んでいる」（12字）

例（ベネズエラ冒頭）：
「2018年3月のある火曜日。」（12字）← ★急に短くなる
「私はまだスーツを着てネクタイを締めていますが、向かうのは銀行ではありません。」（35字）
「レストランの裏にあるゴミ捨て場です。」（16字）← ★短い文でオチ

例（ベネズエラ中盤）：
「なぜか。」（4字）← ★台本中最短。思考を止める
```

長い文（30〜40字）→ **極端に短い文（3〜16字）でオチや転換を入れる**。
映像編集の「カット」と同じ。長い文が「全景」、短い文が「寄り」。

# ═══ 第九部：結末の書き方 ═══

## 教訓

3つ以内に絞る。視聴者が人に伝えられる短さにする。

---

## 結末の4パターン

| パターン | やり方 | 例 |
|---|---|---|
| **冒頭の繰り返し＋意味の変化** | 冒頭の言葉を、意味が変わった状態で繰り返す | 「パン0%。牛乳0%。でもその裏側を、今日君は知った」 |
| **チェックリスト型** | 崩壊の条件を列挙→「あなたの国はいくつ当てはまる？」 | ベネズエラの「ポピュリズムの財源は3つしかない」 |
| **「普通の夢」型** | 普遍的な一文で閉じる | 「普通に暮らす夢が高価になりすぎた」 |
| **2人の対比型** | 2人の主人公が互いに相手の国に憧れている | イタリアのマルコとドイツのハンス |

---

## 締め（CTA）の順番

```
① 問いかけ（「あなたはどう感じましたか？」）→ コメントを誘う
② 価値の確認（「この動画が役に立ったなら」）
③ 行動指示（「高評価とチャンネル登録をお願いします」）
④ 次回への期待（「それではまた次の動画で」）
```

# ═══ 第十部：やってはいけないこと ═══

| NG | 理由 |
|---|---|
| **結論を押しつける** | 「投資すべき」「移民は悪い」は言わない。問いを返す |
| **国や民族を攻撃する** | 「問題は制度であり人ではない」を必ず入れる |
| **データなしで主張する** | 全ての主張に数字か出典をつける |
| **重い感情を3パート連続** | 息継ぎを入れないと視聴者が疲れて離脱 |
| **難しい言葉を放置** | 使ったらその場で日常の言葉に言い換える |
| **冒頭の前置きが長い** | 8文以内に問いを提示する |
| **主人公とナレーターの口調を混ぜる** | 混ざった瞬間に離脱される |
| **主人公にデータ解説をさせる** | 数字の提示・構造解説はナレーターの仕事 |
| **ナレーターに感情を叫ばせる** | 感情表現は主人公の仕事 |

```

Step 3: 主人公設計 → Reddit / Quora / X で現地の声を集める
Step 4: 仕組みの解説素材 → 学術論文・専門家の分析
Step 5: 数字を「1人の生活」に変換するメモを作る
Step 6: 日本との比較を3パターン考える
```
