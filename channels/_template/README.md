# チャンネルテンプレート

このフォルダをコピーして新チャンネル用ワークスペースを作成する。

```
_template/ をコピー → チャンネル名にリネーム → 独立ワークスペースとして開く
```

## フォルダ構成

| Folder | 用途 |
|--------|------|
| 01_channel | チャンネル概要・ペルソナ・KPI |
| 02_research | 競合分析 (competitor)・自チャン分析 (self)・市場調査 (market) |
| 03_planning | 企画立案・プロット |
| 04_script | 台本：draft → review → final |
| 05_material | 文字起こし：transcript_self / transcript_rival |
| 06_prompt | AI用プロンプト |
| 07_character | キャラクター設定 |
| 08_manual | 台本マニュアル・執筆ガイドライン |
| 09_thumbnail | サムネイル素材 |
| tools | スクリプト・ツール |
