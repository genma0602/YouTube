# チャンネルフォルダのテンプレート

このフォルダをコピーして、新しいチャンネル用フォルダを作成してください。

例: `_template` をコピー → `マイチャンネル名` にリネーム

## サブフォルダの役割

| フォルダ | 用途 |
|----------|------|
| 01_競合台本分析 | 競合チャンネルの台本・構成メモ・分析結果 |
| 02_自チャン台本分析 | 自チャンネルの過去動画の台本分析・傾向メモ |
| 03_台本執筆 | これから出す動画の台本・下書き |
| 04_プロンプト | このチャンネル専用のAI用プロンプト |
| 05_文字起こし | このチャンネルの動画の書き起こしデータ |
