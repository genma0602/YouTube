# サムライ経済学 台本マニュアル v2

**目的**: このマニュアルの通りに進めれば、誰でもサムライ経済学の台本が書ける  
**抽出元**: 16本の競合台本分析から共通パターンを抽象化  
**更新日**: 2026-02-16  

---

# ═══ 第一部：3つの原則 ═══

台本の全てはこの3つに集約される。迷ったらここに戻る。

### 原則1：大きな数字は「1人の生活」に置き換える

何兆円、何百万人といった数字をそのまま出しても伝わらない。  
「1人の人間の1日の暮らし」に変換して初めて伝わる。

```
❌「インフレ率130万%」
✅「1ヶ月の給料で卵12個しか買えない」
```

**変換の3段階**:
① まず数字そのものを出す → ② 身近なものと比較する → ③ 映像が浮かぶ表現にする

### 原則2：難しい言葉を使ったら、同じ文の中で言い直す

視聴者が「今のどういう意味？」と思った瞬間に離脱する。  
難しい言葉を使うなら、必ず同じ文の中で日常の言葉に言い換える。

```
❌「アービトラージが発生し…」
✅「安い国で買って高い国で売る仕組み、つまりアービトラージが…」
```

### 原則3：結論を押しつけず、問いを返す

「こうすべきだ」ではなく「あなたはどう思いますか？」で終わる。  
視聴者に考えさせることで、コメント・共有が生まれる。

# ═══ 第二部：ナレーターと主人公の役割 ═══

この台本では、**ナレーター**と**主人公**という2つの役割がある。  
この2つを混同すると台本が壊れるので、違いをはっきり理解する。

---

## ナレーター（＝チャンネルの声）

**誰？**：動画を進行する「語り」の声。画面には映らない。  
**口調**：丁寧だが堅すぎない。「です・ます」調。  
**役割**：

| やること | 具体例 |
|----------|--------|
| 主人公の紹介 | 「これは、ロンドンに住むジョンの物語です」 |
| 制度・背景の解説 | 「イギリスではVATと呼ばれる付加価値税があります」 |
| 数字やデータの提示 | 「IFSの調査によると、恩恵の差は約3倍です」 |
| 話題の切り替え | 「ここからは、誰も語りたがらない話に進みます」 |
| 日本との比較 | 「日本でも2019年に軽減税率が導入されました」 |

**ナレーターがやってはいけないこと**：
- 感情的に叫ぶ、泣く、怒る（それは主人公の仕事）
- 「僕は」「俺は」と一人称で体験を語る（それは主人公の仕事）
- 長い独り言を続ける（主人公の物語を止めてしまう）

---

## 主人公（＝その国に生きる1人の人間）

**誰？**：動画のテーマとなる国に住む架空の人物。  
**口調**：友達に話しかけるような口語。「〜だ」「〜なんだ」「〜だよ」調。  
**設定に必ず含めるもの**：名前・年齢・職業・生活の悩み

**役割**：

| やること | 具体例 |
|----------|--------|
| 自分の体験を語る | 「先月、病院に電話した。3週間待ちだと言われた」 |
| 自分の感情を出す | 「冗談じゃない」「怖かった」「恥ずかしいけど」 |
| 身近な人のエピソードを話す | 「隣人は70代で、毛布を重ねて寝ていた」 |
| 冒頭で視聴者を引き込む | 「僕は32歳なのにまだ親と暮らしている。おかしいと思う？」 |
| 結末で余韻を残す | 「来月33歳になる。そろそろ実家を出たい」 |

**主人公がやってはいけないこと**：
- 制度や政策を教科書のように説明する（それはナレーターの仕事）
- 統計データを淡々と読み上げる（それはナレーターの仕事）
- 途中で急に「です・ます」調になる（口調が壊れる）

---

## 台本の中での使い分け

台本は**主人公の語り**と**ナレーターの解説**を交互に切り替えながら進む。  
1.5〜2分に1回切り替えると、テンポが良くなる。

```
【台本の流れのイメージ】

主人公 ：「僕はマイケル。スーパーで働いている。先月フードバンクに行った」
              ↓
ナレーター：「イギリスではVATという消費税があり、食料品は0%です」
              ↓
主人公 ：「ジャファケーキの裁判、知ってるかい？ これがまた面白い話でね」
              ↓
ナレーター：「ここからは、0%の裏側にある構造に迫ります」
              ↓
主人公 ：「フードバンクで出会ったサラは看護師だった。手が震えていた」
              ↓
ナレーター：「日本でも軽減税率が導入されています。構造は同じです」
              ↓
主人公 ：「今日もスーパーで買い物をした。パン。0%。でも見え方が変わった」
```

**重要**：主人公のパートでは絶対にナレーターの口調を混ぜない。  
ナレーターのパートでは主人公の感情を入れない。  
**口調が混ざった瞬間に、視聴者は「誰が話してるの？」と混乱して離脱する。**

# ═══ 第三部：台本の基本の流れ ═══

どんなテーマでも、台本の流れはこの骨格に沿う。

```
[つかみ] → [問い] → [背景] → [本題の解説] → [感情のピーク] → [日本の話＋教訓] → [締め]
```

| パート | 役割 | 尺の目安 |
|--------|------|---------|
| **つかみ** | 手を止めさせる。スクロールを止める | 全体の5〜10% |
| **問い** | 「最後まで見る理由」を渡す | 5% |
| **背景** | 理解するための土台を作る | 10〜15% |
| **本題の解説** | 問いに答え始める | 30〜40% |
| **感情のピーク** | 視聴者の心が最も動く場面 | 10〜15% |
| **日本の話＋教訓** | 「自分にも関係ある」と思わせる | 10〜15% |
| **締め** | コメント→チャンネル登録→次回 | 5% |

# ═══ 第四部：4つの台本パターン ═══

16本の台本を分析した結果、全ての台本は**4つのパターン**のどれかに当てはまる。  
同じテーマの続編を作る時は、**前回と違うパターンを選ぶ**。

---

## パターンA：物語型（人が主役）

**どんな台本？**：1人の人間の体験を通して、国や経済の問題を描く

**こういう時に使う**：
- その国に住む当事者がいる
- 個人の体験が国の問題を象徴している
- 感情に訴えたい（初回・シリーズ1本目に最適）

**流れ**：

```
① つかみ（衝撃 or 日常→衝撃）
② 主人公の登場（名前・年齢・職業・悩み）
③ かつての平穏（昔は良かったパート）
④ 崩壊の始まり（最初の異変・小さなサイン）
⑤ 崩壊の仕組み（なぜこうなったかの解説を挟む）
⑥ どん底（感情の最深部。体の感覚で描く）
⑦ 転機（希望 or 決意 or 脱出）
⑧ 教訓＋日本の話＋締め
```

**感情の動き**：山型（上がって→落ちて→少し戻る）

```
    平穏         
   ╱    ╲         教訓
  ╱      ╲       ╱    
 ╱        ╲     ╱     
╱          ╲   ╱      
つかみ       どん底        締め
```

**バリエーション**：

| 変形 | 特徴 | 向いている場面 |
|------|------|--------------|
| **一人称体験** | 主人公が「僕は」で自分の体験を語る | 国家崩壊・脱出・生活の苦しさ |
| **3世代にわたる話** | 祖父母→親→自分の3世代で歴史を描く | 人権・自由・長期的な変化 |
| **サバイバル手順** | 脱出・対策を具体的に手順化する | シリーズ完結編・実用性を出したい時 |

**この型で必ずやること**：
- 主人公に**名前・年齢・職業**を必ず設定する
- 「昔は良かった」パートで**匂い・音・手触り**を入れる → 崩壊の落差が大きくなる
- 崩壊は一気にではなく**段階的に**描く（食べ物→医療→命のように）
- 結末で冒頭の場面を繰り返す（冒頭と結末をつなげる）

---

## パターンB：謎解き型（問いが主役）

**どんな台本？**：「なぜ？」「誰が得した？」の問いを解き明かしていく

**こういう時に使う**：
- 事件の「裏側」がある
- 「誰も言えないこと」がテーマ
- 視聴者は結果を知っているが原因を知らない
- シリーズ2本目に最適（1本目で物語型→2本目で謎解き型）

**流れ**：

```
① つかみ（結果をいきなり叩きつける）
② 問い（「なぜ？」「誰が得した？」「次に何が起きる？」）
③ 事実の整理（何が起きたかを先にまとめる）
④ 「ここからが本題」の宣言（表と裏の切れ目）
⑤ 答えを1つずつ開封（「3つの理由」のように番号をつけて）
⑥ 過去にも同じことがあった（歴史の事例を出す）
⑦ 得した人と損した人の整理
⑧ 教訓＋日本の話＋締め
```

**感情の動き**：階段型（好奇心が段階的に上がる）

```
                    答え③
              答え②╱     
        答え①╱   ╱ 歴史
  問い  ╱   ╱   ╱  ╱╲
  ╱╲  ╱   ╱   ╱  ╱  ╲教訓
 ╱  ╲╱   ╱   ╱  ╱    ╲
╱   事実整理 本題の宣言     締め
つかみ
```

**バリエーション**：

| 変形 | 特徴 | 向いている場面 |
|------|------|--------------|
| **事件の裏側** | 「なぜ今？」を政治・外交で分析 | 軍事行動・外交問題 |
| **タブーに触れる** | 誰も口にできない問題をデータで語る | 移民・人種・社会分断 |
| **定説を検証する** | 「○○は本当か？」→データで検証→実は半分嘘 | 通説への反論・後編向き |

**この型で必ずやること**：
- **「ここからが本題」の宣言**を必ず入れる（「ここまでが表。ここから裏」のような短い1〜2文）
- 答えは**3つに番号をつけて**出す（記憶に残り、人に伝えやすくなる）
- タブーに触れる時は**先に予防線**を張る（「これは誰かを責める話ではない」）
- 「感情ではなく数字で語る」と宣言すると信頼が生まれる

---

## パターンC：連鎖型（時間が主役）

**どんな台本？**：出来事がドミノ倒しのように連鎖していく過程を追う

**こういう時に使う**：
- 1つの判断・事件が連鎖的に問題を拡大した
- 「現在進行形」の事件を追っている
- 時間軸で追うと原因と結果のつながりがはっきりする

**流れ**：

```
① つかみ（最も衝撃的な結果 or 「今この瞬間」）
② 問い＋「時間を巻き戻します」の宣言
③ ドミノ①（最初のきっかけ）
④ ドミノ②（連鎖反応）
⑤ ドミノ③（さらに広がる）
⑥ ドミノ④（最も深刻な結果＝感情のピーク）
⑦ 「その後」（数年後・現在はどうなったか）
⑧ 教訓＋日本の話＋締め
```

**感情の動き**：右肩上がり（どんどん深刻になる）

```
                    ドミノ④
              ドミノ③╱╲    
        ドミノ②╱   ╱  ╲ その後
  ドミノ①╱   ╱   ╱    ╲╱╲
     ╱  ╱   ╱   ╱        ╲
    ╱  ╱   ╱   ╱     教訓  締め
つかみ               
```

**バリエーション**：

| 変形 | 特徴 | 向いている場面 |
|------|------|--------------|
| **時間軸ドミノ** | 日付を追って連鎖を見せる | 政策が次々と壊れていく時 |
| **実況中継風** | 「今この瞬間」の緊迫感 | 現在進行中の事件 |
| **問題の積み上げ** | 問題を6〜8層積み上げる | 「勝ち組」の裏側を暴く時 |

**この型で必ずやること**：
- **「わずか○日後」「まだ終わりじゃない」**と繰り返すことで加速感を出す
- 各ドミノの間に**息継ぎ**を入れる（ちょっとした笑い、バランスを取るデータなど）
- 連鎖の途中で**「ここでこう思うかもしれない」**と視聴者の反論を先に潰す
- 「その後」パート（他のチャンネルがやらない追跡）を入れると差別化できる

---

## パターンD：解説型（仕組みが主役）

**どんな台本？**：経済の仕組みや構造を「なるほど」の連続で理解させる

**こういう時に使う**：
- 日本国内の経済テーマ
- 仕組みそのものの理解が価値になる（金利・為替・株価）
- 事件より「構造」が面白い

**流れ**：

```
① つかみ（日常の違和感 or 逆説）
② 主人公（「佐藤さん」のような普通の人＝視聴者の分身）の登場
③ 問題を分解（「3つの原因がある」）
④ 原因①の解説＋日常の例で置き換え
⑤ 原因②の解説＋具体的な計算
⑥ 原因③の解説＋過去のデータ
⑦ 全体像をまとめる（「つまりこういう仕組みだ」）
⑧ 「あなたの家計ではどうですか？」＋締め
```

**感情の動き**：階段型（「なるほど」が積み上がる）

```
              原因③
        原因②╱    まとめ
  原因①╱   ╱   ╱╲
     ╱   ╱   ╱    ╲
    ╱   ╱   ╱  自分事  締め
つかみ                    
```

**バリエーション**：

| 変形 | 特徴 | 向いている場面 |
|------|------|--------------|
| **エッセイ風** | 視聴者の分身（佐藤さん）が疑問を追う | 日経平均・給料・物価 |
| **2つの視点** | 生活者と専門家の2つの目線を行き来 | 為替・金利・投資 |
| **計算シミュレーション** | 具体的な数字でステップ計算する | 住宅ローン・家計・税金 |

**この型で必ずやること**：
- **主人公＝視聴者の分身**にする（「佐藤さん」のような普通の人）
- 仕組みは**3つ以内に分解**する（4つ以上は覚えられない）
- 必ず**日常の行為に置き換える**（「電気をつけるたびに円が海外に出ていく」）
- 結論は**「答えは人それぞれ」**でOK。押しつけない。

---

## パターンの選び方

```
テーマが決まったら、以下の2つを考える：

Q1. このテーマの「一番面白いところ」は何か？
  → 「1人の人生」が面白い → パターンA：物語型
  → 「裏側・真相」が面白い → パターンB：謎解き型
  → 「連鎖・加速」が面白い → パターンC：連鎖型
  → 「仕組み・構造」が面白い → パターンD：解説型

Q2. 続編か？
  → YES → 前回と違うパターンを選ぶ
  → NO → Q1の答えをそのまま使う
```

# ═══ 第五部：導入（冒頭）の書き方 ═══

導入＝台本全体の10%。5000字の台本なら**400〜500字、8〜12文**。

## 3ステップで書く

```
Step 1：つかみ（1〜4文） → スクロールを止める
Step 2：主人公の登場（3〜4文） → この人の話を聞きたいと思わせる
Step 3：問い＋約束（2〜3文） → 最後まで見る理由を渡す
```

---

### Step 1：つかみの書き方

**つかみの鉄則**: 1文目で「え？」と思わせ、4文以内で問いへの橋を架ける。

**つかみには4つの方法がある**（組み合わせてもOK）：

| 方法 | やり方 | 視聴者の心理 |
|------|--------|-------------|
| **衝撃の数字で入る** | 前置きゼロで最も驚く数字を出す | 「そんなに？」→ 続きが気になる |
| **日常から入って壊す** | 何気ない日常を3〜5文並べ、最後の1文で壊す | 安心→衝撃の落差が大きい |
| **結果だけ先に出す** | 理由も経緯も省いて結果だけ叩きつける | 「なぜそうなった？」が爆発する |
| **自分の体験で入る** | 主人公自身の体験（病院・買い物・食卓）で始める | すぐに共感が生まれる |

**つかみの文字数**: 1文あたり**6〜20字**。短いほど衝撃が大きい。

---

### Step 2：主人公の登場

```
[名前] ＋ [年齢] ＋ [職業] ＋ [今の状況]（1〜2文）
→
「なぜこの話をするのか」の理由（1〜2文）
→
前回の動画との接続（あれば、1文だけ）
```

**主人公の種類**：

| 種類 | いつ使うか |
|------|----------|
| **海外の当事者**（「僕の名前は○○」） | 海外テーマで個人の体験を語る時 |
| **視聴者の分身**（「佐藤さん」的な普通の人） | 日本国内テーマ、仕組みの解説 |

---

### Step 3：動画全体の「問い」の作り方

**問いの公式**：
```
[みんなが信じていること] なのに、なぜ [現実は逆] なのか？
```

**良い問いの条件**：
1. 視聴者が感じていたけど**言葉にできなかった疑問**を代弁している
2. **答えを知るまで離脱できない**
3. **1行（15〜25字）**で言える

**問いの出し方**：

| パターン | やり方 | 向いている場面 |
|---------|--------|--------------|
| **シンプルに1つ** | 「なぜ○○なのか？」1つだけ | 基本形。迷ったらこれ |
| **3つ並べる** | 「なぜ？」「誰が得？」「次に何が？」を3連 | 情報量が多いテーマ |
| **2軸で聞く** | 「なぜ○○したのか？＋どうやって？」 | 理由と方法の両方が面白い時 |

問いの直後に**約束**（「最後まで見ればわかる」）と**スタート宣言**（「では始めましょう」）を入れる。

---

# ═══ 第六部：台本で使えるテクニック ═══

以下のテクニックを必要に応じて使う。全部使う必要はない。  
★が多いほど重要。

## 文章のテクニック

| テクニック | 何をするか | ★ |
|-----------|----------|---|
| **数字を生活に変換** | 大きな数字を「1人の1日の暮らし」に置き換える | ★★★ |
| **短い文で落とす** | 長い文の後に極端に短い文（3〜10字）でガツンと落とす | ★★★ |
| **体の感覚で描く** | 匂い・音・手触り・味・温度を言葉で描く。「毛布を重ねて寝ていた」など | ★★★ |
| **日本との比較** | 海外の話を「日本で同じことが起きたら」に接続する | ★★★ |
| **対比で見せる** | 数字vs数字、昔vs今、Aさんの人生vsBさんの人生 | ★★★ |
| **本人の台詞を入れる** | 「彼はこう言った」で実際の言葉を入れる | ★★☆ |

## 構造のテクニック

| テクニック | 何をするか | ★ |
|-----------|----------|---|
| **「ここからが本題」の宣言** | 前半と後半の切れ目で「ここから空気が変わる」と短く宣言する | ★★★ |
| **番号をつけて答える** | 答えを「3つの理由」のように番号つきで出す | ★★★ |
| **冒頭と結末をつなげる** | 冒頭で出した場面・言葉を、結末で「意味が変わった状態で」繰り返す | ★★★ |
| **問題を段階的に積み上げる** | 食→住→医療→命のように、問題をだんだん深刻にしていく | ★★★ |
| **視聴者の疑問を先回りする** | 視聴者が「なぜ？」と思う前に、その疑問を言葉にして回答する | ★★★ |
| **前半に意味深な言葉→後半で解明** | 「おじいちゃんの言葉」のように、前半に謎を置き後半で回収する | ★★☆ |
| **ナレーターと主人公を切り替える** | 感情担当（主人公）と情報担当（ナレーター）を交互に切り替える | ★★☆ |
| **過去に同じことがあった** | 歴史上の似た事例を3つ並べて「繰り返している」と示す | ★★☆ |
| **小さな希望を出してすぐ潰す** | 「これで大丈夫かと思ったが…」→すぐに否定して緊張感を維持する | ★★☆ |

## 心理のテクニック

| テクニック | 何をするか |
|-----------|----------|
| **結果だけ見せて原因を隠す** | 先に衝撃的な結果を見せて、「なぜ？」で最後まで引っ張る |
| **「見ない方がいい」と言う** | 「閉じた方がいい」と言うと逆に見たくなる心理を使う |
| **答えを直接言わず気づかせる** | 数字を並べて視聴者に自分で計算・気づきをさせる |
| **「聞いてくれませんか」と巻き込む** | 「聞く」と決めたら最後まで見る心理を使う |

# ═══ 第七部：感情の設計 ═══

## 場面ごとに使う感情

| 感情 | どこで使うか |
|------|------------|
| **衝撃** | つかみ、「ここからが本題」の直後、どん底 |
| **共感** | 主人公の自己紹介、生活の描写 |
| **好奇心** | 問いの提示、「3つの理由」の予告 |
| **なるほど** | 仕組みの解説、うまい比喩 |
| **恐怖** | 問題がだんだん深刻になる場面、数字を生活に変換した時 |
| **怒り** | 不正・搾取が明らかになる場面 |
| **悲しみ** | 家族の別れ、失ったもの |
| **危機感** | 日本との比較 |
| **余韻** | 締めの直前 |

## 2つのルール

**ルール1**：重い感情（衝撃・恐怖・悲しみ）が**3パート以上続いたら息継ぎを入れる**

息継ぎの方法：
- ナレーターに切り替える
- バランスを取るデータを入れる（「全てが悪いわけではない」）
- ちょっとしたユーモアを入れる
- 仕組みの解説パートに切り替える

**ルール2**：**1.5〜2分に1回**、主人公とナレーターを切り替える、  
または話のモード（体験談/解説/データ/教訓）を変える

# ═══ 第八部：文のリズム ═══

## 場面ごとの文字数の目安

| 場面 | 文字数 |
|------|--------|
| つかみ（冒頭1〜3文） | **6〜20字** |
| 通常の解説 | **20〜40字** |
| 感情が爆発する瞬間 | **3〜10字** |
| 動画全体の問い | **15〜25字** |
| 教訓・結論 | **25〜40字** |

## リズムの鉄則

**短→短→短→長→短（ドン！）** が基本パターン。

台本全体で**10字以下の極端に短い文を5〜10個**配置する。  
それぞれが「話の転換」「感情の爆発」「断言」のどれかとして使う。

```
例：
「フォルクスワーゲン3万5000人削減」（16字）
「メルセデス4万人」（8字）
「冗談じゃない」（6字）  ← ここが爆発
「僕たちは静かに沈んでいる」（12字）
```

# ═══ 第九部：結末の書き方 ═══

## 教訓の書き方

教訓は**3つ以内**に絞る。視聴者が人に伝えられる短さにする。

## 結末の4パターン

| パターン | やり方 |
|---------|--------|
| **冒頭の繰り返し＋意味の変化** | 冒頭の言葉を繰り返し、動画を見た後では意味が変わっている状態にする |
| **チェックリスト型** | 崩壊の条件を列挙→「あなたの国はいくつ当てはまりますか？」 |
| **「普通の夢」型** | 「普通に暮らす夢が高価になりすぎた」のような普遍的な一文で閉じる |
| **2人の対比型** | 2人の主人公が互いに相手の国に憧れている → 構造の矛盾を2文で表す |

## 締め（コメント・登録誘導）の順番

```
① 問いかけ（「あなたはどう感じましたか？」）→ まずコメントを誘う
② 価値の確認（「この動画が役に立ったなら」）
③ 行動指示（「高評価とチャンネル登録をお願いします」）
④ 次回への期待（「それではまた次の動画で」）
```

# ═══ 第十三部：やってはいけないこと ═══

| やってはいけないこと | 理由 |
|-------------------|------|
| **結論を押しつける** | 「投資すべきだ」「移民は悪い」は言わない。問いを返す |
| **特定の国や民族を攻撃する** | 「問題は制度であり人ではない」を必ず入れる |
| **データなしで主張する** | 全ての主張に数字か出典をつける |
| **重い感情を3パート以上連続させる** | 息継ぎを入れないと視聴者が疲れて離脱する |
| **難しい言葉をそのまま放置する** | 使ったらその場で日常の言葉に言い換える |
| **冒頭の前置きが長い** | 8文以内に問いを提示する |
| **主人公とナレーターの口調を混ぜる** | 「誰が話してるの？」と混乱して離脱される |
