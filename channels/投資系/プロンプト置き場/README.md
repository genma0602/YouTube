# プロンプト置き場（投資系）

投資・経済系YouTubeの分析・制作で使うプロンプトを置くフォルダです。

## ファイル一覧

| ファイル | 用途 |
|---|---|
| [投資経済台本分析.md](./投資経済台本分析.md) | 台本の構成・感情設計・視聴維持・伸びる理由・企画者の思考を分析する（外部検索なし） |
| [ファクトチェッカー.md](./ファクトチェッカー.md) | 数字・主張の正確性確認、用語解説、経済の仕組み解説（外部検索あり） |
| [企画割り出しプロンプト.md](./企画割り出しプロンプト.md) | YouTube動画一覧データから、伸びる企画パターン・平均再生回数・マネすべき動画・新企画案を抽出する |
| [新規チャンネルコンセプト立案プロンプト.md](./新規チャンネルコンセプト立案プロンプト.md) | 既存の台本パターン・企画パターンを転用して、新規チャンネルのコンセプト案・運営者キャラクター設定案・初動10本・タイトル型を、立案根拠（なぜその案か）付きで立案する |
| [台本構成整形プロンプト.md](./台本構成整形プロンプト.md) | 台本・文字起こしを「構成｜テキスト」の表形式に整形する（分析前の前処理） |

## 使い方

1. 目的に合ったプロンプトファイルを開き、全文をコピーする
2. Cursor や ChatGPT などに貼り付ける
3. 「【ここに台本を貼り付ける】」（または該当の貼り付け欄）に、分析したい台本・文章を入れる
4. 出力は各チャンネルの `分析結果/` フォルダに保存する（例：`投資のすゝめ/分析結果/`）

### どれを使うか

| やりたいこと | 使うプロンプト |
|---|---|
| なぜ伸びたか・冒頭・感情・構成を分析したい | 投資経済台本分析 |
| 数字や事実が正しいか、用語・仕組みを補足したい | ファクトチェッカー |
| チャンネル全体の企画パターンを洗い出したい | 企画割り出しプロンプト |
| 既存の分析資産から新しいチャンネルを立ち上げたい | 新規チャンネルコンセプト立案プロンプト |
| 生の台本を構成付きの表に直してから分析したい | 台本構成整形 → 投資経済台本分析 |

台本分析とファクトチェックは役割が異なります。同じ台本でも、両方使う場合は別ファイルに分けて保存するのがおすすめです。

## 補足

- **ファクトチェッカー**は外部検索を前提とするため、Web検索が使える環境での実行を推奨
- **投資経済台本分析**はソース検索・出典確認は行わない（YouTube台本としての設計分析に特化）
- 分析結果のファイル名は、既存と揃えて `{テーマ}_台本×分析_{日付}.md` 形式がおすすめ（ファクトチェックのみの場合は `{テーマ}_ファクトチェック_{日付}.md` などでも可）
