# プロンプト置き場（投資系）

投資・経済系YouTubeの分析・制作で使うプロンプトを置くフォルダです。

## ファイル一覧

| ファイル | 用途 |
|---|---|
| [投資経済系YouTube台本_分析・ソース検索プロンプト.md](./投資経済系YouTube台本_分析・ソース検索プロンプト.md) | 競合・参考チャンネルの台本を、テーマ・出典・構成・伸び理由まで分析する |
| [企画割り出しプロンプト.md](./企画割り出しプロンプト.md) | YouTube動画一覧データから、伸びる企画パターン・平均再生回数・マネすべき動画・新企画案を抽出する |

## 使い方

1. 上記プロンプトファイルを開き、全文をコピーする
2. Cursor や ChatGPT などに貼り付ける
3. 「【ここに台本を貼り付ける】」の位置に、分析したい台本（文字起こし）を入れる
4. 出力は各チャンネルの `分析結果/` フォルダに保存する（例：`ガーコチャンネル/分析結果/`）

## 補足

- ソース検索を含むため、Web検索が使える環境での実行を推奨
- 分析結果のファイル名は、既存と揃えて `{テーマ}_台本×分析_{日付}.md` 形式がおすすめ
