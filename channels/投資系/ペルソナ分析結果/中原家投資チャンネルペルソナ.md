## 3. 抱えている悩み・不安（視聴を生んでいる「痛み」）

台本3本に共通して「冒頭で代弁」されている悩みを抽出。

### 投資に関する痛み

1. 「YouTubeで紹介された頃にはもう上がっている」出遅れ恐怖
   - 高配当・永久保有系チャンネルへの不満の言語化
2. 「自分で選んでないから、下がった時に判断できない」不安
   - 人に言われて買った株は、下がっても売れないし、買い増しもできない
3. 「変な株を掴んで大損する」のが何より怖い
   - 儲けたいより「ハズレを引きたくない」が先に立つ
4. 「証券会社の人やインフルエンサーは信用できない」疑い
   - 利害関係のない一般人の経験者の話を聞きたい
5. 「四季報って分厚くて難しそう」入門ハードル
   - 1人で読めない、誰かに横についてほしい

### 生活・人生に関する痛み

6. 老後資金の不安（年金だけでは足りないと言われ続けている）
7. 親の介護がそろそろ視野に入る（→カナミックのような介護系銘柄に反応）
8. 子供の教育費・独立後の生活費の不安
9. 配偶者と「お金の話」をオープンにできていない（中原家の夫婦会話に憧れる）
10. 「自分で判断する自信がない」自己効力感の低さ

## 4. 「何を求めて」この動画を見ているのか

視聴動機を「表向きの欲求」と「裏側の本質的欲求」の二層で整理。

### 表向きに欲しているもの（=サムネ・タイトルでクリックする理由）

| 求めているもの | 具体表現 |
|---|---|
| ① 上がりそうな具体銘柄 | 「買うべき3銘柄」「お買い得5選」 |
| ② 出遅れていない安心感 | 「まだ間に合う」「こっそり聞いた」 |
| ③ 信頼できる経験者の本音 | 「日本株歴28年の夫」 |
| ④ 少額で始められる手軽さ | 「2万円で買える」「2,500円以下」 |
| ⑤ 知らない情報の先取り感 | 「こっそり」「夫が本当は教えたくない」 |

### 本質的に欲しているもの（=最後まで見て満足する理由）

| 本当の欲求 | 中原家チャンネルが満たしている方法 |
|---|---|
| A. 自分で判断できるようになりたい（自己効力感） | 6つの判断基準＝再現可能なルールを毎回反復で教える |
| B. 騙されたくない・損したくない（損失回避） | 「最低5つ揃わないと買わない」と慎重姿勢を見せる |
| C. 専門家に横についてもらう安心感 | 妻役が初心者を代弁し、夫が優しく教える「個別レッスン感」 |
| D. 家族で投資を語れるようになりたい | 夫婦の掛け合いそのものがロールモデルになる |
| E. ガチガチの投資世界から距離を取りたい | 関西弁・寸劇・キティカフェ話で「人間味」を補給 |
| F. 自分の知ってる会社で投資したい（既知性バイアス） | サンリオ・かっぱ寿司・ルネサンスなど生活密着銘柄を選定 |
| G. お金以上の意味を持ちたい | 「投資は人生の縮図」という哲学パートで自己肯定を得る |

→ つまり視聴者は「銘柄を知りたい」のではなく、**「自分で銘柄を選べる自分になりたい」**のです。

## 7. このペルソナが「絶対に見ない」競合動画

ペルソナ像をくっきりさせるための裏側の確認。

| 見ない動画タイプ | 理由 |

| 米国株・グロース株中心のチャンネル | 為替・英語・大型ハイテクで距離感がある |
| FX・暗号資産系 | ギャンブル感が強く損失恐怖が勝つ |

## 9. このペルソナ理解から導ける企画・台本の方針

| 項目 | 方針 |
|---|---|
| タイトル設計 | 「まだ間に合う」「こっそり聞いた」「初心者が買うべき」「N個」のテンプレを必ず1つ以上入れる |
| 冒頭30秒 | 「YouTubeで紹介された頃にはもう上がっている」など、視聴者の痛みを必ず代弁する |
| 銘柄選定 | サンリオ・かっぱ寿司・ルネサンスのような「視聴者の生活圏にある会社」に限定 |
| 株価レンジ | 1株2,500円以下、購入総額3万〜30万円で買えるものに絞る |
| 説明スタイル | 専門用語は必ず家計の比喩（給料・食費・家賃）に置き換える |
| 構成 | 判断基準を先出し → 同じ基準で3〜5銘柄を反復 → 哲学パートで締め |
| 演出 | 妻の合いの手・寸劇を中盤に1回以上挟む |
| エンディング | 妻のセリフで「次は◯◯教えて」のリクエスト型予告を入れる |
| 触れない方が良いこと | 米国株・FX・暗号資産・マクロ難解話・億り人自慢 |
| 強化すると伸びる軸 | ①介護・親世代テーマ（カナミック系） ②株主優待での生活還元 ③新NISAでの最初の1銘柄 |
