# 投資のすゝめ・台本の設計原理（抽象化版）

## 0. 最上位の一文

> **この台本群は「未解決の感情を抱えて来た人を、安全に“納得して動ける状態”へ運ぶ装置」である。**

視聴者は答えではなく「モヤモヤ」を持って来る。台本の仕事は、そのモヤモヤに名前を付け、一度揺さぶり、解消し、最後に「自分で決められる感覚」を渡して返すこと。情報提供は手段にすぎず、本質は**感情の状態遷移の設計**である。

---

## 1. 貫く原理①：構成とは「説得」ではなく「感情の旅程」

具体の5フェーズ（導入→前提→期待醸成→冷却→処方箋）は、抽象化すると次の**5つの心理ステップ**になる。順番はテーマが変わっても不変。

| 具体フェーズ | 抽象化した本質 | 視聴者の内心の変化 |
|---|---|---|
| 導入・フック | **共感**：「あなたの状態はこれですね」 | 警戒 → 当事者「自分の話だ」 |
| 前提整理 → 再定義 | **転覆**：「あなたの前提は間違っている」 | 思い込み → 知的更新「そうだったのか」 |
| 期待醸成 | **論証**：「だから大丈夫／だからチャンス」 | 半信半疑 → 確信 |
| 冷却・揺さぶり | **誠実**：「でも悪い面も正直に出す」 | 信頼の担保「この人は煽っていない」 |
| 処方箋・着地 | **委譲**：「決めるのはあなた」 | 安心＋自律「自分で選べる」 |

**抽象化のキモ**：多くの解説動画は「論証」だけで作られる。このチャンネルが強いのは、論証の前に**共感（当事者化）と転覆（前提破壊）**を必ず置き、論証の後に**誠実（自己批判）と委譲（判断の返還）**を必ず置く点。つまり**論理の外側を感情で挟んでいる**。

---

## 2. 貫く原理②：3つの心理レバーを必ず引く

8本すべてが、視聴者の心を動かすために**同じ3つのスイッチ**を押している。これがコンテンツの「効く成分」。

1. **当事者化（これは自分の話だ）**
   - 抽象：受け手の“言語化前の感情”を、本人より先に言葉にしてあげる。
   - 効果：傍観者を主人公に変える。離脱不能にする。

2. **知的更新（自分は賢くなった）**
   - 抽象：受け手が握っている“常識・思い込み”を1つ、根拠付きで反転させる。
   - 効果：満足感と「語りたい」衝動（＝拡散）を生む。提供価値の中核。

3. **自己肯定（自分は間違っていない）**
   - 抽象：受け手の過去の選択・現在の立場を、否定せずに肯定する。
   - 効果：不快な離脱要因を消し、発信者への信頼に変える。

→ **3レバーの共通点**：いずれも「情報」ではなく「**受け手の自己イメージ**」に働きかけている。人は事実でなく“自分がどう見えるか”で動く、という前提に立った設計。

---

## 3. 貫く原理③：感情は「フラットにせず振り子にする」

情報を平らに並べると飽きる。だから**不安と安心を往復させる**。

```
        山（納得・安心）
       ╱        ╲       ╱╲
 ─────╱──────────╲─────╱──╲──── 中立線
     ╱            ╲   ╱
（不安）          谷（冷却）
```

- **谷（不安・冷却）を必ず作る**：悪材料・リスク・暴落をあえて見せる。
- **谷の直後に山（安心・解消）を置く**：中和・再定義・肯定で埋める。
- **原則**：谷が深いほど、その後の山（納得）は強固になる。ただし谷を深くしすぎると離脱するので、深さは「受け手の痛みの強さ」に反比例して調整する（すでに痛い人には浅く、余裕がある人には深く掘れる）。

抽象化すると、**「安心」は単独では作れず、必ず“一度の不安”とセットでしか生まれない**という設計思想。

---

## 4. 貫く原理④：たった1つの可変パラメータ＝「悪材料の扱い」

8本は同じ設計図の反復で、**変えているのは事実上1つの変数だけ**。

> **悪材料（割高/リスク/暴落）を「いつ・どれだけ・何回」見せるか は、対象に対する受け手の“感情温度”で決まる関数である。**

| 受け手の感情温度 | 悪材料の扱い（抽象） |
|---|---|
| 熱狂（飛びつきたい） | 後ろに置いて軽く冷やす ＝ ブレーキ |
| 過熱だが割安が売り | 冒頭で即否定 ＝ 露払い |
| 痛み（すでに下落） | 中盤に正面から出して埋める ＝ 鎮痛 |
| 恐怖（過去のトラウマ） | 冒頭で最大化し全編で論破 ＝ 反論の的 |
| 混乱（理由が不明） | 悪材料の解明そのものを主題化 ＝ 教育 |

**抽象化の価値**：新しいテーマを扱う時、考えるべきは「構成をどう作るか」ではなく**「視聴者は今この対象に、熱狂・割安期待・痛み・恐怖・混乱のどれを感じているか」**だけ。それが決まれば、悪材料の置き場所＝構成は自動で決まる。

---

## 5. 貫く原理⑤：信頼は「煽らないこと」で積む長期資産

- 断定しない／既存の選択を否定しない／判断を本人に返す ── これらは抽象化すると**「短期の説得力をあえて捨てて、長期の信頼を取る」取引**。
- 強く主張するほど語尾を冷静にするのは、**主張の強さと語り口の冷静さを意図的にミスマッチさせる**ことで「冷静な人が強く言うほど説得力が増す」効果を狙っている。
- CTA（LINE・相談会）やリピート視聴は、この“煽らない誠実な解説者”という**キャラクター資産**の上にしか成立しない。コンテンツ設計とビジネス設計が同じ「信頼」で繋がっている。

---

## 6. 貫く原理⑥：情報を「点」でなく「線」で渡す

- バラバラのニュース・数字を、必ず**1本の物語の幹**（お金の動線／技術の流れ／時代の転換）に束ねる。
- 専門用語は定義せず、**受け手が既に知っている世界の比喩**に翻訳する（時間・損得・身近な物）。
- 抽象化すると、**「理解」とは新情報を受け手の既存の地図に接続することだ**という認知観に基づく設計。情報量の多さではなく“接続のしやすさ”で勝負している。

---

## 7. 抽象モデル（1枚要約）

```
                 ┌─────────────────────────────────┐
 受け手が抱える  │  未解決の感情（不安・欲・痛み・混乱） │
 「宙ぶらりん」  └─────────────────────────────────┘
                              │
   ① 共感   ：感情に名前を付け「自分の話」にする（当事者化）
   ② 転覆   ：常識を1つ壊す（知的更新）
   ③ 論証   ：強み＋権威で確信させる（点を線に）
   ④ 誠実   ：悪材料を自分から出す（信頼の担保）★温度で配置を可変
   ⑤ 委譲   ：肯定して判断を返す（自己肯定＋自律）
                              │
                 ┌─────────────────────────────────┐
 受け手が得る    │  納得して自分で動ける状態 ＋ 発信者への信頼 │
 「着地」        └─────────────────────────────────┘
```

## 8. 転用するための問い（チェックリスト）

新しい台本を作る時、構成を考える前にこの5問に答えれば設計が決まる。

1. **共感**：受け手は今、この対象に対して言葉にできていない“どんな感情”を抱えているか？（熱狂/割安期待/痛み/恐怖/混乱）
2. **転覆**：受け手が握っている“間違った常識”を1つ挙げるなら何か？それを何で反転させるか？
3. **論証**：その確信を支える「強み3〜5個」と「外部権威の数字」は何か？それらを束ねる1本の幹は？
4. **誠実**：あえて見せる悪材料は何か？そして問1の“感情温度”から、それを冒頭/中盤/後半のどこに置くか？
5. **委譲**：受け手の立場を何パターンに分け、それぞれをどう肯定し、どんな行動を渡すか？

→ この5問の答えが揃えば、**具体の構成（細部版ファイル参照）に機械的に流し込むだけ**で台本になる。
