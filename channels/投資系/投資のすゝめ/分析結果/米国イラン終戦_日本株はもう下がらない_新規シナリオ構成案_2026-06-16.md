# 新規シナリオ構成案：米国イラン終戦で日本株は「もう下がらない」のか

## 0. 設計方針（なぜこの型を選ぶか）

競合の **306k再生台本（5/13 日本株は今後もヤバい）** は、
**「最高値で“もう遅い／取り残される”の板挟みが同時発生する大型FOMO局面」** に対し、
「最高値FOMO → バブル不安をPERで割安証明 → 5つの構造理由で常識反転 → 主軸別の処方 → 比率見直しで着地」
の5フェーズで刺さった。今回の 6/14-15 相場は、それとほぼ同じFOMO局面（終戦合意＋日経7万円目前）。

### ★第2版の最重要方針：「並べる」のをやめて「1本の川」にする

投資のすゝめの本当の武器は、情報量ではなく **動線（幹）** ＝「バラバラのニュースが、実は1つの原因から起きていると束ねる力」。横断分析でも「**動線に乗らない情報は浮く**」と指摘されている（半導体「AIの金は最後に日本へ」／日米金利「辿ると全て金利」）。

そこで本案は、**1本の幹**を全編に通す。

> **幹：終戦の“織り込み” → 原油高リスクが後退 → 米インフレ再燃懸念が和らぐ → FRBが追加利上げを急ぐ理由が弱まる → 世界がリスクオン＆日本株が再評価される**

★金利の主語は **FRB（米）** で統一する。理由：今回市場が警戒していたのは「ホルムズ→原油高→米インフレ再燃→FRB利上げ」という連鎖で、それが終戦で逆回転したから。日本株へは2経路で効く。
- 直接経路：原油安 → 日本（資源輸入国）の家計・企業コストが軽くなる。
- 間接経路：米インフレ懸念↓ → FRB利上げ確率↓ → 世界リスクオン・グロース再評価・海外マネー → 日本株。

★さらに“政治の下支え”を支流に加える：**中間選挙を控えるトランプ政権は、景気・株価・企業業績を崩したくない**。市場は「選挙前は強い経済が維持されやすい」と見ており、これが下値を固くする。

この川に、すべての材料（日経の急騰13銘柄／理由／買う株・逃げる株／中間選挙）を**支流として合流させる**。
独立した「強みリスト」ではなく、「**1本の川の、上流から下流まで**」として語る。これが核。

### ポジティブトーンの担保
競合の型は元々「強気1解釈＋全員肯定＋断定回避」で放っておくとポジティブに振れる。冷却（リスク）は**軽く・短く・1回だけ**にし、着地は「もう下がらない」と断定せず「**持たない方がリスク**」へ変換して背中を押す。

- ⚠ 政治予想に深入りせず「市場がそう“見ている”」という織り込みの話に留める（断定回避）。

**ブロック12：河口＝たくさんに見えた材料は、ほぼ1本の川だった（収束）★権威の多層化（歴史データ）**

### ④ 処方箋（競合⑯〜⑳に対応＝“今回の相場の取り方”に純化）

> ★第4版の整理：処方箋は「**今回のニュースを、目の前でどう取るか**」に限定する。指数選択・買う/逃げる・出遅れ肯定の3つ。**“コアはインデックス”という思想的な結論は処方箋に置かず、着地（⑤）へ移す**（チャンネルの王道＝最後にインデックスへ戻す着地を強くするため）。

**ブロック13：指数の取り方＝今回は“広い上昇”だからTOPIX的に取るのが素直**
- ブロック7の通り、今回は半導体だけでなく建設・航空・自動車まで上がった＝**TOPIX型の広い上昇**。
- 日経＝値がさ・半導体の影響大／TOPIX＝時価総額加重で内需金融まで広い（S&P500・オルカンと同じ方式）。
- 「AIに集中して乗るなら日経寄り、今回のような**広い回復に乗るならTOPIX寄り**」と1つの軸で渡す。

**ブロック14：買うべき株・逃げるべき株も“同じ幹”で説明（タイトル回収）**
- 買い寄り＝**原油安・金利低下が追い風の側**：AI半導体周辺、建設・航空・自動車・素材（＝ブロック7で上がった面々）。
- 逃げる/不利寄り＝**原油↓が逆風の側**：戦時の高油価を織り込んでいた**エネルギー・資源**。
- 「買う株も逃げる株も、**判断基準は1つ。原油と金利がどっちへ動くか**」と幹で締める。
- ※断定せず「相場の力学として追い風／逆風」に留める（炎上回避）。

**ブロック15：出遅れ・乗れなかった人の自己肯定（★競合の核）**
- ★今回の急騰に乗れていない人を必ず救う一文：「今日の上げに乗れていなくても、まったく問題ない」。
- 理由を幹に乗せる：「終戦の織り込みは“一瞬のニュース”ではなく、原油→金利→再評価という**時間をかけて効く構造**。だから**出遅れ＝まだ仕込む時間が残っている**ということ」。
- 含み損・上がっていない銘柄を持つ人にも「全面高でもまだ出遅れている銘柄はある＝それは割安に放置されているだけ」と前向き解釈を渡す。
- ※「一度も視聴者を責めない／全員を勝者にする」核（構成分析9節）を担保。

### ⑤ 着地（競合㉒〜㉔に対応＝★最後はインデックスに戻すのがこのチャンネルの王道）

**ブロック16：60日ホルムズ開放の意味（独自パート）**
- 「60日の停戦・交渉ウィンドウ」が意識されている。ポイントは“60日で完全正常化するか”ではない。
- **市場にとっては、60日あれば「今すぐ最悪になる」リスクを外せる**ことが大きい。だから原油が下がり株が買われた。

**ブロック17：冷却（軽く1回だけ＝割高/上昇ネタの定石）**
- ただし和平はまだ**実装段階**。19日の正式締結前、核問題は先送り、ホルムズの実務正常化には時間がかかりうる。
- ★金利の釘刺し：「**FRBの利上げが完全になくなったわけではない**。年内に利上げ自体はありうるし、原油が再び上がればインフレ懸念も金利不安も戻る」。＝“急がなくなった”であって“消えた”ではない、を再確認。
- だから「どのシナリオに転んでも生き残れる組み方」が必要、と次のインデックス結論へ橋渡しする。

**ブロック18：★インデックス着地（この動画の思想的な結論／主軸別の全員肯定をここへ集約）**
- 結論：どの終戦シナリオになるかは誰にも確実に読めない。**だからこそ、コアはオルカンやS&P500のような低コスト分散インデックスでいい**。中東情勢・AI・金利・景気の全部を、一社一社の個別株で当てにいく必要はない。
- ★保有別の全員肯定をここで一括（処方箋から移設）：
  - オルカン勢：すでに日本株も米国株も内蔵。今回の上昇も自動で取り込めている＝**もう乗れている**。
  - S&P500勢：米金利低下なら追い風。地域分散で日本株（TOPIX）を少し足すのは攻めとして有効。
  - NASDAQ/FANG+勢：終戦・金利低下で最も伸びる一方、悪化時に最も傷む。**コアでなくサテライト（スパイス）**。順番を逆にしない。
- 「個別の“買うべき株・逃げるべき株”は、あくまでサテライトの遊び。**主食はインデックス**」と、タイトルの個別株関心を最後にインデックスへ着地させる。
- ※競合の「すでに持っている肯定」＋「乗り換えでなく検討材料」＋「コアは分散」を、着地に集約してチャンネルの王道形にする。

**ブロック19：比率の権威付け＋「持たない＝リスク」（行動誘導）**
- 日本株比率は株式部分の【★30〜50%＝GPIF・企業年金で権威化、台本化時に数値確認】を目安に、年齢・目的で幅。
- 「日本株を全く持たない＝知らないうちに為替リスクを取り続けている。終戦で追い風が吹く局面で、**持たないことの方がリスク**」。
- ※「もう下がらない」と断定せず、“持たないリスク”で同強度の背中押し。

**ブロック20：まとめ・判断を返すクロージング＋★円環回収**
- 一言でまとめると、動いているのは“戦争”ではなく、**終戦プロセスが原油と金利を通じて日本株を再評価させている**こと。
- ★冒頭に戻す円環：「冒頭で見た村田もJALもトヨタも、結局ぜんぶ“原油と金利”で動いた。**次に中東ニュースが出たら、見るべきは戦況の映像じゃなく、原油と金利です**」。
- 「ニュースは毎日変わる。でも、インデックスの積立ルールは毎日変えなくていい。ご自身のリスク許容度と年齢で、日本株比率を見直す判断材料に」で締め。
- ※冒頭の13銘柄→幹→着地で再び13銘柄、と1本の物語として閉じる。
