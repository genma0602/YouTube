# 分析レポート

## 0. 30秒サマリー（最初に結論）
- **核となるテーマ：** 「金融資産別の人生攻略ロードマップ」。50万円未満→100万〜300万円→500万〜1000万円→1000万〜3000万円→5000万円以上という**6段階の階段**を作り、「自分は今どこにいて、次に何をすればいいか」を一本道で示す**ステージ攻略型ロードマップ**。各段階に必ず「3つのステップ」を置き、最後は「資産額より持ち方／お金を人生に戻せ」と哲学着地させる。
- **最大の強み：** **視聴者が必ずどこかの段階に当てはまる「資産額の階段」設計**【導入L4の金額列挙／各品書き】。誰が見ても「自分はここだ」と自分ごと化でき、しかも上の段が見えるので「次のステージを見たい」という前進欲で完走を促す。さらに「同じ300万円でも意味が違う（生活防衛費／車／NISA）」【品書き1】という**役割分解**で序盤から納得感を作る。
- **最大の弱点／改善余地：** 速報性・時事フックがゼロで、**冒頭の数字インパクトが弱い**。階段を1段ずつ丁寧に上るため**中盤（100万〜300万、500万〜1000万）が構造的に似た反復**になり中だるみしやすい。サムネの煽り「人生勝ち組」に対し、本編は「勝ち負けではない」と逆を説くため、**サムネと結論の温度差**がある。
- **総合評価（高／中／低）：** **高**（時事に依存しない"永久ストック型"テーマ×全視聴者の自分ごと化×ステップの再現性が噛み合った教養ロードマップの完成形）
- **一言で：** この台本は「自分は今いくらの段階か」を「次のステージへ進む3ステップ」で引っ張り、「資産額より持ち方＝お金を人生に戻せ」で締める設計。

## 3. 視聴維持の設計 ［軸1×軸2：どこで見続け、どこで離脱するか］

### 3-1. 冒頭 0〜90秒（最重要）
| 項目 | 内容 |
| --- | --- |
| 最初の一文の役割 | 「今日は金融資産別 人生攻略ロードマップを解説していきます」【導入】＝動画の地図を即提示。掴みは弱いが約束は明確 |
| 冒頭で作っている不安・疑問 | 「自分は今どの段階か」「同じ1000万円でも人生を楽にする方と不安を増やす方がある」【導入】＝自分の現在地と持ち方への疑問 |
| 続きを見たくなる理由 | 「自分は今どこにいて、次に何をすればいいのか見えてくるはず」【導入】＝現在地特定＋次の一手という実利の予告 |
| 自分ごと化している要素 | 50万〜5000万の金額列挙【導入】で全視聴者がどこかに該当。「順番を間違えないこと、持ち方を整えること」 |
| 離脱を防いでいる仕掛け | 「本日の品書きは全部7つ」【品書き】で全体マップとゴールを提示 |

**冒頭の型（「高」は1つに絞る）：** 自分ごと化
→ 該当する型と理由：速報数字や危機で殴る型ではなく、「50万〜5000万」の金額を並べて**全視聴者を必ずどこかの段に立たせる**ことで「これは自分の話だ」と確定させる導入。「同じ1000万円でも人生を楽にする1000万円と不安を増やす1000万円がある」【導入】という二項対比で、金額だけでは決まらないという核を冒頭で握らせる。

### 3-2. 中盤以降の維持と離脱リスク
| 箇所（パート／時間目安） | 維持を生む要素 or 離脱リスク | 理由 |
| --- | --- | --- |
| 品書き1 お金の5つの役割・3原則 | 維持：全編の土台 | 「同じ300万円でも生活防衛費／車／NISAで意味が違う」が具体的で、以降の判断軸を与える |
| 品書き2 50万円未満「貧困の罠」 | 維持：強い共感の山 | 「お金がないと安く済むのが逆に高くつく」「意志ではなく仕組みの問題」と当事者を責めず救う設計で感情が動く |
| 品書き3 100万〜300万円 | **離脱リスク**：品書き2と構造が似る | 「守りが中心→分ける→3ステップ」の型が反復で、下層に該当しない視聴者には既視感 |
| 品書き4 投資額100万円の壁 | 維持：盛り返し | 「100万円超で10%＝10万円動く」「もっと早く増やしたい誘惑」と心理面の新しい論点で再加速 |

### 4-2. 数字の体感化
- できている例（台本内）：**下落を金額で実感させる**＝「投資額1000万円で30%下落したら-300万円、3000万円だと-900万円、5000万円なら-1500万円」【品書き5・6】と段階ごとに損失額を提示し「増える力が大きい＝減る力も大きい」を腹落ちさせる。さらに「投資リターン30万円×10%＝3万円 vs 固定費月1万円削減＝年12万円」【品書き3】と**投資より家計改善が効く**ことを並列の数字で示す。
- できていない／惜しい箇所：各段階の「生活費1ヶ月分」「3〜6ヶ月分」が**具体的な金額イメージ（例：手取り25万なら〇万）に落とし込まれていない**。視聴者が自分の数字に変換する一手間が残る。

### 4-3. 論理チェック
- どの情報から、どの結論につなげているか：「お金には5つの役割・3原則がある」【品書き1】→「だから同じ資産額でも持ち方で意味が変わる」→各段階で「守り→育てる→使う」の比重を移す→「最終的に資産額より持ち方／お金を人生に戻す」【まとめ】という一貫した演繹。
- 納得感が高い部分：「貧困の罠＝意志ではなく仕組みの問題」【品書き2】で自己責任論を排し共感を作る流れ。「無利資産は精神安定剤＝暴落時に株を売らずに済む価値」【品書き5】という機能の再定義。
- 飛躍して見える部分：「1000万円を作れた人はたまたまではない＝強い習慣を持っている」【品書き5】は相続・退職金で得た層には当てはまらず、一般化がやや強い（本人も「もちろん相続等もある」と補足はする）。
- 触れられていない重要論点：年齢・家族構成・収入水準による段階スピードの違い（冒頭で「状況は違う」と一言触れる程度）、具体的な無利/有リスク資産の比率の目安、インフレ率と必要資産額の関係。
- 視聴者に誤解を与えやすい表現：サムネ「この金額超えれば人生勝ち組だ」と本編「資産が多い人が偉いわけではない／人と比べるゲームから降りろ」【まとめ・品書き6】の方向が逆。クリック後に「煽りと違う」と感じる層がいうる一方、ギャップ自体が満足度になる設計とも読める。

### 7-2. 再現すべき勝ちパターン（次回への持ち帰り）
- 構成：**「金額（または年齢・年収）でN段階の階段を作り、各段に必ず同じ"3ステップ"を置く」ロードマップ型**。視聴者を必ずどこかの段に立たせ、上の段を「未来の自分」として見せて前進欲で完走させる。
- 冒頭の作り方：**対象レンジを冒頭で全列挙して「自分はどこか」を確定させる自分ごと化導入**。速報がなくても「現在地特定＋次の一手」の実利予告で引ける。
- 感情フック：**「不安のゴールポストが動く」＝増やしても不安が消えない図星**を、頑張る視聴者ほど刺さる中盤の山場に置く。最下層には「意志ではなく仕組みの問題」と共感で寄り添う。
- 情報の見せ方：**「同じ◯◯万円でも中身で意味が反転する」比較**（生活防衛費／車／NISA、人生を守る1000万 vs 不安を増やす1000万）で、金額の額面ではなく"持ち方"を主役にする。下落は必ず金額（-300万/-900万）で体感させる。
- コメント誘発要素：**「自分は今いくらの段階」と現在地を報告させる射幸ワード（勝ち組ライン）**。賛否と自己開示が同時に起きる。

### 7-3. 改善点（優先順）
1. **中盤2段（100万〜300万／500万〜1000万）の反復構造を圧縮・差別化する**。同じ「守り→分ける→3ステップ」が続き既視感が出るため、各段に固有の"1つの新概念"（例：100万円の壁、お金に名前をつける）をより前面に立てて段ごとの色を強める。
2. **冒頭の引きを強化する**。「あなたが何段階目かで人生の難易度は〇倍変わる」など、自分ごと化に数字インパクトを一滴足すと穏やかな出だしを補える。
3. **各段の金額目安を生活実感に変換する**。「生活費1ヶ月分」を「手取り◯万なら◯万円」と例示し、視聴者が自分の数字に直結できるようにする。

### 7-4. 一言結論
> この台本は「金融資産別 人生攻略ロードマップ」というテーマを「自分は今どの段階で、次に何をすべきか」という視聴者心理に接続し、「6段階の階段×各段の3ステップ→資産額より持ち方／お金を人生に戻せ」という自分ごと化×ステップ構成で最後まで引っ張っているため、YouTube台本として【強い】。

### 新たに発見された技法・パターン
- **「金額（レベル）でN段階の階段を作り、視聴者を必ずどこかに立たせる」全レンジ自分ごと化フック**。時事フックがなくても全視聴者を当事者にできるストック型の強力テンプレ。
- **各段に必ず同じ"3ステップ"を反復配置**し、フォーマットの統一で「次の段も同じ枠で学べる」安心感と予測可能性を与える。

### 自チャンへの蓄積メモ（今回新たに真似できる要素）
- **「資産額／年齢／年収でステージ分けし、各ステージに3ステップ」のロードマップ型**は、時事に左右されず長期再生を狙える鉄板テンプレ。自チャンの定番企画として持てる。
- **冒頭で対象レンジを全列挙して「自分はどこか」を確定させる**導入は、速報がない回でも自分ごと化で引ける。
- **「同じ◯◯万円でも持ち方で意味が反転」「下落は必ず金額で体感（-300万/-900万）」**は、インデックス推し・長期投資推しの自チャン文脈で「だから役割分担・無利資産を持て」の論拠に直結。
- **「不安のゴールポストが動く」の図星**は、増やす系視聴者の刺さりが強く、コメント（現在地報告・賛否）を誘発する。自チャンでも「あなたは今いくら？」と現在地を問う設計に転用できる。
