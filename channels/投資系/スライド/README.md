# スライド（投資のすゝめ）

## ファイル構成

| ファイル | 用途 |
|---|---|
| `日経平均7万円突破.html` | 元データ（編集はここ）。ブラウザで開けば全画面プレビュー |
| `日経平均7万円突破.pdf` | **横送りプレビュー用**。全画面 → ←→キーで1枚ずつ送れる。どのPCでも開ける1ファイル |
| `PNG/slide_01.png …` | **動画編集用**。1枚ずつ・1920×1080(フルHD)。タイムラインに並べて声に合わせてカット |
| `template.html` | 各スライド型の見本（流用元） |
| `convert.py` | HTML → PDF → PNG の一括変換スクリプト |

## 使い方

### 撮影しながら台本を読むとき
`日経平均7万円突破.pdf` を開いて全画面表示 → **←→キー**で横に送る。

### 動画編集で声に合わせて動かすとき
編集ソフト（Premiere / DaVinci / CapCut など）に `PNG/` の画像をまとめて読み込み、
タイムラインに順番に並べて、ナレーションに合わせて表示時間を調整する。

### 別のPCで見る
`.pdf` をコピーすればOK（ソフト不要、ブラウザでも開ける）。
編集したい場合は `.html` をコピー。

## 作り直すとき（HTMLを編集したあと）

```
python convert.py "日経平均7万円突破.html"
```

PDFとPNGが上書き再生成されます。

### 必要なもの
- Google Chrome または Microsoft Edge
- Python ＋ PyMuPDF … `pip install pymupdf`

## サイズ
全スライド 16:9 / 出力PNGは 1920×1080（YouTubeそのまま）。
