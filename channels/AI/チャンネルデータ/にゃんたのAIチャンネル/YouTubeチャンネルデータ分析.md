# にゃんたのAIチャンネル YouTubeチャンネルデータ分析

## 1. このデータからわかること

このチャンネルは、単なるAIニュース紹介よりも、**「AIをどう使うか」「どのツールに課金すべきか」「Claude Code/Cursor/Codexなどを実務でどう使うか」** の方が強いです。

特に伸びているのは以下です。

| 動画タイトル | 再生回数 |
|---|---:|
| Claude Code完全入門 | 176,237 |
| ChatGPT/Claude/Geminiどれに課金すべき？ | 160,131 |
| Googleが発表したプロンプトを2回入れる手法 | 147,497 |
| GPT-4oについて徹底解説 | 140,418 |
| Difyを徹底解説 | 133,136 |
| AIがゴマすりをする理由 | 126,212 |
| Coworkの実践的な使い方 | 111,167 |

傾向として、**大型モデル発表だけを扱う動画は安定するが、爆発するとは限らない**です。

一方で、以下の要素が入ると再生が跳ねやすいです。

- 「完全入門」「実践入門」「どれに課金すべき」など、初心者が判断に使える企画
- Claude Code / Cursor / Codex / Agent Skills など、実務直結のAI開発ツール
- プロンプト、AIの使い方、AIとの付き合い方など、視聴者自身の悩みに近いテーマ
- Dify / RAG / LangChain など、作れる・業務化できるテーマ

コメント率が高い動画は、再生数とは少し違う性質があります。  
`生成AIあるある`、`日本の生成AI利用率が低い`、`AIの忖度`、`AI時代の勉強方法` など、**視聴者が自分の意見を言いたくなるテーマ**はエンゲージメントが強いです。

## 2. タイトルから企画のパターン分類

### 大パターンA：AI開発ツール・コーディングエージェント系

中パターン：

- Claude Code実践・入門・便利機能
- Cursor / Copilot / Codex / Gemini CLI / Kiroなどの開発環境比較
- Agent Skills / SubAgents / Hooks / SDKなど自律ワークフロー
- 非エンジニア向けAI開発ツール活用
- AIエージェントの作り方・考え方

このカテゴリはチャンネルの現在の主力です。特に2025年以降、Claude Code系が強いです。

### 大パターンB：AIモデル・新機能ニュース解説系

中パターン：

- OpenAI新モデル・新機能
- Claude新モデル・Anthropic発表
- Gemini / Google発表
- DeepSeek / Mistral / Llamaなど注目モデル
- Sora / 画像生成 / 音声などモダリティ系

話題性はあるものの、平均はやや控えめです。  
ただし `GPT-4o`、`Gemini 2.0 Flash`、`Sora` のように市場インパクトが大きいものは伸びます。

### 大パターンC：プロンプト・AI活用術・思考法系

中パターン：

- プロンプト術・プロンプト研究
- AIの使い分け・課金判断
- AIとの付き合い方・勉強法・忖度対策
- 普段の使い方・情報収集法
- 音声入力・検索・業務活用

このカテゴリは強いです。  
特に「視聴者が明日から変えられる使い方」に落ちている動画が伸びやすいです。

### 大パターンD：Dify / RAG / LangChain / 業務アプリ構築系

中パターン：

- Dify入門・基本設定
- RAG構築・精度改善
- LangChain / LlamaIndex / PDF QA
- Knowledge Pipeline / Contextual Retrieval
- 業務アプリ・チャットボット作成

Difyの基本・入門系は非常に強いです。  
一方で、RAGの細かい改善手法になるとやや専門的になり、平均再生は下がります。

### 大パターンE：比較・課金・選び方系

中パターン：

- ChatGPT / Claude / Geminiの比較
- どのサービスに課金すべきか
- Perplexity / Gensparkなど検索AI比較
- Cursor / Copilotなど開発ツール比較
- 生成AIサービスの使い分け

このカテゴリは再現性が高いです。  
「結局どれを選べばいいの？」という悩みに答えるタイトルは強いです。

## 5. マネすべき動画

### 1. Claude Code完全入門

176,237再生。  
「完全入門」形式で、伸びているClaude Codeテーマを初心者向けに整理している。再現性が高いです。

### 2. ChatGPT/Claude/Geminiどれに課金すべき？

160,131再生。  
視聴者の直接的な悩みである「結局どれを選ぶべきか」に答えています。比較・判断系は強いです。

### 3. Googleが発表したプロンプトを2回入れる手法

147,497再生。  
論文・研究ネタを、すぐ使えるプロンプト術に変換しています。専門性と実用性のバランスが良いです。

### 4. Difyを徹底解説

133,136再生。  
ツールの基本入門は強いです。特にDifyのように「非エンジニアでも作れる」文脈は相性が良いです。

### 5. AIがゴマすりをする理由

126,212再生。  
技術解説だけでなく、視聴者が日常的に感じる違和感をテーマ化しています。コメント率も高いです。

## 結論

このチャンネルで次に狙うなら、**「Claude Code・Codex・Gemini CLIなどのAI開発ツール比較」** と **「課金判断・使い分け」**、そして **「AI活用の失敗パターンを研究や実体験から解説する企画」** が最も再現性高いです。
