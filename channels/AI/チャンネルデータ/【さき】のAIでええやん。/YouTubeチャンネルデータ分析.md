# 【さき】のAIでええやん。 YouTubeチャンネルデータ分析

作成日: 2026-05-13

対象チャンネル: **【さき】のAIでええやん。**

## 1. このデータからわかること

対象データは60本です。総再生回数は **4,138,906回**、平均再生回数は **68,982回**、中央値は **39,237回** です。平均より中央値がかなり低いため、一部の大ヒット動画がチャンネル全体の平均を大きく押し上げています。

最大の外れ値は、**【脱初心者】AIへの指示出しがうまくいくプロンプトの基礎徹底解説するで！** の **695,344回** です。登録者数106,000人に対して拡散率6.5598と非常に高く、単なるツール紹介よりも「AIをうまく使うための考え方・指示出し」の需要が強いことが分かります。

次に強いのは、Gemini / NotebookLM / Google系の保存版・神活用系です。**Gemini基本解説 266,080回**、**Gemini×NotebookLM最強活用 203,663回**、**NotebookLM裏ワザ 176,627回**、**最新版Geminiスライド作成 206,118回** など、Google系AIを「初心者でもすぐ使える」「仕事・資料作成に使える」形で見せた動画が安定して伸びています。

Claude系も強く、特に **Claude初心者向け最新基本活用 302,617回**、**Claude Code基礎と神活用 131,563回** が目立ちます。一方で、2026年4月後半のChatGPT基礎シリーズは1,000〜7,000回台が多く、短尺・細分化された基礎機能解説だけでは伸びにくい傾向があります。

このチャンネルで強い企画は、タイトルに **「初心者」「脱初心者」「基礎」「神活用」「最強」「革命」「資料作成」「NotebookLM」「Gemini」「Claude」** が入り、さらに視聴者の得が明確なものです。弱い企画は、単一機能の基礎解説だけで、成果物・損得・変化が見えにくいものです。

## 2. タイトルから企画のパターン分類

### 大パターンA：プロンプト・AI指示出し改善型

中パターン:

- 脱初心者系: `脱初心者`、`基礎徹底解説`
- 指示出し改善系: `AIへの指示出し`、`プロンプトのコツ`
- 考え方転換系: `プロンプトはもう古い`、`能力フル活用`
- 損失回避系: `大損`、`ガチ勢だけが使っている`

このパターンは最も再生上限が高いです。ツール名に依存せず、ChatGPT / Gemini / Claudeすべての利用者に刺さるため、視聴対象が広くなります。

### 大パターンB：Google / Gemini / NotebookLM神活用型

中パターン:

- Gemini基礎・完全マスター系
- NotebookLM裏ワザ・神機能系
- Gemini×NotebookLM神連携系
- Google AI Studio / Gem / Nano BananaなどGoogle周辺機能系
- スライド・資料作成への応用系

このチャンネルの主力パターンです。特に **Google最強AI**、**NotebookLM**、**Gemini** を組み合わせ、実務成果に落とす企画が強いです。

### 大パターンC：Claude特化・Claude仕事活用型

中パターン:

- Claude初心者向け基礎解説
- Claude Code / ClaudeCode活用
- Claude Cowork活用
- Claudeによる資料・スライド作成
- Claude新機能・Design連携

Claude系は爆発力があります。特に「初心者でも使える」「仕事が終わる」「神活用4選」のように、難しそうなツールを分かりやすく実務化するタイトルが強いです。

### 大パターンD：ChatGPT基礎・機能細分化型

中パターン:

- ChatGPTの基本的な使い方
- Web検索 / Canvas / Deep Researchなど単機能解説
- GPTs / カスタム指示など設定機能解説
- 画像生成・仕事活用など用途別解説

本数は多いですが、平均再生は低めです。単体機能を短く解説するだけだと、既視感が出やすくクリックの動機が弱くなります。

### 大パターンE：業務効率化・成果物作成型

中パターン:

- パワポ・スライド資料作成
- Excel / スプレッドシート効率化
- 全自動資料作成ツール比較
- 業務の無駄作業自動化
- 仕事が秒で終わる系

「AIで何ができるか」ではなく「仕事の成果物がどう変わるか」を見せる型です。再現しやすく、自チャンネルでも応用しやすいです。

### 大パターンF：他AIツール・比較紹介型

中パターン:

- Manus / Copilot / Opalなどの基礎解説
- Gensparkなどの自動化ツール紹介
- おすすめAIツール厳選
- ChatGPT / Geminiなどの比較解説
- 最新AIまとめ系

平均は中程度です。ツール単体紹介よりも、比較・ランキング・用途別おすすめにした方が伸びやすいです。

## 3. パターンごとの平均再生回数

大パターン別では、**プロンプト・AI指示出し改善型** が平均154,268回で最も強いです。次に **Google / Gemini / NotebookLM神活用型** が平均99,903回、**Claude特化・Claude仕事活用型** が平均75,063回です。

ChatGPT基礎・機能細分化型は14本で合計128,173回、平均9,155回に留まります。これはChatGPTの需要が低いというより、タイトル上の切り口が「基礎機能の説明」に寄りすぎると、視聴者の得が弱く見えるためです。
