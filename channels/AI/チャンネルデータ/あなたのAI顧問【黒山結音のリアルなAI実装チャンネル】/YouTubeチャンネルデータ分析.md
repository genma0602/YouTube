# YouTubeチャンネルデータ分析

対象チャンネル：あなたのAI顧問【黒山結音のリアルなAI実装チャンネル】

## 1. このデータからわかること

一番強いのは、単なるAIニュースではなく、**「無料」「初心者でも学べる」「2026年に備える」「実際に使える」系の実用企画**です。

最大の外れ値は以下です。

| 動画 | 再生数 | 拡散率 | 特徴 |
|---|---:|---:|---|
| 【無料】GoogleがAIスキルを学べる講座『Google Skills』 | 73,990 | 7.55 | 無料×学習×Google×2026年 |
| 【日本初】大バズり中のAIショート動画を10分で作る方法 | 25,985 | 2.65 | 日本初×バズ×短時間で作れる |
| 【新企画】全アナログ業務を1日で撲滅 | 18,407 | 1.88 | 実務改善×密着×変化が見える |
| Claudeのデータ移植機能 | 15,842 | 1.62 | 速報×実務インパクト |
| Claudeの進化が速すぎる | 13,732 | 1.40 | 強い感情ワード×Claude需要 |

逆に弱いのは、**抽象的な感想系ショート**、**採用告知**、**細かすぎる不具合解説**、**新機能でも視聴者の得が弱いもの**です。

ただし、低再生でもコメント率・高評価率が高い動画はあり、濃いファン向けテーマとしては機能しています。

## 2. タイトルから企画のパターン分類

### 大パターンA：無料・保存版・学習コンテンツ

中パターン：

- 無料講座・無料配布
- 初心者向け学習ロードマップ
- プロンプト・テンプレ配布
- 2026年に備える系
- Google / OpenAI / Manusなど大手の学習資産紹介

このパターンは最も再生上限が高いです。特に「無料」「学べる」「初心者」「2026年」が揃うと強いです。

### 大パターンB：速報・新機能・新モデル解説

中パターン：

- Claude / ChatGPT / Geminiの新モデル
- Office・Gmail・Slackなど業務ツール連携
- 画像生成・動画生成などクリエイティブAI
- AIエージェント系アップデート
- ベンチマーク・比較系

安定して再生されますが、単なる速報よりも「実務で何が変わるか」まで入る動画の方が強いです。

### 大パターンC：実演・業務改善・自動化

中パターン：

- 営業・メール・請求書・議事録の自動化
- 企業密着・業務改善ドキュメント
- AI社員・AI組織化
- コピペ可能なGAS / プロンプト配布
- 経営者・会社員向け生産性改善

爆発力があります。特に「Before→After」がタイトルで見える企画が強いです。

### 大パターンD：危険・警告・残酷な現実

中パターン：

- AIを使えない人への警告
- 情報漏洩・偽サイト・セキュリティ
- 課金しないリスク
- AI活用ミスへの警告
- 仕事がなくなる系

再生は中〜高水準。強い言葉でクリックは取れますが、内容が具体的でないと伸び切りません。

### 大パターンE：Claude / OpenClaw特化

中パターン：

- Claude Code / Cowork / Chrome / Excel
- OpenClawの料金・エラー・設定
- Claude関連サービスまとめ
- Claude vs ChatGPT / Gemini
- 開発者向け使い方

チャンネルの核になっているテーマです。視聴者との相性は良いですが、細かい技術トラブル系は再生数が小さくなりやすいです。

### 大パターンF：ショート・一言感想系

中パターン：

- 「これ便利」「やばい」系
- 長尺動画への誘導
- トレンドAIの短尺紹介
- ハッシュタグ型AIニュース
- 感想・リアクション型

再生数はばらつきます。伸びるものもありますが、長尺と比べるとタイトルだけでは価値が伝わりにくいです。

## 3. パターンごとの平均再生回数

目視分類ベースの概算です。ショートと長尺が混在しているため、特に外れ値の影響が大きいです。

| 大パターン | 件数目安 | 合計再生数目安 | 平均再生数目安 | 評価 |
|---|---:|---:|---:|---|
| 無料・保存版・学習 | 約18本 | 約119,000 | 約6,600 | 最強。外れ値を生む |
| 速報・新機能・新モデル | 約40本 | 約140,000 | 約3,500 | 安定して強い |
| 実演・業務改善・自動化 | 約20本 | 約60,000 | 約3,000 | 当たると大きい |
| Claude / OpenClaw特化 | 約25本 | 約72,000 | 約2,900 | コア視聴者向けに強い |
| 危険・警告・残酷な現実 | 約10本 | 約28,000 | 約2,800 | クリックは取れる |
| ショート・一言感想系 | 約35本 | 約62,000 | 約1,700 | ばらつき大 |

重要なのは、**平均だけで見ると「無料・学習」が圧勝**ですが、これはGoogle SkillsやAIショート動画のような大当たりに引っ張られています。

再現性で見るなら、次の順番です。

| 優先度 | 狙うべき型 | 理由 |
|---:|---|---|
| 1 | 無料×初心者×大手AI学習 | 爆発力が最も高い |
| 2 | 実演×業務改善×Before / After | チャンネルの信頼に直結 |
| 3 | 速報×実務インパクト | 安定して再生される |
| 4 | Claude特化×具体手順 | コア層の満足度が高い |
| 5 | 危険・警告×解決策 | クリックは強いが乱用注意 |

## 5. マネすべき動画

### 1. 【無料】Google Skills解説

再生数が圧倒的です。

「無料」「Google」「AIスキル」「2026年」という強いフックが全部入っています。今後もこの型は再現価値が高いです。

### 2. 【日本初】AIショート動画を10分で作る方法

「日本初」「大バズ」「10分で作る」が強いです。

視聴者にとって成果物が明確なので、クリック理由が強いです。

### 3. 【新企画】全アナログ業務を1日で撲滅

このチャンネルの独自性が最も出ています。

AIニュースではなく、黒山さん自身の実装力がコンテンツになっています。

### 4. Claudeの進化が速すぎる系

Claude需要とチャンネルの相性が良いです。

ただし今後は「すごい」だけでなく、「会社員・経営者が今日から何に使うか」まで入れるとさらに強いです。

### 5. Genspark / Manus / NotebookLMの実演系

ツール紹介単体より、実際に成果物を作る動画が強いです。

「資料作成」「営業リスト」「マニュアル作成」「メール返信」など業務に直結させるのが良いです。

## 結論

このチャンネルは **「AIニュースを解説するチャンネル」よりも、「AIを実務に落とし込んで、視聴者の仕事を減らすチャンネル」** として伸ばす方が強いです。

次に狙うなら、**無料学習系 × 実演 × 業務改善** の掛け合わせが一番再現性高いです。
