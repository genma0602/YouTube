# 04. 勝ち筋サマリーと再現テンプレ

作成日: 2026-05-13

`03_伸びる要素の共通項.md` の結論を圧縮し、**「これさえ守れば伸びやすい」勝ち筋を5型** に整理しました。各勝ち筋にタイトル構造のテンプレ、適用ツール、適用用途、注意点を付けています。

新企画案は本書には含めません（共通項の凝縮に集中）。

---

## 勝ち筋を貫く5原則（全テンプレ共通）

| # | 原則 | 違反すると |
|---:|---|---|
| 1 | **タイトル冒頭の【フック】は必ず付ける** | クリック率が落ちる |
| 2 | **ツール名 or 上位概念（プロンプト・習慣・リサーチ等）を必ず入れる** | 検索流入が取れない |
| 3 | **ベネフィット（実利／成果物／時短）を必ず明示する** | 視聴者の得が伝わらない |
| 4 | **「数字」か「対象」のどちらかは必ず入れる** | 再現性が落ちる |
| 5 | **強い形容（神／最強／革命）は2語まで** | 過剰になり信頼が落ちる |

# 勝ち筋1：保存版・完全解説型

## なぜ強いか
- **10ch中10chで主力**。横断ヒット動画47本（10万回超）のうち約25本以上が該当
- 再生数の上限は約30万回、平均は10万〜15万回が目安
- 検索流入と保存需要を同時に取れる、最も再現性が高い型

## タイトル構造テンプレ

```
【完全○○】[対象] が [ツール名] を [動作] する方法を [動詞] [数字]
```

### 穴埋めパーツ
- **完全○○**: 完全版／完全ガイド／完全解説／完全マスター／完全入門／完全攻略／完全理解／徹底解説／保存版／決定版／神回
- **対象**: AI初心者／脱初心者／会社員／管理職／30代以上／非エンジニア／（省略可）
- **ツール名**: ChatGPT／Gemini／NotebookLM／Claude／Claude Code／Copilot
- **動作**: 使いこなす／業務効率化／自動化／資料作成／プロンプト改善
- **動詞**: 全部教える／徹底解説する／完全マスターする／この1本で理解する
- **数字**: ○つの／○選／TOP◯（任意）

### 実例（既存ヒット）
```
【完全マスター】絶対に覚えるべきGeminiの基礎を全部教えます (299,075)
【神回】最強のGoogle無料AI「NotebookLM」の機能と使い方をこの動画1本で完全理解 (268,273)
【保存版】ChatGPT 2025年最新入門 (236,512)
【超完全入門】Copilotの特徴・機能・活用法を完全解説 (100,295)
Claude Code完全入門 (176,237)
```

## 適用条件
- **強いツール**: NotebookLM／Gemini／Claude／Claude Code／ChatGPTのいずれか
- **強い用途**: 基礎・全機能解説／業務自動化／プロンプト改善／資料作成
- **弱いツール**: 認知度の低い単体ツール（Manus、Genspark単独など）

## 注意点
- **更新頻度に注意**：ツールが進化するため、ChatGPT入門・Gemini入門は半期〜年単位で出し直すべき
- **完全○○を入れたら本当に網羅すること**：タイトルだけ「完全」で中身が薄いと信頼が落ちる

# 勝ち筋2：ランキング・厳選TOP・○選型

## なぜ強いか
- **採用ch 10/10**、横断ヒットで最多級
- 「TOP◯」「○選」が入る動画はヒット動画の70%以上
- 視聴者の選択コストを下げ、保存・共有されやすい

## タイトル構造テンプレ

```
【強い形容 or 損失回避フック】[ツール名 or 上位概念] の [動作/便利機能/活用法] [TOP◯ / ○選]
```

### 穴埋めパーツ
- **強い形容 or 損失回避フック**: 神／最強／決定版／厳選／知らないと損／知らない人多すぎ／使わないと損／早く知りたかった
- **ツール名 or 上位概念**: NotebookLM／Gemini／ChatGPT／Claude / プロンプト／AI習慣／AIリサーチ
- **動作/便利機能/活用法**: 便利な使い方／活用法／神機能／裏ワザ／神連携／NG行為
- **数字部分**: TOP3／TOP5／TOP7／TOP10／3選／5選／7選／10選／19選／20選

### 実例（既存ヒット）
```
【決定版】AIガチ勢が毎日やってる習慣TOP10 (703,582)
【知らない人多すぎ】作業効率が劇的に上がる最新ChatGPT活用術20選 (555,497)
【Googleの神AI】「NotebookLM」厳選活用法TOP7 (300,425)
【厳選】知らないともったいない！ChatGPTの便利な使い方 TOP10 (269,163)
【使わないと損】たった「1行」でAIを思い通りに動かす「最強プロンプト」TOP5 (165,778)
```

## 適用条件
- **強いツール**: 全ツール対応
- **強い用途**: 活用法／神機能／裏ワザ／NG行為／設定／使い方
- **数字の選び方**:
  - **TOP3/5**: 結論を急ぐ視聴者向け（ショート向き）
  - **TOP7/10**: 保存版的に幅を出す（長尺向き）
  - **20選/25個/30種**: 「網羅性アピール」で完全解説型と融合

## 注意点
- **数字の信頼性**：「TOP10」と言ったら本当に10個出すこと
- **数字を入れただけでは弱い**：必ず「強い形容 or 損失回避フック」と組み合わせる

# 勝ち筋3：業務効率化・実演・自動化型

## なぜ強いか
- **「成果物が見える」ため自分ごと化されやすい**
- いけとも・KEITO・ダイさんで平均23,000〜60,000回の安定帯
- 上限は55万回（ChatGPT活用術20選）まで伸びる

## タイトル構造テンプレ

```
【強い形容 or 実演明示】[ツール名] で [具体作業] を [効果量] にする方法
```

### 穴埋めパーツ
- **強い形容 or 実演明示**: 実演あり／実演解説／革命的／必見／衝撃／神アプデ／2026年最新
- **ツール名**: ChatGPT／Gemini／Claude／Claude Code／Copilot／NotebookLM
- **具体作業**: Excel／資料作成／スライド／メール返信／議事録／マニュアル作成／動画編集／Webサイト作成／請求書／マクロ／VBA
- **効果量**: 生産性10倍／3倍／一瞬で／秒で／○分で／全自動／自動化／爆速

### 実例（既存ヒット）
```
AIで機能付きWEBサイト出来るの凄い (345,575)
【神アプデ】最新版Geminiでパワポ・スライド資料作成が一瞬で終わる方法 (206,118)
【革命的】AIで動画編集が全自動（Claude Code） (205,039)
【実演あり】ExcelでAI活用して誰でも簡単に生産性を10倍以上にする方法 (187,945)
【必見】ExcelマクロをChatGPTで自動作成し、ルーティン作業を自動化 (112,554)
```

## 適用条件
- **強い作業領域**: Excel／資料作成・スライド／動画編集／Webサイト／メール・議事録／マニュアル作成
- **弱い作業領域**: AI作曲／旅行計画／単発のスマホTips
- **対象明示の追加で強化可**: 会社員／管理職／非エンジニア

## 注意点
- **タイトルだけ「自動化」と書いて中身が手動だらけだと失敗する**
- **成果物（資料／Webサイト／動画）はサムネでも一目見えるとよりクリック率が上がる**

# 勝ち筋4：プロンプト・AIリテラシー改善型（爆発帯）

## なぜ強いか
- **再生上限が最も高い**（さき 695,344回、本気AI 703,582回など）
- ツール非依存のため、ChatGPT/Gemini/Claudeの全ユーザーが対象＝母数が広い
- さき・本気AI・にゃんたで chトップ平均

## タイトル構造テンプレ

```
【対象明示 or 損失回避】[AI / プロンプト / ガチ勢 / 指示出し / リサーチ] の [テーマ] を [動詞] [数字]
```

### 穴埋めパーツ
- **対象明示 or 損失回避**: 脱初心者／知らないと損／後悔する／今さら聞けない／使わないと損／AI下手必見
- **テーマ（ツール非依存）**: プロンプトの基礎／指示出し／AI習慣／AIリサーチ／AIへの偏見／AIがバカになる使い方／天才にさせる
- **動詞**: 徹底解説する／全部教える／変える／引き出す
- **数字部分**: 1行で／TOP5／TOP10／5選

### 実例（既存ヒット）
```
【脱初心者】AIへの指示出しがうまくいくプロンプトの基礎徹底解説 (695,344)
【決定版】AIガチ勢が毎日やってる習慣TOP10 (703,582)
【早く知りたかった】ChatGPTを賢くする魔法の言葉 (516,653)
【今さら聞けない】AIリサーチの超基本 (203,417)
【使わないと損】たった「1行」でAIを思い通りに動かす「最強プロンプト」TOP5 (165,778)
Googleが発表したプロンプトを2回入れる手法 (147,497)
```

## 適用条件
- **最強の組み合わせ**: 上位概念（プロンプト／習慣／リサーチ）× ガチ勢 or 脱初心者 × TOP◯
- **弱い場合**: 単体小技「○○の使い方1つ」だけだと埋もれる

## 注意点
- **「ガチ勢」「AIオタク」を使うときは権威性の裏付けが必要**：実演や検証で説得力を持たせる
- **AI論文・研究ネタは強い**：「Googleが発表した」「ハーバード論文」など権威タグも効く

# 勝ち筋5：神連携・複数ツール組み合わせ型

## なぜ強いか
- **NotebookLM × Gemini の組み合わせは複数chで安定して10万回超**

- さき・ダイさん・トモミツでヒット連発

## タイトル構造テンプレ

```
【神連携 or 強い形容】[ツールA] × [ツールB] で [具体成果物 / 業務] を [効果量]
```

### 穴埋めパーツ
- **神連携 or 強い形容**: 神連携／最強連携／衝撃／革命／禁断テクニック／神コラボ
- **ツールA × ツールB**: NotebookLM×Gemini / Gemini×NotebookLM / NotebookLM×Claude / Gemini×NanoBanana / Gemini×Excel
- **具体成果物**: スライド資料／議事録／メール／調査レポート／マニュアル
- **効果量**: 自動化／爆上げ／一瞬で／○倍

### 実例（既存ヒット）
```
【AI神連携】これがNotebookLMとGeminiの最強活用 (203,663)
【衝撃】NotebookLM + Gemini 2.5 Pro 業務生産性爆上げ禁断テクニック (90,038)

Geminiは単体で使うな！仕事が劇的に変わる『神連携』機能 TOP 5 (50,289)
【2026年最新】AIスライド作成が革命的に変わる神連携 (38,825)
```

## 適用条件
- **最強の組み合わせ**: NotebookLM × Gemini（複数chで実績）

## 注意点
- **両ツール共に視聴者が知っている前提**：片方が無名だと検索が取れない

# 勝ち筋の融合（さらに強くする）

5つの勝ち筋は **単独でも強いが、融合すると爆発する** ことが分かっています。

## 融合パターンA：保存版 × ランキング

> 完全○○ + TOP◯／○選

実例：
- `【完全解説】20以上の便利機能をこの動画1本で全て解説` (161,357)
- `【厳選】知らないともったいない！ChatGPTの便利な使い方TOP10` (269,163)
- `【決定版】AIガチ勢が毎日やってる習慣TOP10` (703,582)

**最も再現性高い融合パターン**。

## 融合パターンB：業務効率化 × ランキング

> 業務／実演 + ○選／TOP◯

実例：
- `【知らない人多すぎ】作業効率が劇的に上がる最新ChatGPT活用術20選` (555,497)
- `【30代以上は必見】GeminiとNotebookLMを管理職の仕事を効率化6選` (65,123)
- `【実演あり】CopilotだけできるAI仕事術5選` (32,980)

## 融合パターンC：神連携 × 業務効率化

> 神連携 + 成果物明示 + 効果量

実例：
- `【衝撃】NotebookLM + Gemini 2.5 Pro 業務生産性爆上げ禁断テクニック` (90,038)
- `【AI神連携】これがNotebookLMとGeminiの最強活用` (203,663)

## 融合パターンD：プロンプト改善 × 損失回避

> 脱初心者／知らないと損 + プロンプト

実例：
- `【脱初心者】AIへの指示出しがうまくいくプロンプトの基礎徹底解説` (695,344)
- `【使わないと損】たった「1行」でAIを思い通りに動かす「最強プロンプト」TOP5` (165,778)

## 融合パターンE：保存版 × プロンプト改善 × 損失回避

> 完全○○ + プロンプト + 知らないと損

このパターンは現時点で実例が少なく、**今後狙える未開拓領域** の可能性。

# 横断スコアカード（自社企画チェック用）

新企画を作るとき、以下の項目を埋めてスコアを確認すると、伸びる確率が見える化できます。

| チェック項目 | あり=1点 | 解説 |
|---|:---:|---|
| 冒頭に【フック】がある | □ | 必須 |
| ツール名 or 上位概念がある | □ | 必須 |
| ベネフィットが明示されている | □ | 必須 |
| 数字（TOP◯／○選／○倍／○分）がある | □ | 推奨 |
| 対象（初心者・会社員・管理職など）がある | □ | 推奨 |
| 強い形容（神／最強／革命）が1〜2語ある | □ | 推奨 |
| 損失回避フック（知らないと損・NG・危険）がある | □ | 任意 |
| 「完全○○」「徹底解説」がある | □ | 保存性 |
| 成果物が見える（資料/動画/Webサイト/Excel） | □ | 業務効率化型なら必須 |
| ツール × 用途の組み合わせが C-3 の◎に該当 | □ | 必須 |

**目安**:
- **7点以上**: ヒット候補（10万回超期待）
- **5〜6点**: 安定帯（数千〜数万回）
- **4点以下**: タイトル再設計が必要

# 守るべき禁則（弱い動画への共通項から）

| 禁則 | 理由 |
|---|---|
| 機能名だけのタイトル | 視聴者の得が見えない |
| 単発の小技を独立動画にする | 範囲が狭く再生上限が低い |
| 抽象的な「すごい」「ヤバい」だけ | 中身が想像できずクリックされない |
| 副業／稼げる系の一般論 | 視聴者の不信感が強い |
| 速報のみで活用なし | 一過性で伸びない |
| アプリ別Tipsを独立動画にする | 「業務別まとめ」に束ねるべき |
| 強い形容を3つ以上重ねる | 信頼性が落ちる |
| 数字を盛ってタイトル詐欺 | コメント・低評価が増える |

# 最終要約：1行で言う「勝ち筋」

**「ChatGPT／Gemini／NotebookLM／Claude（または Claude Code）を、初心者にもわかる完全○○ × TOP◯ × 業務効率化／自動化／神連携 で見せ、損失回避フックで入口を作る動画。」**

これが10チャンネル横断で観察された、AI系YouTubeで最も伸びる動画の共通項です。
