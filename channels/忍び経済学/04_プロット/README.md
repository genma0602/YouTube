# プロット

各動画の**構成案・プロット**（導入→展開→締めの骨子、話の流れ）を置くフォルダです。  
**工程では台本執筆の直前**です。先に `03_企画リサーチ` で調査し、このフォルダで構成を固めてから `05_台本執筆` で本文・修正に入ります。

## ファイル名の例

- `新企画_[テーマ]_プロット.md` … 1本につき1ファイル
- テーマが分かるよう、国名・政策名・キーワードをファイル名に含める

## このフォルダと前後の工程（順番どおり）

| 順番 | フォルダ | 内容 |
|:----:|----------|------|
| 1 | `03_企画リサーチ` | 調査・根拠・企画メモ |
| 2 | `04_プロット`（ここ） | 構成・プロット |
| 3 | `05_台本執筆` | 台本本文・修正指示 |
