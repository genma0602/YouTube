## 採用した型の根拠

### この企画の核：「常識を壊す」のはどこか

サムライ経済学が最も伸びる瞬間は**「なんとなく持っていた常識が数字で壊される瞬間」**（横断分析より）。

この企画で壊せる常識は1つに絞られる。

> **「ドイツは環境のために原発を止めた」→「でも実際にはCO2が増えた」**

視聴者が持っている「脱原発＝エコ・環境に良い」という常識を、データで静かに破壊する。これが台本全体の中心軸。

### なぜ「告発エスカレーション型 × 請求書の開封」か

- **告発エスカレーション型**（ドイツ台本で最も伸びた型）: 問題が層ごとに積み上がり「まだあるの？」の驚きが持続する。
- **請求書の開封**というメタファーを軸にすることで、視聴者は「次の請求書には何が書いてある？」というサスペンスで最後まで引っ張られる。
- 「封筒を1枚ずつ開ける」という映像的な軸があるため、初見の視聴者でも構成を見失わない。

## この動画の狙い（ライター向け要約）

**一言**: 「脱原発＝エコ」という常識を、4枚の請求書で静かに壊していく動画。

**構造**:

```
Layer 1（財布）:  産業用0.19ドル/kWh→米国比2.4倍 / 570億ユーロ         ←「予想通り」の請求書。クラウスの実感
                   ↓「でもお金の問題だけじゃなかった」
Layer 2（CO2）:   「エコのつもりが」→ 石炭で代替 → CO2追加7.3億トン   ←常識破壊の核心
                   ↓「しかもまだ終わらなかった」
Layer 3（命）:    早期死亡1万9,200人 / 重篤疾患17万7,000人              ←最大の衝撃・感情的クライマックス
Layer 4（システム）: 純輸入国転落 / 系統崩壊 / 1,280億ユーロ           ←構造崩壊
→ 逆転：「過ちだった」とメルツ首相 ＋ 世界のUターン（ベルギー・日本）
→ 日本ブリッジ：福島→全炉停止→Uターンの15年
```

**感情の流れ**:
驚き（EU1位）→ 疑問（なぜ？）→ 歴史（決断の背景）→ 怒り（財布） → **常識破壊（「エコのつもりが石炭だった」）** → 最大衝撃（命） → 構造崩壊（システム） → 逆転の驚き（「過ちだった」） → 危機感（日本） → 余韻

**CQ**: 「環境のために原発を止めた。その請求書には、何が書かれていたのか？」

**トーン**: 「原発推進の動画じゃない。請求書の中身を確認するだけだ」の予防線を冒頭に1文置く。感情的に叫ばない。データで語る。怒りは語り手自身の「自分も信じていた選択の裏側を見た苦さ」として表現する。

### Block 1：冒頭フック ＋ CQ（0:00〜1:30 / 400字）

**機能**: 「電気代の請求書」という日常の物体からスタート。最初の30秒は**クラウスの個人体験**から入る（必須要素ルール2）。数字マシンガンで「EU1位」の異常さを叩きつけてから、CQを明示する。

**入れるべき内容**:

- クラウスが工場で今月の電気代の請求書を受け取る場面
- 数字を連射: 「産業用電力、0.19ドル。アメリカは0.08ドル。EU加盟国の中で最高水準。」
- 自己紹介: 「ミュンヘン郊外で工場を動かしている。毎月この請求書が届く」
- 「なぜこんなに高いのか。本気で調べてみた」
- 予防線（1文）: 「原発を推したいわけじゃない。請求書の中身を確認したかっただけだ」
- CQ提示: 「環境のために原発を止めた国。その請求書には何が書かれていたのか」

> **例文案**: 「今月も届いた。工場の電気代の請求書だ。ドイツの産業用電力、1キロワット時あたり0.19ドル。アメリカは0.08ドル。同じものを作っても、電気代だけで2倍以上のハンデがある。ドイツはEU加盟国で最も電気代が高い国のひとつだ。僕はミュンヘン郊外で自動車部品の工場を動かしている。毎月この請求書を見るたびに、いったいなぜこうなったのかを考える。原発を推したいわけじゃない。ただ、請求書の中身を全部確認したかった。環境のために原発を止めた国。その請求書には、何が書かれていたのか。」

| EU最高水準の電力価格                                     | Eurostat                           |

### Block 2：「決断の背景」— 30秒でわかる脱原発の歴史（1:30〜3:30 / 500字）

**機能**: 視聴者が「なぜ止めたのか」を知らないまま請求書を開いても怒りが届かない。**最小限の歴史**を高速で見せてから、請求書の開封に入る。長くしない。トラスショック台本の「トラスが何を約束したか」（Before）と同じ機能。

**入れるべき内容**:

- 1986年 チェルノブイリ事故 → ドイツ国民に「原発＝恐怖」が刻まれた
- 2000年 「原子力合意」→ 2022年までに全炉廃止を法律で定めた
- 2011年 福島事故 → メルケル政権が8基を即時停止 → 加速
- 2023年4月 最後の3基停止 → 半世紀以上の歴史に幕
- 「ドイツは言った。『原発なしでやれる。環境先進国になる』と」
- クラウスの個人的な感情: 「正直、当時は僕も間違っていないと思っていた。だから今日の話は、他人を批判する話じゃない」

> **例文案**: 「話は1986年から始まる。チェルノブイリ。旧ソ連で原発が爆発した。その映像はドイツ国民の中に深く残った。『原発は怖い。いつかは止める』。その感情が2000年に法律になり、2011年の福島事故でアクセルが踏まれた。メルケル首相は8基を即日停止した。そして2023年4月、最後の3基が止まった。ドイツは言った。『私たちには再エネがある。原発なしでやれる』。当時、僕も間違っていないと思っていた。だから今日の話は、誰かを責める話じゃない。請求書の中身を確認する話だ。」

### Block 3：ゲート宣言 ＋ 請求書1枚目「財布の請求書」（3:30〜6:00 / 650字）

**機能**: ゲート宣言で本題に入り、「予想通りの請求書」を1枚目として置く。冒頭（Block 1）ですでに電気代の話をしているので、1枚目が財布であることは視聴者の「予想通り」になる。「予想通りが確認される安心感」の直後に「でもお金だけじゃなかった」でBlock 4（CO2）へのサスペンスを作る。ドイツ台本の「じゃあドイツ人は得じゃないか→でも現実は違った」のホープ＆クラッシュと同じ設計。

**入れるべき内容**:

- ゲート宣言: 「では請求書を開けていこう。1枚目だ」
- 請求書1枚目のタイトル: **「財布の請求書」**
- CO2排出権（EU ETS）購入費用 → **570億ユーロ** → 1世帯あたり**約1,390ユーロ（約22万円）**の損失
- 換算（ルール7）: 「ドイツ全世帯に配ったら、1家族あたり22万円の穴が開いた計算になる」
- PwC分析: 原発を維持していれば2024年の電力価格は**23%安かった（18ユーロ/MWh安）**
- クラウスの実感（自己開示）: 「この工場の電気代は、10年前の2倍以上になった。同じものを作っているのに」
- 産業用電力 0.19ドル（独）vs 0.08ドル（米）→ 「この差がある限り、工場はいつかどこかに逃げる」
- 政府の対応: 2030年までに**113億ドルの電力価格補助金** → 「電気代の穴を税金で埋めている」
- 「請求書の1枚目は財布だった。予想通りだ」
- エスカレーション宣言: **「でもお金の問題だけじゃなかった」**

> **例文案**: 「請求書を開けよう。1枚目。財布だ。ドイツは排出権を買い続けた。その額、570億ユーロ。ドイツ全世帯に配れば、1家族あたり約22万円の穴が開いた計算になる。もし原発を維持していたら、今の電力価格は23%安かったという試算もある。この工場の電気代は、10年前と比べて2倍以上になった。同じものを作っているのに。アメリカの同業他社は0.08ドルで電気を使っている。僕たちは0.19ドルだ。この差がある限り、工場はいつかどこかに逃げる。1枚目の請求書は財布だった。予想通りだ。でもお金の問題だけじゃなかった。」

### Block 4：請求書2枚目「CO2の請求書」（6:00〜8:30 / 700字）★常識破壊・最大の驚き

**機能**: 台本最大の「常識を壊す」瞬間。1枚目の財布で「予想通りだ」と思わせてから、「でも」で常識を破壊する。「脱原発＝エコ」という視聴者の常識がここで崩れる。サムライ経済学の横断分析が示す**「知っていたつもりを壊される快感」**を最大化する。

**入れるべき内容**:

- 「2枚目の請求書が来た。今度は環境だ」
- 2011〜2023年で失われた原発電力: **800 TWh**
- その穴を何で埋めたか → **石炭と褐炭**（「二酸化炭素を最も多く出す発電方法で埋めた」）
- 結果: CO2追加排出 **7.3億トン**
- 換算（ルール7）: 「ドイツが2024年1年間で出した温室効果ガスの総量を上回る規模だ」
- 皮肉（台本の核）: 「ドイツは地球を守るために原発を止めた。でも請求書の2枚目はCO2だった」
- 「脱原発＝エコ」という常識が逆転する瞬間として明示する
- エスカレーション宣言: **「しかもまだ終わらなかった」**

> **例文案**: 「2枚目の請求書。今度は環境だ。ドイツは2011年から2023年の間に、800テラワット時の原発電力を失った。その穴を何で埋めたか。石炭と褐炭だ。二酸化炭素を最も多く出す発電方法で埋めた。その結果、追加で7億3,000万トンのCO2が大気に放出された。ドイツが2024年1年間で出した温室効果ガスの総量を上回る規模だ。ドイツは地球を守るために原発を止めた。でも請求書の2枚目はCO2だった。環境のために原発を止めたのに、環境への請求書が届いた。しかもまだ終わらなかった。」

### Block 5：請求書3枚目「命の請求書」（8:30〜11:30 / 700字）★感情的クライマックス

**機能**: 台本全体の感情的クライマックス。財布（怒り）→ CO2（驚き）ときて、最後に「命」で感情の天井を打つ。「確率論的リスク（原発事故）を避けた代償として、確定的な健康被害を招いた」という逆説が核心。大きな数字のあとに**1人の人物を描く**（必須要素ルール4）。

**入れるべき内容**:

- 「3枚目の請求書が来た。今度は命だ」
- 石炭燃焼による大気汚染（PM2.5・越境汚染）→ **早期死亡1万9,200人** / 重篤疾患**17万7,000人**（WHO基準の推計）
- 換算（ルール7）: 「地方都市が1つ、丸ごと消えた規模の人数が、早く亡くなった」
- 1人の人物を描く（ルール4）: 「工場の隣に住む70代の男がいる。数年前からぜんそくが悪化したと言っていた。直接の因果関係は証明できない。でもこの数字を知った後では、同じようには聞けない」
- 予防線（必須）: 「これはドイツだけの問題じゃない。大気汚染は国境を越える。ポーランド、チェコ、フランス。隣国にも及んでいる」
- 追加データ: 水銀4トン、カドミウム2.5トン、鉛20トンが環境中に放出
- 「ドイツは放射性廃棄物のリスクを避けた。その代わりに石炭の煙を選んだ。請求書の3枚目は命だった」

> **例文案**: 「3枚目の請求書。命だ。石炭を燃やすと、CO2だけじゃない。細かいちりが空気に漂う。それが肺に入る。ドイツとその周辺国で、1万9,200人が通常よりも早く亡くなったと推計されている。17万7,000人が重篤な疾患を抱えた。地方都市が1つ、丸ごと消えた規模の人数が、早く亡くなった。工場の隣に70代の男がいる。数年前からぜんそくが悪化したと言っていた。直接の証明はできない。でもこの数字を知った後では、同じようには聞けない。ドイツは原発事故のリスクを避けた。その代わりに石炭の煙を選んだ。請求書の3枚目は命だった。」

## 注意点（ライターへ）

1. **「原発推進」の動画にしない**
   Block 1に予防線を1文（「原発を推したいわけじゃない、請求書の中身を確認したかっただけだ」）、Block 7のUターン紹介でも「これが正解だとは言わない」のニュアンスを維持する。主語は「脱原発政策の選択コスト」であり、「原発が正しい」の結論は押しつけない。

2. **クラウスが「工場長」であることを活かす**
   Block 1・3・5に「この工場の電気代が〜」の自己開示を入れることで、産業用電力の高さが他人事ではなく**当事者の切実な問題**として伝わる。Block 7以降は「それでも工場を動かし続けている」というクラウスの姿で締める。

3. **ルール4「1人の人物」はBlock 5に必ず入れる**
   「1万9,200人」という数字の直後に、クラウスの隣人・知人など具体的な1人を数行描く。これがないとBlock 5は「数字の羅列」に終わる。感情的クライマックスがここにあるため、人物化は必須。

4. **全ての大きな数字を換算する**（ルール7）
   - 800 TWh → 「東京の全家庭が16年間使う電力量」などに換算検討
   - 7.3億トン → 「ドイツの2024年単年排出量を超える」
   - 570億ユーロ → 「1世帯あたり約22万円」
   - 1万9,200人 → 「地方都市1つが丸ごと消えた規模」
   - 1,280億ユーロ → 「日本の国家年間予算の約1割」など

5. **口調の統一**（ルール8）
   クラウスは「だ・だよ・なんだ」体で統一。「です・ます」は絶対に混在させない。工場長らしい「短文・断言」の口調が基本。

6. **Block 7のUターン紹介は「観察」にとどめる**
   「だから原発に戻るべきだ」とは言わない。「世界は動き始めている。ドイツはまだ途中にいる」の観察で締める。答えを押しつけず、視聴者に考えさせる。
