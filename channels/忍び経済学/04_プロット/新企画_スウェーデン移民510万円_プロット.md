**参考**: サムライ経済学（構成・技法をベンチマーク）

## 採用した型の根拠（全15本の分析から）

全15本の台本分析を精査し、以下の理由からこの型を採用した。

### なぜ「告発エスカレーション型」（ドイツ型）を不採用にしたか

- ドイツ台本のエンジンは**「怒り」**。ハンスの個人的な怒りが6層の問題を駆動する。
- 今回の企画のコンテンツは**財政データの構造分析**（企画1）。怒りではなく「なぜこうなったのか」の**知的好奇心**が最も自然なエンジン。
- 怒りを無理に乗せると「移民排斥」に読まれるリスクが高い。**データで語る冷静さ**がこの企画のブランド。

### なぜ「ミステリー型 × 論文プレゼン型」（オランダ前編型）を採用したか

- オランダ前編は「言ってはいけない」=タブーのデータを、学術論文を軸に**段階的にズームイン**して開示する構成。最も伸びた台本の一つ。
- 今回の企画は「510万円で帰ってください」=**衝撃の事実**を冒頭に置き、「なぜこうなったのか」を**NIERの財政データ**で段階的に解明していく。**構造が完全に一致**する。
- 510万円 = ミステリーのフック（Why?）
- NIER財政データ = 論文プレゼンの軸（How much? / Who?）
- 政策転換 = ミステリーの解（答え合わせ）

### エスカレーション要素を加えた理由

- オランダ前編は前後編の前半で「データ提示」に徹した。今回は**単発動画**なので、データ層が重くなっていく**ドイツ的なエスカレーション**を組み込み、リテンションを維持する。
- 「全体数字→分解→構造問題→政策転換→日本」と層ごとにデータの**衝撃度が上がる**設計。

| リング | 「食卓」→「食卓」 | 「冗談じゃない」 | **「510万円」→「510万円の意味が変わる」** |

## この動画の狙い（ライター向け要約）

**一言**: 「510万円で帰ってください」の衝撃フックから入り、「なぜ福祉天国がここまで変わったのか」を40年分の財政データで解明する動画。ミステリーの謎解きのように、データが1枚ずつ開示されるたびに「510万円」の意味が重くなっていく。

**構造**: オランダ前編の「段階的データ開示」＋ドイツのエスカレーション的積み上げ。

```
Layer 1: 510万円の衝撃 → なぜ？
Layer 2: 40年分の財政データ → 34年間ずっとマイナスだった
Layer 3: 中身を分解 → 難民-145万/人 vs インド系+170万/人
Layer 4: なぜマイナスなのか → 団地・脆弱地域・就業率37%の構造
Layer 5: 510万円の正体 → Tidö合意・政策大転換は40年の答えだった
→ 日本ブリッジ: 日本の123万人計画に「統合の設計」はあるか
```

**感情の流れ**: 衝撃（510万？）→ 好奇心（なぜ？）→ 驚き（34年マイナス）→ 知的興味（分解データ）→ 重い理解（構造問題）→ 納得（だから510万か）→ 危機感（日本も同じ道に？）→ 余韻

**CQ**: 「なぜ移民を世界一歓迎した国が、510万円を払って送り返すことにしたのか？」

**トーン**: オランダ前編のエマの「知的誠実さ＋冷静な好奇心」に最も近い。怒りではなく「理解したい」の姿勢。データに語らせる。ただしオランダ前編ほどアカデミックにはせず、語り手の**日常の実感**も交える（イタリア・マルコの切なさの要素）。

**エンジン**: **「好奇心＋構造理解」**。「510万円」のインパクトが「なぜ？」を起動し、データが1枚ずつ開かれるたびに「次は何が出る？」で引っ張る。オランダ前編と同じ**「隠されたものが見える快感」**が主報酬。

### Block 4：分解 —「一括りにしてはいけない」（6:00〜8:00 / 600字）★知的クライマックス

**機能**: オランダ前編Block Dの「87地域×4理由の内訳分析」に相当。全体数字を**カテゴリー別に分解**し、「一括りにしてはいけない」のメッセージを示す。オランダ前編で最も知的満足度が高かったブロックの再現。サムライ経済学のDNA＝**「フェアな分解」**が最も発揮される。

**入れるべき内容**:
- 難民: 年間1人あたり **-100,000クローナ（約-145万円）**
- 非難民（労働移民）: 年間1人あたり **+48,000クローナ（約+70万円）**
- インド系移民（トップ貢献者）: 年間1人あたり **+119,000クローナ（約+170万円）**
- 「来た理由とスキルで、数字が完全に反転する」
- ヨーテボリ大学研究: 難民の生涯財政コスト = GDP比 -12%〜-22%/人（出身国の労働市場成績による）
- 語り手の実感: 「僕のインド系の同僚は、確かにスウェーデンの税金を払って、スウェーデンの経済に貢献している。でも郊外の団地にいる人たちは、全く別の状況にいる」
- 予防線: 「これは"良い移民と悪い移民"の話じゃない。来た理由と、来た後の環境の話です」

> **例文**: 「難民。年間1人あたり、マイナス10万クローナ。約マイナス145万円。一方、非難民の労働移民はプラス4万8千クローナ。約プラス70万円。インド系移民に至っては、プラス11万9千クローナ。約プラス170万円。同じ"移民"でも、来た理由とスキルで数字が完全に反転する。これは良い移民と悪い移民の話じゃありません。来た理由と、来た後に何があったかの話です。」

### Block 5：構造問題 —「なぜマイナスが34年続いたのか」（8:00〜9:30 / 500字）

**機能**: オランダ前編Block Eの「世代追跡」に相当する**Why?の深掘り**。数字がマイナスだった**構造的原因**を示す。ドイツ台本Block D「住宅戦争」と同じく、**場所の具体性**で数字を人間化する。トラスショックの因果連鎖（善意の政策→逆効果）の設計も活用。

**入れるべき内容**:
- ミリオンプログラム（1965-74年）: 10年で100万戸の団地を建設 → 善意の住宅政策
- 1990年代の経済危機 → スウェーデン人が団地を出る → 移民が集住
- 結果: 65地区が「脆弱地域」指定（うち19が「特に脆弱」）
- 脆弱地域の就業率: 37-38%（一般地域は80%超）
- 福祉依存率: 65%
- 「数字がマイナスだった理由は、移民のせいではなく、この構造のせいです」

> **例文**: 「1965年、スウェーデンは"全国民に良い家を"という理想のもと、10年で100万戸の団地を建てました。ミリオンプログラム。でも90年代の経済危機でスウェーデン人が都心に移り、空いた団地に移民が入った。60年後、その団地は65の"脆弱地域"になりました。就業率37%。福祉依存65%。駅を2つ動くだけで、別の国に来たような数字になる。34年間マイナスだった理由は、人ではなく、この構造にあります。」

## 注意点（ライターへ）

1. **「移民＝問題」に絶対にしない**: この台本の主語は「移民」ではなく「構造」。ドイツ台本のハンスが「悪いのは政策」と明言したように、「構造を設計しなかった側」に矛先を向ける。オランダ完結編の「悪いのは移民ではなく、政策」の着地を踏襲。

2. **予防線のタイミング**: Block 4の分解データの直後に「良い移民と悪い移民の話じゃない」、Block 8の結びで「移民を責めているんじゃない」。**最低2回**は明示的に入れる。

3. **「510万円」のリフレイン**: 4回使う。毎回意味が変わっていく設計。特にBlock 8で「冒頭と同じ数字を聞いた時、意味が変わっている」ことを意識させる。

4. **ギャング・銃犯罪の扱い**: Block 5の構造問題で脆弱地域に触れるが、**ギャングを主語にしない**。あくまで「構造がもたらした結果のひとつ」として短く触れるか、完全に省略しても構成は成立する。銃犯罪63%減のデータを入れてバランスを取ると安全。

5. **数字の円換算**: スウェーデン・クローナは全て日本円に翻訳する（1 SEK ≈ 14.5円で概算。端数は丸める）。

6. **トーンの一貫性**: 怒りではなく「理解したい」。語り手は問題を告発しているのではなく、**自分が住む国の構造を理解しようとしている**。イタリアのマルコの「この国が好きだからこそ」の切なさに近い。

7. **オランダ前編との差別化**: オランダ前編は前後編で20分以上使った。今回は**単発10-15分**。データの深さよりも「510万円→答え合わせ」のミステリー構造のテンポを優先。各ブロックのデータは1-2個に絞る。
