# 忍び経済学

忍び経済学チャンネルの管理フォルダです。

## 制作の流れ（工程順）

新規で動画1本を作るときの**中核の順番**は、フォルダ番号と一致しています（`03` → `04` → `05`）。

| 順番 | フォルダ | 役割 |
|:----:|----------|------|
| 1 | `03_企画リサーチ` | テーマの調査・データ・引用・企画メモ（事実の土台） |
| 2 | `04_プロット` | 構成案・話の流れ（導入→展開→締めの骨子） |
| 3 | `05_台本執筆` | 台本の下書き・本番・修正指示書 |

## いつでも参照する・並行して使うもの

| フォルダ | 用途 |
|----------|------|
| `00_マニュアル` | 台本マニュアル・執筆ガイドライン |
| `01_競合台本分析` | 競合チャンネルの台本・構成メモ・分析結果 |
| `02_自チャン台本分析` | 自チャンネルの過去動画の台本分析・傾向メモ |
| `06_プロンプト` | AI用プロンプト（スキル化済みも原本を保管） |
| `07_キャラクター設定` | 語り手のキャラクター設計書・動画編集者向け設定 |
| `08_競合文字起こし` | **競合（他チャンネル）動画の文字起こし**（分析・新企画プロットの材料）。自チャン公開分の用途とは別 |

| # | フォルダ | 用途 |
