# 忍び経済学

忍び経済学チャンネルの管理フォルダです。

## サブフォルダの役割

| フォルダ | 用途 |
|----------|------|
| 00_マニュアル | 台本マニュアル・執筆ガイドライン |
| 01_競合台本分析 | 競合チャンネルの台本・構成メモ・分析結果 |
| 02_自チャン台本分析 | 自チャンネルの過去動画の台本分析・傾向メモ |
| 03_台本執筆 | これから出す動画の台本・下書き・修正指示書 |
| 04_プロンプト | AI用プロンプト（スキル化済みも原本を保管） |
| 05_文字起こし | 動画の書き起こしデータ |
| 06_企画リサーチ | 企画テーマの調査結果・データ・引用元 |
| 07_キャラクター設定 | 語り手のキャラクター設計書・動画編集者向け設定 |
