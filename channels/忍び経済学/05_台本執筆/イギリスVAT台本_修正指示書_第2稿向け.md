# イギリスVAT台本｜修正指示書（第2稿向け）

**対象台本**: 食料品は0%、なのに300万人がご飯を食べれない｜イギリス減税の"不都合な真実"
**作成日**: 2026-02-19

---

## この指示書の使い方

修正箇所は **6つ** あります。重要度の高い順に並べています。
**①②を優先** してください。この2つだけで台本の質が大きく変わります。

各修正に「今の台本」→「何が問題か」→「修正後イメージ」の3つを記載しています。
修正後イメージはそのまま使っていただいても、ご自身の言葉に書き換えていただいても構いません。
大事なのは「何を直すか」と「なぜ直すか」を理解していただくことです。

## 修正①：冒頭を圧縮する【最重要】

### 今の台本（冒頭〜CQまで）

```
昨日、引換券を握りしめ、フードバンクに行った。
妻と僕の3日分の食料をもらうためだ。
フードバンクでは、まず受付でアンケートに答える。
食生活や食べられない物などを答えると、自分にあった食品をセットにしてもらえる。
セットができるまでの間、僕はぼうっと考え事をしながら待った。
2歳年下の妻は2か月後に出産予定だ。
妻も働いているが、高い家賃や光熱費で生活は苦しい。
前の日の夜、フードバンクに行くことを告げると、妻は「ありがとう」と一言だけ言った。
いつもは明るい妻が、少し悲しげな表情をした。
食料品セットの中に、大好物のポテトチップスが入っていたら、妻は笑顔をみせるだろうか。
いいや、ポテトチップスは入っていないだろう。
なぜなら、ポテトチップスは高級品だから。
イギリスでは食品によって税率が違う。
パン0%、卵0%、牛乳0%、野菜に果物も0％。
でも、ポテトチップスは税率が20％だ。
あと、ビスケットは0%だが、チョコがコーティングされると20％になる。
なんだかややこしいなあ、と思ったことはあるが、その理由までは考えたことはなかった。
どうしてポテトチップは20％も税金がかかるのだろうか。
逆に、なぜほとんどの食料品は0％なんだろう。
食料品が誰でも買えるように？
でも、僕はスーパーで食料品を買えず、ここにいる。
0％って、一体誰のための制度なんだろう。
```

### 何が問題か

CQ（「0%って、一体誰のための制度なんだろう」）に到達するまでに **22文** かかっています。
サムライ経済学の冒頭は **4〜8文** でCQに到達します。

遅くしている原因は2つです。

1. **フードバンクの手続きの説明**（3〜5行目）→ 物語の進行に必要ない情報です
2. **ポテトチップスの話が長い**（10〜17行目）→ CQの前に制度説明が入ってしまっています

### 修正後イメージ

```
昨日、引換券を握りしめ、フードバンクに行った。
妻と僕の3日分の食料をもらうためだ。
2歳年下の妻は2か月後に出産予定だ。
前の日の夜、フードバンクに行くと告げたら、妻は「ありがとう」と一言だけ言った。
いつもは明るい妻が、少し悲しげな表情をした。
パン0%、卵0%、牛乳0%。
この国では食料品に税金がかからない。
なのに僕は、食料品を買えずにここにいる。
0%って、一体誰のための制度なんだろう。
```

**9文でCQに到達します。** カットしたものは以下のとおりです。

- フードバンクの受付手続き（3〜4行目）→ 完全カット
- 「ぼうっと考え事をしながら待った」→ カット
- ポテトチップスの話（10〜17行目）→ ここでは使わず、後の制度説明パートに移動していただいて構いません

妻のエピソードはマイケルの人間性を見せるのに必要ですので残しています。
「パン0%、卵0%、牛乳0%」のリフレインもそのまま使ってください。結末のリングに繋がります。

## 修正②：ナレーター口調の箇所をマイケルの声に書き換える【最重要】

以下の6箇所が「教科書の読み上げ」になっています。
すべて **マイケルが友達にしゃべっている口調** に書き換えをお願いします。

---

### 箇所A：VAT制度の説明

**今の台本：**
```
僕らの国では消費税のことをVAT（付加価値税）と呼ぶ。
基本の標準税率は20％で、商品やサービスの価格に課税されている。
でも、軽減税率というのがあって、光熱費やチャイルドシートなどは5%の税率になっている。
そして、食料品や医薬品、子供の衣服などは0％だ。
```

**問題**：「商品やサービスの価格に課税されている」「軽減税率というのがあって」が教科書的です。マイケルは友達にこうは話しません。

**修正後イメージ：**
```
僕らの国には買い物するたびに取られる税金がある。
VATって呼ばれてるやつで、基本は20%。
でも全部に20%がかかるわけじゃない。
電気やガスは5%。
そして食べ物は0%だ。パン、卵、牛乳、野菜、果物。全部0%。
一方で、ポテトチップスは20%。贅沢品扱いだ。
ポテトチップスが贅沢品って、笑えるだろ？
```

---

### 箇所B：食料品の税率区分の説明

**今の台本：**
```
ここで、ややこしいのが食料品の税金なんだ。
食料品といっても、レストランの食事や出前、温かい料理のテイクアウトは20％の標準税率。
お菓子やポテトチップス、アイスクリーム、ソフトドリンクも標準税率だ。
これらは、食べなくても生きていけるからね。
贅沢品扱いというわけだ。
一方、ケーキや普通のビスケットは0％。
ただし、チョコレートがかかったビスケットは標準税率となっている。
基準がややこしいので、課税当局と企業の間で判断が分かれることもよくある。
```

**問題**：「標準税率」が3回出てきます。リストの羅列になっています。「課税当局と企業の間で判断が分かれることもよくある」はナレーターの声です。

**修正後イメージ：**
```
問題はここからだ。
同じ食べ物でも、税率が全然違う。
レストランで食べたら20%。テイクアウトでも温かければ20%。
ポテトチップス、アイス、炭酸飲料も20%。
「生きるのに必要ない」から贅沢品なんだってさ。
でもケーキは0%。ビスケットも0%。
ただし、チョコがかかったビスケットは20%になる。
なんだそれ、って思うだろ？ 僕もそう思う。
```

---

### 箇所C：0%の恩恵の解説

**今の台本：**
```
でも0%の恩恵を金額ベースで最も多く受けているのは、実は高所得世帯なんだ。
たとえば、200円のカップ麺と、1万円の高級ステーキ肉。
税率が20％なら、ラーメンの税額は40円、高級肉は2000円だ。
これが税率0%になれば、お得なのは当然、肉のほうだろう。
こうした恩恵を単純に金額でみると、最も裕福な層は、最も貧しい層の約3倍も得をしている。
これはれっきとしたイギリスのシンクタンク、IFS（英国財政研究所）のデータだ。
つまり「庶民のための制度」と言いながら、実は恩恵を受けているのは金持ちなんだ。
```

**問題**：「金額ベースで」「こうした恩恵を単純に金額でみると」「れっきとした」がナレーター調です。内容自体は良いので、リズムだけ直していただければと思います。

**修正後イメージ：**
```
ここで1つ、不都合な事実を話す。
0%で一番トクをしているのは、実は金持ちだ。
考えてみてくれ。
200円のカップ麺と、1万円の高級ステーキ。
20%の税金がかかるとしたら、ラーメンは40円。ステーキは2000円。
0%で浮く額が大きいのは、どう見てもステーキの方だろ？
IFSっていうイギリスの研究所が計算したんだけど、
最も裕福な層は、最も貧しい層の約3倍トクしている。
3倍だよ。
「庶民のための制度」のはずが、一番恩恵を受けているのは金持ちだったんだ。
```

### 箇所D：生活費の説明

**今の台本：**
```
たとえば、家賃が高い。
ロンドンでは平均家賃は40万円以上する。
サラリーマンの平均の手取りは月60万円前後。
僕のような倉庫の作業員なら50万程度だ。
とてもではないが、僕の給料だけでは、平均的な家賃の家になんて住めない。
妻と二人の収入でなんとか暮らしていける。
しかも、最近は家賃が年10％程度上がっている。
```

**問題**：「サラリーマンの平均の手取りは月60万円前後」がナレーターの統計読み上げになっています。マイケルなら自分の手取りから話すはずです。

**修正後イメージ：**
```
まず家賃。ロンドンでは平均40万円以上する。
僕の手取りは月50万程度。
妻と二人で稼いで、やっと暮らしていける。
それでも家賃は毎年10%ずつ上がっていく。
去年払えた部屋が、来年は払えなくなるかもしれない。
```

「サラリーマンの平均」は不要です。マイケルの数字だけで十分伝わります。

### 箇所E：フードバンクの説明

**今の台本：**
```
貧しい人に食料を提供するフードバンクはマクドナルドの2倍以上の数だ。
フードバンクでは、毎日、昼食のために多くの人が訪れる。
そして、温かい食事を食べたら、食料品を受け取って帰っていく。
全国最大のフードバンク団体では、2025年に300万個の食料品セットを配った。
でも、これは一部に過ぎない。
この団体によると、イギリスでは900万人が食料品の支援を必要としているそうだ。
国全体で7人に1人の割合だよ。
今、イギリス国内の最大の課題は飢餓対策なんだ。
```

**問題**：「フードバンクでは、毎日、昼食のために多くの人が訪れる。そして、温かい食事を食べたら、食料品を受け取って帰っていく」がパンフレットの説明文になっています。マイケルは利用者ですので、体験として語れるはずです。

**修正後イメージ：**
```
知ってるかい。
イギリスにあるフードバンクの数は、マクドナルドの2倍以上だ。
最大の団体だけで、去年300万個の食料品セットを配った。
それでも足りない。
支援を必要としている人は900万人。
国民の7人に1人だよ。
```

手続きの説明をカットし、数字を会話のテンポで出す形にしています。「知ってるかい」で巻き込み、「7人に1人だよ」で落とすリズムです。

### 箇所F：増税と負の連鎖の説明

**今の台本：**
```
実際に政府は税収不足を補うため、2024年に大規模な増税を行った。
企業が支払う国民保健料の引き上げや、私立学校の授業料への課税などさ。
もちろん、庶民に直接負担させることはしなかった。
しかし、国民保険料の負担が増えた企業は、増税分を商品の価格に上乗せした。
結局、負担は庶民に回ってくる。
税率が上がらなくたって、食料品の値段も上がって負担が増えることには変わらない。
そうすれば、景気が悪くなって、ますます仕事がなくなっていく。
それで、ますますフードバンクの利用者も増えていく。
まさに、負の連鎖というやつさ。
```

**問題**：「企業が支払う国民保健料の引き上げや、私立学校の授業料への課税など」が政策解説になっています。マイケルの肌感覚で語れていません。

**修正後イメージ：**
```
去年、政府は税収が足りないからって、企業への税金を上げた。
庶民には直接手を出さなかった。
でも、どうなったと思う？
企業は増税分をそのまま商品の値段に乗せたんだ。
スーパーの棚の値札が、じわじわ上がっていく。
税率は変わっていない。でも値段は上がっている。
そうすると景気が悪くなる。仕事が減る。
フードバンクの列が長くなる。
負の連鎖だ。
```

「負の連鎖だ。」を独立した短文にすることで、パンチが出ます。

## 修正⑤：ジェシカの後を短くする

### 今の流れ（ジェシカ以降）

```
ジェシカのエピソード（感情のピーク）
    ↓
「税金が0%になっても負担がゼロになるわけではない」（構造の解説）
    ↓
年間250億ポンドのスケール翻訳
    ↓
左のポケット→右のポケットの比喩
    ↓
IFSの提言（7文にわたる解決策の紹介）
    ↓
「ケーキかビスケットかで言い争うのは馬鹿げている」
    ↓
「でも食料品に税金をかけられると困る」
    ↓
「簡単には後戻りできない」
    ↓
日本ブリッジ
    ↓
リング構造で結末
```

### 何が問題か

感情のピーク（ジェシカ）の後に **構造の解説が長くなっています。** せっかくの感情が冷めてしまいます。

サムライ経済学の型ではこうなっています。

| 台本 | 感情ピークの後 |
|---|---|
| ドイツ | 「結局、誰も幸せじゃない。EUって何だったんだろうな」（2文で終了）→ 締め |
| イタリア | 「3つの設計ミス」（短い整理）→「来月33歳」（リング回収） |
| ロンドン | 「政治家は変わった。暮らしは変わらない」（1文の対比）→ 締め |

感情ピークの後は **短く、余韻を残して閉じる** のが型です。

### お願いしたいこと

**IFSの提言パート（7文）を2〜3文に圧縮** してください。

今の台本：
```
たとえば、IFSでは軽減税率の食料品0％の撤廃を提言している。
なぜなら、0%や軽減税率の恩恵の多くは、高額所得者が受けてしまうからだ。
しかも、煩雑な税制は政府の行政コストも増やしてしまう。
だから、いったん0％や軽減税率は廃止し、税率は一律にしたほうが良いというわけだ。
そのうえで、低所得者へは給付金の支給や所得税減税を行う。
そうすれば、税収を確保したうえで、低所得者への支援ができる。
こうしたIFSの主張に賛成する人も多い。
```

修正後イメージ：
```
専門家の中には、「いっそ0%をやめて全部に税金をかけ、
その分を低所得者に直接配った方がいい」と言う人もいる。
理屈としては正しいのかもしれない。
```

その後のマイケルの反応（「それはわかっている、でも困る」）はそのまま活かしてください。マイケルの声で語られており、庶民の葛藤としてうまく機能しています。

## 修正⑥：マイケルの感情トーンを統一する

### 何が問題か

マイケルの感情が場面によってバラバラになっています。
怒っている箇所、淡々としている箇所、教師のように説明している箇所が混在していて、読後に「マイケルはどういう人か」が残りにくい状態です。

サムライ経済学の主人公は全員、**感情の色が1色** で一貫しています。

| 主人公 | 感情の色 | 特徴 |
|---|---|---|
| マルコ（イタリア） | **切なさ・静かな諦め** | 怒らない。受け入れつつ、構造を理解しようとする |
| ハンス（ドイツ） | **怒り** | 「冗談じゃない」「幻想だよ」。カジュアルに怒る |
| ジョン（ロンドン） | **冷静な失望** | 丁寧な口調のまま、淡々と絶望的な数字を積む |

### マイケルの色

マイケルの最大の特徴は **「食料品を毎日運んでいるのに、自分は食べられない」** という皮肉な立場です。

この皮肉に対する感情は **「苦笑い混じりの怒り」** が最も自然です。
爆発的に怒るのではなく、「笑うしかない」「なんとも皮肉な話だ」というトーンです。

### 統一していただきたい箇所

**箇所A：自己紹介直後**（変更不要です。これがトーンの核になっています）

```
こうした食料品を店頭に送り出す仕事を僕は誇りに思っている。
その僕が食べ物をもらいに行ったんだ。
なんとも皮肉な話だと思わないか。
```
→ この「なんとも皮肉な話」のトーンを台本全体に広げてください。

**箇所B：Heat or Eatの直後**
```
今 ：「これが食料品の消費税が0%の国で起きている現実なんだ。」
修正：「これが0%の国の冬だ。笑えないだろ？」
```

**箇所C：ポケットの比喩の後**
```
今 ：「それが、食料品の税率0%でも食べ物に困る人が減らない理由なんだ。」
```
