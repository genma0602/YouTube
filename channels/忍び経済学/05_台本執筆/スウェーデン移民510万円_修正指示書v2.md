## 【致命的】修正①：カリムの設定＋510万円制度の描写を現実に寄せる

### なぜ直すのか（制度との整合性）

台本で使っている「510万円（35万クローナ）の帰国支援」について、
スウェーデン移民局（Migrationsverket）と政府サイトで公式の仕組みを確認した。
**台本の描き方と制度の実態に3点のズレがある。**
このまま出すと、制度を知っている視聴者（北欧在住者・移民政策に詳しい層）から
「事実と違う」と指摘される。

---

### ズレ①：510万円は「帰国勧告」ではなく「申請制の支援金」

#### 制度の事実

- 移民局の公式ページでは **"frivillig återvandring"（自主的な帰国）** と明記
- 本人が「帰りたい」と決めた上で **自分から移民局に申請する** 仕組み
- 政府が個人に「帰ってください」と通知・勧告する制度ではない
- 出典：Migrationsverket「Ansök om återvandringsbidrag för frivillig återvandring」

#### 今の台本の問題箇所

```
僕の名前はエリック、33歳。
（中略）
その友人に、国が「帰ってください」と言っている。
```

```
かつて世界一の歓迎を誇った国が、
40年かけてたどり着いた最終回答が、
「お金を払うから、どうか帰ってください」
というメッセージだった。
```

→ 「国が帰ってくださいと言っている」は、制度上は正確ではない。
**ただし、「帰ってください」のフレーズ自体は台本の最大のフックなので残す。**
問題は「国が個人に通知している」と聞こえる描写の方。

#### 直し方

**タイトル・冒頭フックの「帰ってください」はそのまま残してよい。**
変えるのは、カリムがこの制度を**どう知ったか**の部分だけ。

**今の台本：**
```
先月、チームの昼食中に、
同僚のカリムが静かにこう言った。
「エリック、俺、帰国するかもしれない」
（中略）
「510万円もらえるらしいんだ。妻とも話した」
```

**修正案：**
```
先月、チームの昼食中に、
同僚のカリムがスマホの画面を見せてきた。
スウェーデン移民局のページだった。
「帰国するなら35万クローナ出すってさ。申請すればもらえるらしい」
カリムは笑いながら言った。
でも、目は笑っていなかった。
「国が札束を積んで『帰りませんか』って言ってるんだよ。
制度としては『自主的な帰国支援』らしいけどさ。
受け取り方は一つだろ」
```

→ 制度が「申請制」であることを台詞の中で正確に示しつつ、
  カリムの受け止め方として「実質的には"帰れ"のメッセージだ」と語らせる。
→ 事実は正確に、感情は台本のフックとして維持。**両立できる。**

### ズレ②：対象は「保護理由（難民等）で滞在許可を持つ人」限定

#### 制度の事実

510万円の対象になるのは以下の条件を**すべて**満たす人：

| 条件 | 内容 |
|------|------|
| 滞在許可の理由 | **難民・補完的保護・特別事情・クォータ難民**のいずれか |
| 取得時期 | **2024年9月12日以前**に滞在許可を取得 |
| 移住先 | **EU/EEA/スイス以外の国**に永住する意思 |
| 除外 | 退去強制決定を受けている人は対象外 |

- 出典：Migrationsverket「Vem kan få återvandringsbidrag?」
- 出典：Government.se「Repatriation assistance」（2024年10月）

**つまり、就労ビザで来た「普通の労働移民」は対象外。**

#### 今の台本の問題箇所

```
カリムはシリア出身のエンジニアで、
5年前にこの国に来た。
腕のいいプログラマーで、チームに欠かせない存在だ。
彼の書くコードは美しく、バグが少ない。
```

→ この設定だと「就労目的の優秀なエンジニア」に見える。
  就労ビザの移民なら**510万円の対象外**。
  さらに台本自身が後半で「インド系労働移民は年間＋170万円の貢献」と言っているので、
  **優秀なエンジニアが帰国を検討する設定は台本のロジックと矛盾する。**

#### 直し方

カリムの来た経緯を「**難民として来て、努力でエンジニアになった人**」に変える。
こうすると：

1. 制度上、510万円の対象に入る（在留資格が「難民」のため）
2. 「個人の努力で這い上がっても、制度は"難民"のままで扱う」という皮肉が生まれる
3. 台本の核心メッセージ「問題は人ではなく設計にある」と完全に一致する

**修正案：**
```
カリムはシリア出身で、8年前に難民としてこの国に来た。
最初は言葉も分からず、学校の清掃員をしていた。
夜はスウェーデン語の教科書を開き、
週末は無料のプログラミング講座に通った。
3年前、ようやく僕たちのチームに入った。
今ではチームに欠かせない存在だ。
彼の書くコードは美しく、バグが少ない。
夜遅くまで残業している時も、
カリムが持ってくる妻の手作りのクッキーに何度も救われた。

でも、カリムの在留資格は今も「保護が必要な人」のままだ。
制度の上では、8年経っても「難民」なんだ。
だから、あの510万円の申請条件に当てはまる。
```

→ 「コードが美しい」「クッキーで救われた」の既存の良い描写はそのまま残せる。
→ 追加するのは「来た経緯」と「在留資格のまま」の2点だけ。

### ズレ③：支払いは「一括」ではなく「3段階の分割」

#### 制度の事実

| 段階 | タイミング | 割合 |
|------|-----------|------|
| 第1回 | 申請が承認された時（まだスウェーデンにいる） | 20% |
| 第2回 | 移住先に到着した後 | 残額の40% |
| 第3回 | 出国から15ヶ月以降＋スウェーデンの滞在許可が失効/取消後 | 残額の60% |

- スウェーデンに戻ったら**全額返還義務**
- 12ヶ月以内に出国しなければ**失効**

#### 今の台本の問題箇所

```
510万円。慈悲じゃない。
電卓を叩いてみよう。
（中略）
一方、510万円は一度きりの支払いだ。
2年分のコストで、それ以降の全てをゼロにできる。
```

→ 「一度きりの支払い」は嘘ではないが、分割払い＋返還義務の存在を無視している。
  知っている視聴者が「一括じゃないのに一度きりと言うのは誤解を招く」と指摘する可能性がある。

#### 直し方

電卓パートに1文だけ追加する。大きな変更は不要。

**修正案：**
```
510万円。慈悲じゃない。
電卓を叩いてみよう。
NIERのデータによれば、
難民1人あたりの年間の財政赤字は、数十万クローナに上る。
仮に年間20万クローナ、約300万円だとしよう。
それが5年続けば1500万円。10年で3000万円。
一方、510万円は分割で支払われる。
しかも、スウェーデンに戻ったら全額返さなきゃいけない。
それでも、トータルで見れば圧倒的に安上がりだ。
「帰ってください」は感情ではない。
引き算の結果だ。
```

→ 「分割」「返還義務」を入れることで正確性が上がる。
→ かつ「それでも安上がり」と続けることで、フックの衝撃は失わない。

## 【重大】修正②：ナレーター口調の3箇所

### 箇所A：NIER研究の導入

**今の台本：**
```
スウェーデン経済研究所、通称NIER（ニエール）が、
2025年に発表した研究がある。
タイトルは「移民の純財政貢献」。
簡単に言えば、移民が国に払った税金から
彼らが受け取った社会保障費を差し引いた、
純粋な利益の計算だ。
```

**修正案：**
```
2025年、スウェーデンの経済研究所NIERがある数字を出した。
移民が払った税金から、受け取った社会保障を引いた「差し引きの収支」だ。
これを見たとき、僕は目を疑った。
```

→ 「タイトルは〜」が論文紹介口調。エリックは友人に論文タイトルを教えない。

### 箇所B：収支プラス転換の解説

**今の台本：**
```
なぜなら、これは全移民を一つにまとめた
平均値に過ぎないからだ。
エリートエンジニアの黒字と、
長期間仕事に就けない人の赤字を、
強引に平均して薄めただけの数字。
本当に重要なのは、
その「中身」がどうなっているかだ。
どの層が支え、どの層が重荷になってしまっているのか。
その内訳を分解しない限り、
僕たちは真実を見誤る。
```

**修正案：**
```
でも、この数字に騙されちゃいけない。
これは全員をひとまとめにした「平均」だ。
年収1000万のエンジニアと、
5年間仕事に就けなかった人を足して割ったらプラスになった。
それだけの話だ。
大事なのは中身だ。
誰が支えて、誰が沈んでいるのか。
```

→ 「内訳を分解しない限り、真実を見誤る」はナレーター。エリックなら「大事なのは中身だ」で切る。

### 箇所C：NIER推計の注意書き

**今の台本：**
```
なお、このNIERの推計にはひとつ注意点がある。
医療費やインフラ費用など、
個人別に切り分けが難しいコストは
一定の仮定のもとで推計されている。
完璧な計算ではなく、
最良の推計だということは、
頭に置いておいてほしい。
それでもなお、この数字が示す傾向は、
他の研究とも一致している。
```

**修正案：**
```
1つだけ正直に言っておく。
この数字は完璧じゃない。
医療費やインフラのコストは、
きれいに1人ずつ切り分けられるものじゃないからだ。
でも、方向性は他の研究とも一致している。
完璧な数字じゃなくても、向きは間違っていない。
```

→ 「なお」「一定の仮定のもとで」「それでもなお」が論文調。エリックは「正直に言っておく」で入る。

## 【軽微】修正⑤：結末CTA直前の説教調

**今の台本：**
```
人手不足だからという短期的な
数字だけで人を呼ぶのは簡単だ。
でも、その後の設計を怠れば、
そのツケは数十年の時を経て、
膨大な清算金となって
あなたの子供や孫の世代に跳ね返ってくる。
```

**修正案：**
```
僕たちは40年前、人手が足りないからと人を呼んだ。
40年後の今、510万円を積んで帰ってくれと頼んでいる。
設計を怠った代償は、こういう形で届くんだ。
```

→ エリックが「自分たちの失敗」として語ることで説教感が消える。
