# 台本修正指示書：イギリスVAT食料品0%

**対象台本**：食料品は0%、なのに300万人がご飯を食べれない / イギリス減税の"不都合な真実"  
**ベンチマーク**：サムライ経済学（ロンドン生活崩壊 / ドイツ不都合な真実 / イタリア給料ユーロ欠陥）

---

## 修正の全体方針

**今の台本の問題を一言で言うと：**  
「VATの制度解説としては正確だが、"人間の物語"になっていない」

サムライ経済学の視聴者が最後まで見る理由は「制度の正確さ」ではなく、  
「マルコが実家を出られない切なさ」「隣人が毛布を重ねて寝る寒さ」「同僚が国を出ていく悲しさ」。  
マイケルにも同じ**体温**が必要。

---

## 修正①：マイケルに「人生」を与える【最重要】

### 今の問題

マイケルには名前しかない。年齢・職業・生活状況・具体的な悩みが一切ない。  
制度を説明するための「器」であって、「人間」になっていない。

### サムライ経済学の語り手はこう作られている

| キャラ | 年齢 | 職業 | 生活状況 | 悩み | トーン |
|--------|------|------|----------|------|--------|
| マルコ | 32歳 | 会計士 | 親と同居 | 実家を出たい | 切なさ・諦め |
| ハンス | 明示 | 会社員 | ベルリン在住 | 税で手取り半分 | 怒り |
| ジョン | 明示 | 会社員 | ロンドン郊外 | NHS3週間待ち | 冷静な失望 |

### 修正方法

マイケルに以下を設定し、冒頭の自己紹介で示す。

```
例：
「ハロー。僕はマイケル。34歳、ロンドン北部に住んでいる。
スーパーの物流センターで働いている。
毎日、何トンもの食料品を仕分けしている。
パン、卵、牛乳、果物。全部税率0%の商品だ。
でも先月、僕は生まれて初めてフードバンクに行った。
食料品を毎日運んでいる僕が、食べ物をもらいに行ったんだ。
おかしな話だろう？」
```

**ポイント**：
- 職業が「食料品に関わる仕事」だと、0%制度との皮肉が生まれる
- 「フードバンクに行った」という自己開示が、マルコの「32歳で親同居」と同じ信頼獲得の機能を持つ
- この設定は台本全体を通じて維持する（結末で回収する）

## 修正②：冒頭を「マイケルの体験」から始める

### 今の問題

> 「日本では、女性の首相が誕生して、選挙で大勝したそうだね」

これはマイケルの体験ではなく、日本の視聴者へのサービス。  
サムライ経済学の冒頭は**全て語り手の個人的体験から始まる**。

| 台本 | 冒頭 |
|------|------|
| ジョン | 「先月、医者に電話した。3週間待ちと言われた」 |
| マルコ | 「僕は32歳なのにまだ親と暮らしている」 |
| ハンス | 「25万人が国を捨てて出ていく。冗談じゃない」 |

### 修正方法

**冒頭**＝マイケルの「スーパーでの体験」or「フードバンクでの体験」から始める。  
**日本の女性首相の話**＝後半の「日本ブリッジ」セクションに移動する。

```
修正後の冒頭イメージ：

「スーパーで買い物カゴを見たことがあるかい。
パン。0%。牛乳。0%。卵。0%。ビスケット。20%。
同じカゴの中に、0%と20%が混在している。
僕はこの国で34年間暮らしてきて、
それが当たり前だと思っていた。
でも先月、フードバンクの列に並んだ時、
初めて気づいたんだ。
0%って、一体誰のための制度なんだろう、って。」
```

**ポイント**：
- 「パン。0%。牛乳。0%。」のリズムは現行台本のラストにあるが、**冒頭に持ってくる方が強い**
- フードバンクの体験＝伏線（後半で回収）
- 「誰のための制度か」＝CQの提示

## 修正③：フードバンク利用者の「1人のエピソード」を追加する

### 今の問題

台本に「救急隊員や教師がフードバンクを利用している」とあるが、  
**名前がない。台詞がない。五感がない。**

サムライ経済学の最も強い技法は「統計→1人の人物→一般化」の三段構成。

| 台本 | 統計 | 1人の人物 | 一般化 |
|------|------|-----------|--------|
| ロンドン | 200万世帯が暖房なし | 70代の隣人が毛布を重ねて寝ていた | 「珍しくない」 |
| ドイツ | 25万人が国を出ていく | 同僚がマラガに移住。送別会の台詞 | 「誰も幸せじゃない」 |
| イタリア | ギリシャGDP25%縮小 | 同僚ニコス「アテネは地獄だった」 | 「ユーロという鎖」 |

### 修正方法

「フードバンク300万個」のデータの**直後**に、1人の具体的なエピソードを入れる。  
この人物が台本の**感情的クライマックス**を担う。

```
追加するエピソード例：

「フードバンクで僕の前に並んでいた女性がいた。
名前はサラ。NHSの看護師だ。週50時間働いている。
先月初めてここに来たと言っていた。
引換券を手に取った時、手が震えたそうだ。
『まさか自分がここに来るとは思わなかった。
患者さんには栄養のある食事を取ってくださいと言っているのに、
自分の冷蔵庫は空っぽなの。おかしいでしょう？』
彼女が買うパンの税率は0%だ。
でもパンを買うお金がない。
0%というのは、そういうことなんだ。」
```

**ポイント**：
- 「看護師が患者に栄養指導→自分は食べられない」の皮肉＝ドラマティック・アイロニー
- 「手が震えた」＝五感描写（触覚）
- 直接話法の台詞がある（サムライ経済学の感情的クライマックスは常に直接話法）
- このエピソードの後に「それが、食料品の税率0%の国の現実さ」を置くと着地が強くなる

## 修正④：リング構造を作る

### 今の問題

| 項目 | 冒頭 | 結末 | 呼応しているか |
|------|------|------|--------------|
| イタリア台本 | 32歳で親同居 | 来月33歳、出たい | ✅ 完璧 |
| ロンドン台本 | NHS3週間待ち | 「あの電話の続き」 | ✅ 完璧 |
| **今の台本** | 日本の女性首相 | 「0%の社会を想像して」 | ❌ 繋がっていない |

### 修正方法

冒頭を修正②の通り「スーパー＋フードバンク」に変えた場合、  
結末を以下のように変える。

```
修正後の結末イメージ：

「今日もスーパーで買い物をした。
パン。0%。牛乳。0%。卵。0%。
数字は何も変わっていない。
でも、僕にはもう前と同じようには見えない。
0%の値札の裏側で何が起きているか、
今日、君は知ったはずだ。
減税で得をする人がいる。
その裏で、知らないうちに何かを失う人がいる。
フードバンクの列に並ぶ看護師のサラのことを、
僕はこれからもずっと忘れないと思う。」
```

**ポイント**：
- 冒頭の「スーパーのカゴ」が結末で回収される
- 冒頭の「パン。0%。牛乳。0%。」のリフレインが結末で**意味が変わって**繰り返される
- サラのエピソードが結末で回収される（二重のリング）
- 「前と同じようには見えない」＝マイケルの認識が変わったことを示す

## 修正⑤：一人称のブレを直す

### 今の問題

台本の途中でマイケルの声が突然消え、ナレーターの声になる箇所がある。

❌ ナレーター口調（消す）：
> 「日本では食料品の消費税率引き下げが盛んに議論されています。」  
> 「実際、世界には食品の税率を抑えた例がいくつもあります。」  
> 「イギリスのフードバンクを運営しているのは主にボランティア団体です。」

✅ サムライ経済学の処理方法：
> マルコ：「ここで少し経済学の話をさせてほしい。難しくないから安心して。」  
> ジョン：「ここで少し慎重に扱うべき話題に触れなければなりません。」

### 修正方法

ナレーター口調の箇所を、全てマイケルの口語に変換する。

```
修正例：

❌「イギリスのフードバンクを運営しているのは主にボランティア団体です」
✅「フードバンクがどう動いているか、少し説明させてくれ」

❌「日本では食料品の消費税率引き下げが盛んに議論されています」
✅「聞いた話では、日本でも食料品の税率を下げようって動きがあるそうだね」

❌「消費税率の税率は社会のさまざまな場所に波及します」
✅「税率をいじるってことは、社会のあちこちに波紋が広がるってことだ」
```

**ルール**：台本の全ての文を「マイケルが友達に話している口調か？」でチェックする。  
「です・ます」調とマイケルの「〜だ」「〜なんだ」調が混在してはいけない。

## 修正⑥：ゲート宣言を短く鋭くする

### 今の台本

> 「ここまでが制度の表側の話。ここから食料品0%の裏側にある問題に迫ります。」

### サムライ経済学の例

> ドイツ：「でもね、お金の問題だけならまだマシなんだ」  
> オランダ：「ここからが核心です」

### 修正後

```
「ここまでが表の話。
ここからは、誰も言いたがらない話をする。」
```

短い。鋭い。マイケルの口調。「誰も言いたがらない」でオランダ台本の「タブー感」も加わる。

---

## 修正⑦：「4兆8000億円」をスケール翻訳する

### 今の問題

> 「その減収分は年間250億ポンド。日本円で4兆8000億円にのぼります。」

数字を出しただけ。サムライ経済学は必ず**「〇〇と同じ規模」に翻訳する**。

| 台本 | 数字 | 翻訳先 |
|------|------|--------|
| オランダ | 74兆円 | 60年かけて掘った天然ガスの総収入と同じ |
| ロンドン | 26兆円 | NHSの年間予算とほぼ同じ |

### 修正後

```
「年間250億ポンド。日本円で4兆8000億円。
これがどれくらいの規模か。
イギリスの防衛費の半分に相当する。
戦闘機や空母に使う予算の半分を、
食料品の税率0%が静かに食いつぶしているんだ。」
```

※ 比較対象は正確な数字を調べて差し替えてよい。  
大事なのは「4兆8000億」を**映像が浮く別のもの**に翻訳すること。

## 修正⑧：日本ブリッジの位置を整理する

### 今の問題

日本の話が**冒頭（女性首相）**と**終盤**の2箇所にあり、散らばっている。

### サムライ経済学の日本ブリッジの型

日本の話は**終盤に1回だけ、短く**。

> ドイツ台本：「みんなの国はどうだろう」（暗黙のブリッジ）  
> ロンドン台本：直接的な日本ブリッジなし（テーマ自体が普遍的）

### 修正方法

- **冒頭の日本の話（女性首相・消費税）**→ 全カット。冒頭はマイケルの体験にする
- **終盤の日本ブリッジ**→ 今の位置のまま残すが、マイケルの口調に変える

```
修正例：

「日本でも消費税の軽減税率のことが話題になっているそうだね。
食料品の税率を下げれば暮らしが楽になる、って。
でも、僕らの国では0%だ。それでもこうなっている。
構造は同じだよ。違うのは税率の数字だけだ。」
```

これで十分。長い説明は不要。

## 修正の全体構成（Before → After）

### Before（現行）

```
① 日本の女性首相の話 ← マイケルの体験ではない
② マイケル登場（名前だけ）
③ イギリスの食料品0%の紹介
④ ジャファケーキ裁判
⑤ ゲート宣言
⑥ 金持ちが3倍得（IFSデータ）
⑦ 生活が苦しい説明（Heat or Eat）
⑧ フードバンクの説明（統計のみ、人物なし）
⑨ ポケットの比喩
⑩ 日本ブリッジ
⑪ 結末（想像してごらん）
```

### After（修正後）

```
① マイケルの体験から始める ← スーパー or フードバンクの個人体験
  「パン。0%。牛乳。0%。でも先月、僕はフードバンクに行った」
  → CQ提示：「0%って、誰のための制度なんだ？」

② マイケルの自己紹介 ← 年齢・職業・生活状況を含む
  「34歳。スーパーの物流センターで働いている」

③ イギリスのVAT制度の紹介 ← マイケルの口調で
  「僕らの国では〜」（現行の内容を活かす）

④ ジャファケーキ裁判 ← そのまま活かす（ユーモアパート）
  ★ここで笑いを取る。次のゲート宣言との落差を作る

⑤ ゲート宣言 ← 短く鋭く
  「ここからは、誰も言いたがらない話をする」

⑥ 金持ちが3倍得（IFSデータ） ← そのまま活かす
  松坂牛の例は良い。ここが「知的な衝撃」

⑦ Heat or Eat ← 五感描写を足す
  「暖房か、食事か。朝、寒さに震えながらパンに手を伸ばす」
  ★現行にある五感描写はここに集約する

⑧ 【追加】フードバンクで出会った1人のエピソード ← 感情的クライマックス
  看護師サラ（or 別の人物）の直接話法の台詞
  ★台本で最も感情が高まるポイント

⑨ ポケットの比喩＋4兆8000億円のスケール翻訳 ← 構造の解説
  「左のポケットから右のポケットへ」は良い比喩、活かす

⑩ 日本ブリッジ ← 短く、マイケルの口調で
  「日本でも同じ議論があるそうだね。構造は同じだよ」

⑪ 結末 ← リング構造で回収
  「今日もスーパーで買い物をした。パン。0%。
  でも僕にはもう前と同じようには見えない」
```
