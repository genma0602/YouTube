# 競合文字起こし

**他チャンネル（競合・参考にしたい動画）の台本の文字起こし**を置くフォルダです。  
`01_競合台本分析` の分析メモや、`06_プロンプト` の新企画プロット作成と組み合わせて使います。

## 自チャンネル用との違い

| 種類 | 置き場所の考え方 |
|------|------------------|
| 競合・他チャンネル動画の文字起こし | **このフォルダ（`08_競合文字起こし`）** |
| 自チャンネル公開動画の文字起こし | 用途が別のため、このフォルダには置かない（別フォルダで管理する） |

## おすすめファイル名例

- `[チャンネル名または略称]_[動画タイトル]_文字起こし.md`
- 競合が複数ある場合、ファイル名の先頭で区別できるとよい

取得ツールはリポジトリの `tools/文字起こし/` を参照。
