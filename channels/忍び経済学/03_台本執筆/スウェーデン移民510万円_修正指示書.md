# 台本修正指示書：スウェーデン移民510万円

**対象台本**：「帰ってください」に510万円 — 移民を"受け入れた国"が"送り返す国"に変わった日  
**現状スコア**：5点（10点中）  
**目標**：8点（チャンネルに出せるレベル）

---

## この指示書の使い方

修正箇所は **8つ** ある。  
重要度の高い順に並べてある。**上から順に直すこと。**  
各修正箇所に「今の台本」「何が問題か」「修正後イメージ」を書いている。  
修正後イメージはそのまま使ってもいいし、自分の言葉に変えてもいい。  
大事なのは「何を直すか」と「なぜ直すか」を理解すること。

## 修正①：エリックに「個人的な利害関係」を作る【最重要・致命的】

### 今の台本

エリックは510万円のニュースを聞いて「衝撃を受けた」「調べた」と言っているだけ。  
台本全体を通して、エリック自身の生活と510万円の政策が**一度も交差しない。**

```
なぜ、楽園と呼ばれたこの国が、
大金を積んでまで「帰ってほしい」と言うのか。
40年分の数字の中に、
いったい何が隠されていたのか。
そして日本はその教訓を、
もう知っているのか。
気になって調べていくうちに、
僕は自分が信じていたスウェーデンの姿が、
実は40年という長い時間をかけて、
決定的に「壊れていた」ことに気づかされました。
```

### 何が問題か

サムライ経済学の主人公は全員、テーマと**個人的な利害関係**を持っている：

| 主人公 | 個人的ステーク |
|--------|-------------|
| マルコ（イタリア） | 32歳で実家を出られない＝ユーロの構造問題が**自分の人生**を縛っている |
| エマ（オランダ） | クリスマスの家族喧嘩で何も言えなかった＝移民問題が**自分の家族**を壊した |
| ハンス（ドイツ） | 同僚がスペインに移住した＝産業空洞化が**自分の職場**から人を奪った |
| サラ（イラン） | 9歳で母が道徳警察に怒鳴られた＝抑圧が**自分の幼少期の記憶**に刻まれた |

エリックは「調べた人」であり「巻き込まれた人」ではない。  
これだと視聴者は「エリックがどうなるか」に興味を持てない。

### 修正後イメージ

以下の**どれか1つ**をエリックのバックストーリーとして追加する。

**案A：同僚が帰国補助金の対象になった**
```
先月、チームの昼食中に、同僚のカリムが静かに言った。
「エリック、俺、帰国するかもしれない」
カリムはシリア出身のエンジニアで、5年前にスウェーデンに来た。
腕のいいプログラマーで、チームに欠かせない存在だ。
「510万円もらえるらしいんだ。妻とも話した」
僕は何も言えなかった。
カリムは僕が毎日ランチを一緒に食べている友人だ。
その友人に、国が「帰ってください」と言っている。
```
→ エリックの職場から人が去る＝ドイツ台本のマラガ移住エピソードと同じ構造

**案B：エリック自身が「脆弱地域」の隣で育った**
```
正直に言うと、僕は子供の頃、
リンケビーという地域のすぐ隣で育った。
友達のアフマドは、そのリンケビーに住んでいた。
小学校では一緒にサッカーをした。
中学に上がる頃、僕たちの道は分かれた。
僕は郊外の普通の学校へ。
アフマドはリンケビーの中に残った。
最後に会ったのは18歳の時だ。
彼は働いていなかった。
同じ国で育ったのに、駅をふたつ隔てるだけで、
人生がこれほど変わるものなのか。
```
→ エリック自身の幼馴染が脆弱地域の当事者＝個人的な記憶として問題に関わる

**どちらの案でも、このエピソードは2箇所で使う：**
1. 自己紹介の直後（伏線として植える）
2. 脆弱地域の解説の直後（回収として戻る）

## 修正②：脆弱地域に「1人の人間」を入れる【最重要・致命的】

### 今の台本

```
現在、この国には65の
「脆弱地域（Utsatta områden）」が存在します。
そのうち19は、
警察すら介入が困難な「特に脆弱な地域」です。
その団地の廊下には、
スウェーデン語の声が聞こえてこない。
子供たちは学校に行っても、
教科書とは違う言語の世界に戻ってくる。
就業率：37%。一般地域は80%超。
住民の6割以上が福祉に依存しています。
駅をふたつ。まったく別の国のようだ。
```

### 何が問題か

サムライ経済学のルール4：「大きな数字を出したら1人の人間を描く」に違反。  
65地域・37%・6割という数字が並んでいるが、**名前のある人物が1人もいない。**

比較：
- オランダ台本：74兆円の後に→友人サラ（看護師、パンク寸前の病棟）
- ドイツ台本：25万人流出の後に→同僚（マラガ移住、送別会の台詞）
- イラン台本：インフレ率の後に→テヘランの商人（朝7時に仕入れ、夕方5時に売っても赤字）

### 修正後イメージ

統計の**直後**に、1人の人物エピソードを入れる。

```
就業率：37%。一般地域は80%超。
住民の6割以上が福祉に依存しています。
駅をふたつ。まったく別の国のようだ。

去年、あるドキュメンタリーで、
ソダーテリエという町に住む少年が出ていた。
17歳のオマルという名前だった。
「学校ではスウェーデン語。帰ると全部アラビア語。
宿題を手伝える人が家にいない。
バイト先もない。友達は全員同じ団地の子だ」
彼はこう言っていた。
「僕はスウェーデン人なのか、それとも違うのか。
自分でもわからない」
```

→ 「名前（オマル）・年齢（17歳）・直接話法・体験の具体性」の4点が入る。  
→ 37%という就業率の数字が「オマルの人生」として体感できるようになる。

## 修正③：510万円の「電卓の計算」を実際に見せる【最重要】

### 今の台本

```
510万円。慈悲じゃない。
一人あたり毎年生じる財政負担を垂れ流し続け、
社会の分断を深めていくより、
一度だけ510万円を払って退場してもらうほうが、
はるかに安上がりだ。
そんな、冷徹極まりない
電卓の計算から弾き出された数字です。
```

### 何が問題か

「冷徹な電卓の計算」と言いながら、**その計算を一切見せていない。**  
サムライ経済学は必ず計算過程を視聴者と一緒にたどる：

- トラスショック：月9万円 × 12ヶ月 ＝ 年100万円
- オランダ：天然ガス60年分の収入 ＝ 74兆円 ＝ 移民コスト25年分
- ドイツ：額面4500ユーロ → 手取り2700ユーロ

### 修正後イメージ

```
510万円。慈悲じゃない。
電卓を叩いてみよう。
NIERのデータによれば、
難民1人あたりの年間の財政赤字は、数十万クローナに上る。
仮に年間20万クローナ、約300万円だとしよう。
それが5年続けば1500万円。10年で3000万円。
一方、510万円は一度きりの支払いだ。
2年分のコストで、それ以降の全てをゼロにできる。
この数字が何を意味しているか、分かるだろうか。
「帰ってください」は感情ではない。
引き算の結果なんだ。
```

→ 「年間300万円 × 年数 vs 510万円の一括」を視聴者と一緒に計算する  
→ 「引き算の結果」という短文パンチで落とす

**注意**：NIERの具体的な数値がリサーチで確認できない場合は「仮に」を使ってよい。ただし推計であることを明示すること（今の台本でも「一定の仮定のもとで推計」と注記しているので、その姿勢は維持）。

## 修正④：予防線を3回に増やす【重要】

### 今の台本（予防線は1回だけ）

```
ここで、はっきり言っておきたいことがあります。
これは、誰かが怠慢だとか、
能力が低いという「人間性」の話ではありません。
ましてや、良い移民と悪い移民を差別する話でもない。
問題は、人ではなく「設計」にあります。
```

### 何が問題か

オランダ前編は予防線が**3回**。ドイツ台本も**3回**。  
移民テーマは最もセンシティブなテーマであり、予防線は1回では不十分。  
視聴者が「差別的な動画だ」と感じた瞬間に離脱する。

### 修正方法：以下の2箇所に予防線を追加する

**追加①：データ区分の説明の直前（難民 vs 労働移民の話に入る前）**

```
ここで1つだけ、先に言わせてほしい。
これから話すのは、人を区分する話ではない。
「来た理由」と「受け入れた側の設計」の話だ。
```

**追加②：日本ブリッジの直前**

```
誤解しないでほしい。
僕はスウェーデンが移民を受け入れたことを
間違いだったとは思っていない。
僕の職場が豊かなのは、彼らがいるからだ。
問題は「受け入れた後」の設計を怠ったことだ。
```

→ これで予防線が**3回**になる（データ前・データ後・日本ブリッジ前）

## 修正⑤：冒頭を圧縮する【重要】

### 今の台本（冒頭〜CQまで：22文）

```
510万円。
スウェーデン政府が、提示した金額です。
条件はたったひとつ。
「自分の国へ帰ってください」。
もしあなたが、これを自分には関係のない、
どこか遠い北欧の国のニュースだと思っているなら、
今すぐこの動画を閉じたほうがいい。
でも、もしあなたの国の未来に、
正体のわからない「ズレ」や不安を感じているなら
どうか、僕の話を最後まで聞いてくれませんか。
僕の名前はエリック、33歳。
ストックホルムのIT企業で
プロジェクトマネージャーをしている会社員です。
僕の職場は、まるで小さな国連です。
隣のデスクにはインド出身のエンジニアが座り、
ランチの時間になれば、誰かが持ってきた
シリアのスパイスの香りと、
僕が好きなコーヒーの匂いが混ざり合う。
そんな光景が、
僕にとっての「スウェーデンの日常」でした。
だからこそ、このニュースを耳にしたとき、
僕は自分の感覚が根底から揺さぶられるような
衝撃を受けたんです。
```

### 何が問題か

CQ（「なぜ楽園が帰ってほしいと言うのか」）に到達するまでに**22文もかかっている。**  
サムライ経済学の冒頭は**4〜8文**でCQに到達する。

遅くしている原因：
1. **「閉じたほうがいい」〜「聞いてくれませんか」**（5〜10行目）→ 視聴者への呼びかけが長い
2. **職場の描写**（14〜19行目）→ 雰囲気は良いが情報過多
3. **「衝撃を受けた」の言い回し**（20〜22行目）→ 冗長

### 修正後イメージ

```
510万円。
スウェーデン政府が、移民に提示した金額だ。
条件はたったひとつ。
「自分の国へ帰ってください」。

僕の名前はエリック、33歳。
ストックホルムのIT企業で働いている。
ランチの時間になれば、
シリアのスパイスの香りとコーヒーの匂いが混ざり合う。
それが僕にとっての「スウェーデンの日常」だった。
その日常を作っている人たちに、
国が「帰ってくれ」と言っている。

なぜ、楽園と呼ばれたこの国が、
大金を積んでまで人を送り返すのか。
僕と一緒に、その中身を覗き込んでみてほしい。
```

**13文でCQに到達。** カットしたもの：
- 「閉じたほうがいい」の呼びかけ → 完全カット（ベネズエラ前編の真似だが、この台本では不要）
- 「まるで小さな国連です」「インド出身のエンジニアが座り」 → 圧縮
- 「自分の感覚が根底から揺さぶられるような衝撃」 → カット（大げさ。エリックの反応は行動で示す）

**残したもの**：
- 510万円の数字フック → そのまま。これは強い
- スパイスの香りとコーヒー → 五感描写として残す（ただし短縮）
- 「日常を作っている人たちに国が帰れと言っている」 → これがエリックのステーク

## 修正⑥：「歴史パート」を大幅に圧縮する【重要】

### 今の台本（1960年代〜2015年の歴史：約600字）

```
この国が本格的に移民を受け入れ始めたのは、
1960年代のことでした。
当時のスウェーデンは、
空前の高度経済成長期。
工場も、建設現場も、インフラも、
とにかく労働力が足りていなかった。
そこでフィンランドや旧ユーゴスラビアから、
「働くための移民」を積極的に受け入れました。
彼らはこの国の繁栄を支える、
大切なパートナーでした。
夏には白夜の柔らかな光が真夜中まで窓を照らし、
どの家庭のテーブルにも
熱いコーヒーとシナモンロールが並んでいた。
スウェーデンは、豊かで、静かで、公平な国でした。
しかし、1990年代に入り、
その性格が大きく変わります。
受け入れるのは「労働力」ではなく、
内戦や戦争から逃れてきた難民へとシフトしたんです。
バルカン紛争、中東、アフリカ……。
国境を越えて助けを求める人々に、
この国は惜しみなく手を差し伸べました。
新しい住居、質の高い医療、そして無料の教育。
「困っている人を助けるのは当然だ」という空気が、
国全体を包んでいました。
そして迎えた2015年。
難民危機のピークです。
人口わずか1,000万人のこの国に、
たった1年で16万3,000人もの難民が押し寄せました。
```

### 何が問題か

1. **「白夜の柔らかな光」「シナモンロール」の五感描写は良いが、ここに置く必要がない。** 台本マニュアルのルール：「昔は良かった」パートで五感を使うのは物語型（パターンA）の技法。この台本は連鎖型（パターンC）に近く、歴史パートは「下地づくり」でしかない。長すぎると本題に入る前に飽きる。
2. **「彼らは大切なパートナーでした」「国全体を包んでいました」がナレーター調。** エリックの口調ではない。
3. **1960年代→1990年代→2015年の3段階を全部詳しく説明しすぎ。** トラスショック台本では「2022年夏のイギリス」は6文で終わっている。

### 修正後イメージ

```
少しだけ時計の針を戻させてほしい。

1960年代、スウェーデンは好景気で人手が足りなかった。
フィンランドやユーゴスラビアから労働者を呼び、
彼らはこの国の繁栄を支えるパートナーになった。

1990年代、受け入れる相手が変わった。
戦争から逃れてきた難民だ。
バルカン紛争、中東、アフリカ。
この国は惜しみなく手を差し伸べた。
住居、医療、教育。全て無料で。
「困っている人を助けるのは当然だ」。
それがスウェーデンだった。

そして2015年。
人口わずか1,000万人のこの国に、
たった1年で16万3,000人の難民が来た。
当時の首相はこう言った。
「私たちの国に、壁はない」と。
```

**カットしたもの：**
- 「白夜の柔らかな光」「シナモンロール」 → カットまたは別の場所（結末のリング）に移動
- 「空前の高度経済成長期」「とにかく労働力が足りていなかった」 → 「好景気で人手が足りなかった」に圧縮
- 「大切なパートナーでした」 → 「パートナーになった」に短縮
- 「その性格が大きく変わります」 → カット（「受け入れる相手が変わった」に統合）

**五感描写（白夜・シナモンロール）の移動先：**
→ 結末のリング構造で使う。修正⑧で指定。

## 修正⑦：日本ブリッジの5つの質問を整理する【中程度】

### 今の台本

```
日本には、「何人受け入れるか」という議論はあっても、
受け入れた後、どこに住ませ、
どう統合するかという設計図が、
どこにも見当たらないからです。
123万人の外国人を、
特定の地域に密集させない仕組みはありますか？
スウェーデンのような「脆弱地域」を作らないための、
住宅政策の議論はされていますか？
日本語教育のコストを誰が、
どの予算で、何年先まで持つのか、決まっていますか？
彼らに子供が生まれたとき、
学校や地域社会にはそれを受け止める覚悟がありますか？
```

### 何が問題か

1. **質問が4つもあり、畳み掛けすぎ。** サムライ経済学の日本ブリッジは**2〜4文**で短く触れる。ここは**8文以上**ある。
2. **「〜ありますか？」の反復が説教調になっている。** オランダ後編のエマは「私たちの経験から学んでください」と1文で伝えた。質問4連射は攻撃的に見える。
3. **エリックの口調ではなくナレーターの口調になっている。** エリックは日本の政策の専門家ではない。

### 修正後イメージ

```
日本は今、2028年度までに最大123万人の外国人を
受け入れる計画を進めていると聞いた。
僕がどうしても気になるのは、
「何人受け入れるか」の議論はあっても、
「受け入れた後どうするか」の設計図が
見えてこないということだ。
どこに住ませるのか。
言葉をどう教えるのか。
その子供たちをどう育てるのか。
スウェーデンが40年かけて学んだ教訓は、
たった一つだ。
受け入れることより、
受け入れた後のほうが100倍難しい。
```

**変更点：**
- 4つの質問を「どこに・言葉を・子供を」の**3語に圧縮**
- 「〜ありますか？」の反復をやめ、「〜のか」の疑問形に統一
- 「聞いた」でエリックの立場（外国人として聞いた情報）を明示
- 「100倍難しい」の短文パンチで教訓を叩きつける

## 修正⑧：結末のリング構造を強化する【中程度】

### 今の台本（結末部分）

```
510万円は、僕たちの国からの、未来への警告です。
同じ数字を、日本は支払う側として用意することになるのか。
それとも、今ここで設計する側に回って、
違う未来を選ぶのか。
その分かれ道は、
もう目の前にあります。
ここまで見て、あなたはどう感じましたか。
この510万円という数字、
あなたならどう受け止めますか。
ぜひ、あなたの率直な視点を
コメント欄で教えてください。
```

### 何が問題か

1. **冒頭の五感描写（スパイスの香り、コーヒー）が結末で回収されていない。** リング構造が「510万円→510万円」だけで、感情的な回帰がない。
2. **「あなたはどう感じましたか」が汎用的すぎる。** ドイツ台本の「残る？出ていく？」のような具体的な二択がない。
3. **修正①で追加した同僚/幼馴染のエピソードが結末で回収されていない。**

### 修正後イメージ

（修正①で案Aの「同僚カリム」を採用した場合）

```
510万円。
この動画の冒頭でこの数字を聞いたとき、
あなたはどう思っただろうか。
「高すぎる」と思ったかもしれない。
```
