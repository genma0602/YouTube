## この指示書の使い方

修正箇所は **7つ** あります。重要度の高い順に並べています。
**①〜⑤は今日中に対応必須**です。事実と矛盾したまま公開すると視聴者の信頼を失います。
⑥⑦は小さい修正ですが、完成度を上げます。

各修正に「今の台本」→「なぜ直すか」→「修正後」の3つを記載しています。

## 修正①：冒頭に2行追加する【🔴 今すぐ・最重要】

### 今の台本

```
2026年1月。
夜明け前だった。まだ暗かった。
同僚3人の名前が呼ばれた。
```

### なぜ直すか

今日この動画を開く視聴者は「ハメネイが死んだ」というニュースを見た直後です。
1月の話から始まると「知っている話の前置き」に感じられ、最初の5秒で離脱します。

ハメネイの死を1文目に置くことで「この動画が今見るべき動画だ」と即座に判断させられます。
同時に「なぜファルハンは1月を思い出しているのか」という疑問が生まれ、
1月の話が「フラッシュバック」として機能するようになります。

### 修正後

```
今日、ハメネイが死んだ。
私はその知らせを聞きながら、1月のことを思い出していた。

2026年1月。
夜明け前だった。まだ暗かった。
同僚3人の名前が呼ばれた。
```

## 修正②：ジュネーブ協議の末尾〜「私の工場は射程圏内」を更新【🔴 今すぐ・最重要】

### 今の台本

```
1週間以内に次の協議を開くと言った。
交渉している。攻撃も辞さないと言いながら、テーブルに座っている。
私の工場は、今この瞬間も射程圏内にある。
```

### なぜ直すか

「次の協議を1週間以内に」は実現せず、2日後に攻撃が来ました。
「交渉している」は現実と逆です。
「今この瞬間も射程圏内にある」は過去形にすべきです。

このまま残すと視聴者が「情報が古い」と判断して信頼を失います。
また「私の工場は射程圏内にあった」という事実は、攻撃が実際に来たことで
この1行の重さが格段に増します。

### 修正後

```
1週間以内に次の協議を開くと言った。
その2日後、攻撃が来た。

2026年2月28日。
米国とイスラエルによる共同作戦。
戦闘機200機。標的500箇所。
ハメネイが死んだ。
IRGC司令官が死んだ。国防相が死んだ。

私の工場は、射程圏内にあった。
2月28日の朝、空にエンジンの音が聞こえた。
私は窓の外を見ていた。
```

## 修正③：「逆だ」〜「外圧は体制を強化する」を更新【🔴 今すぐ・最重要】

### 今の台本

```
「それなら体制は崩れるのでは？」
そう思う人は多い。
逆だ。
2025年6月、イスラエルとの「12日間戦争」。
空爆の夜、テヘランの路上で何が起きたか。
反政府デモは沈静化した。
イランの国旗が掲げられた。
外から殴られると、内部の亀裂は閉じる。
アフワーズでも同じだった。
昨日まで「体制を倒せ」と言っていた同僚が、国旗を持って立っていた。
私はその光景を見て、何も言えなかった。
こういう時、人は旗の下に集まる。
パナマ侵攻の前夜もそうだった。
イラク戦争の前夜もそうだった。
外から叩かれると、国民は政府の後ろに並ぶ。
外圧は体制を強化する。
そして交渉は体制の時間を買う。
```

### なぜ直すか

「外圧は体制を強化する」と断言した直後に「ハメネイが死んだ」という事実が来るため、
論旨が矛盾します。視聴者は「さっきと言ってること違う」と感じます。

12日間戦争の例（外圧が国民を一時的に団結させた）は有効な事実として残しますが、
「強化する」という断言を「2025年6月まではそうだった」という形に変えることで、
2月28日の攻撃との対比が生まれ、むしろ「3つの防壁は崩れたか？」という
次の問いへの最高の導線になります。

### 修正後

```
「それなら体制は崩れるのでは？」
そう思う人は多い。
2025年6月まで、逆だった。

「12日間戦争」の夜、テヘランで何が起きたか。
反政府デモは沈静化した。
イランの国旗が掲げられた。
外から殴られると、内部の亀裂は閉じる。
アフワーズでも同じだった。
昨日まで「体制を倒せ」と言っていた同僚が、国旗を持って立っていた。
私はその光景を見て、何も言えなかった。

しかし2026年2月28日。規模が違った。
最高指導者が死んだ。

では、3つの防壁は崩れたか。

IRGCはまだ生きている。
「史上最も凶暴な反撃を行う」と声明を出した。
工場も、銀行も、通信会社も、まだIRGCの傘下にある。
最高指導者が変わっても、3つの防壁の構造は変わらない。
むしろIRGCが権力を握り、防壁がより強固になるかもしれない。
```

## 修正④：ホルムズ「閉じたとき」と「協議が1週間以内」を更新【🔴 今すぐ・最重要】

### 今の台本

```
ホルムズ海峡が閉じたとき、日本に何が起きるか。
日本は原油輸入の9割超を中東に依存している。
石油が届かなくなるのではない。
届くが、高くなる。
タンカー1隻あたりの保険料が跳ね上がる。
運賃が跳ね上がる。
それが製造コストに乗る。
電気代が上がる。
ガソリン代が上がる。
食料品の輸送コストが上がる。
最終的に、あなたのスーパーのレジに届く。
今この瞬間も、ジュネーブの次の協議が1週間以内に迫っている。
その結果次第で、ホルムズの緊張度は変わる。
レジの前で気づくのでは、遅い。
```

### なぜ直すか

2点あります。

1つ目：ホルムズは「閉じたとき」という仮定ではなく、2月28日にイランが閉鎖を宣言しています。
仮定形のまま残すと視聴者に「情報が古い」という印象を与えます。
逆に「閉じた」という事実にすることで、「レジに届く」という予測が
現実として迫ってきます。

2つ目：「ジュネーブの次の協議が1週間以内に迫っている」は、
攻撃が実行されたことで完全に過去の話になりました。この2文は削除します。

### 修正後

```
ホルムズ海峡が閉じた。
2026年2月28日、イランが閉鎖を宣言した。
日本は原油輸入の9割超を中東に依存している。
石油が届かなくなるのではない。
届くが、高くなる。
タンカー1隻あたりの保険料が跳ね上がる。
運賃が跳ね上がる。
それが製造コストに乗る。
電気代が上がる。
ガソリン代が上がる。
食料品の輸送コストが上がる。
最終的に、あなたのスーパーのレジに届く。
もう、遠い話じゃない。
レジの前で気づくのでは、遅い。
```

```
【変更前】
②外圧は逆効果。
攻撃は体制を強化し、壊れるのは世界経済だ。

【変更後】
```

## 修正⑤：結末「わからない」セクションを更新【🔴 今すぐ・最重要】

### 今の台本

```
この仕組みは永遠に続くのか。
正直に言えば、わからない。
歴史は意外な場所で揺らいできた。
3つの防壁のうち1つが崩れたとき、残りの2つも連鎖する可能性がある。
1979年も、誰も「今年倒れる」とは思っていなかった。
それでも、倒れた。
どこが最初に崩れるかは、誰にもわからない。
ニュースでイランのデモを聞いたとき、「大変そうだな」で流すのか。
「3つの防壁のどれかが揺れているのか」と読むのか。
「ホルムズは大丈夫か」と自分の財布に目をやるのか。
```

### なぜ直すか

「歴史は意外な場所で揺らいできた」という言葉が今まさにリアルタイムで起きています。
「揺らぐかもしれない」という未来形ではなく「今揺れている」という現在形で語ることで、
視聴者が「自分はその瞬間を目撃している」という感覚を持ちます。

ハメネイの死という事実を結末で一度正面から受け止め、
そのうえで「でも3つの防壁はまだある」という問いに着地することで、
この動画の分析が今まさに使えるレンズとして機能します。

### 修正後

```
2026年2月28日。
ハメネイが死んだ。
1979年の革命から47年間、このマニュアルを守り続けた人間が、消えた。

では、3つの防壁は崩れたか。
まだ、わからない。

IRGCが権力を握り、防壁がより強固になるのか。
それとも、今日が崩壊の始まりなのか。
歴史が動く瞬間は、こういう顔をしている。

1979年も、誰も「今年倒れる」とは思っていなかった。
それでも、倒れた。

ニュースで「イランの最高指導者が死んだ」と聞いたとき、「大変そうだな」で流すのか。
「3つの防壁のどれかが今崩れているのか」と読むのか。
「ホルムズは今どうなっているか」と自分の財布に目をやるのか。
```

## 修正⑥：「マニュアルは完成した」が2回出る【🟡 重要】

### 今の台本

```
声の穴は、完全に塞がれた。
マニュアルは完成した。        ← 1回目

3つの防壁。それぞれが独立しているように見える。
でも、根っこは同じだ。
（中略）
整理します。
軍を銀行にした。
経済を武器にした。
声を消すスイッチを作った。
マニュアルは完成した。        ← 2回目
```

### なぜ直すか

「マニュアルは完成した」はこの台本のリング構造の最大の着地点です。
1回目で使うと2回目が来ても感情が動きません。
2回目（「整理します」の末尾）だけで使うことで、
冒頭の独裁者思考実験と完全に対応した強烈なリング回収になります。

### 修正後

```
声の穴は、完全に塞がれた。
3つの防壁が揃った。           ← 「マニュアルは完成した」を削除してここに変更

（以降そのまま）

整理します。
軍を銀行にした。
経済を武器にした。
声を消すスイッチを作った。
マニュアルは完成した。        ← ここだけで使う
```
