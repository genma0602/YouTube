# 電気代EU1位の国、ドイツ｜台本修正指示書

**対象台本**：「電気代EU1位の国、ドイツ　脱原発の請求書はいくらだった？」  
**語り手**：クラウス（ミュンヘン郊外・自動車部品工場の運営担当）  
**作成日**：2026-03-01

---

## この指示書の使い方

修正箇所は **8つ** あります。重要度の高い順に並べています。  
**①②は公開前に必ず直してください。事実誤認です。**  
③〜⑥はクラウスへの感情移入を大きく左右します。  
⑦⑧は仕上げの修正です。完成度を最終的に高めます。

各修正に「今の台本」→「なぜ直すか」→「修正後イメージ」の3つを記載しています。

---

## ✅ まず残してほしいもの（削らないでください）

この台本の完成度が高い部分です。絶対に削らず、変えないでください。

| 箇所 | なぜ残すか |

| 「毎月、月末になると、僕のデスクには一通の白い封筒が置かれる」 | 台本最強の冒頭。情景が即座に浮かぶ。この1文でクラウスの世界に入れる |

## 修正①：「福島第二」→「福島第一」に修正する【🔴 今すぐ・最重要】

### 今の台本

```
2011年の東京電力福島第二原発事故だ。
```

### なぜ直すか

事実誤認です。2011年に深刻な事故が起きたのは**福島第一原発**（Fukushima Daiichi）です。  
福島第二原発（Daini）は大きな事故は発生していません。  
このまま公開すると視聴者のコメント欄が荒れ、チャンネルの信頼を損ないます。

### 修正後

```
2011年の東京電力福島第一原発事故だ。
```

## 修正②：「EC各国」「EC全体」→「EU各国」「EU全体」に修正する【🔴 今すぐ・最重要】

### 今の台本

```
2024年には電力不足のため、EC各国から大量の電力を買い集めたほどだ。
ドイツのせいでEC全体の電気料金が上がったと文句まで言われてしまった。
```

### なぜ直すか

ECはEuropean Community（欧州共同体）の略称で、1993年にEUへ改組されています。  
2024年の出来事を語る文脈で「EC」を使うと、歴史的に30年以上前の組織名になります。  
クラウスがドイツ人として語る台本ですので、現在の正式名称「EU」を使ってください。

### 修正後

```
2024年には電力不足のため、EU各国から大量の電力を買い集めたほどだ。
ドイツのせいでEU全体の電気料金が上がったと文句まで言われてしまった。
```

## 修正③：冒頭の自己紹介にクラウスの「個人的な苦しさ」を1文追加する【🟡 高優先】

### 今の台本

```
僕はクラウス。
ドイツ南部、ミュンヘン郊外の自動車部品工場の運営の仕事をしている。
ドイツではそこそこ知られた自動車部品メーカーの工場だ。
```

### なぜ直すか

現状のクラウスは「工場の管理者」として登場しており、視聴者は**企業の問題**として受け取ってしまいます。  

サムライ経済学の主人公は全員、制度のせいで**自分の生活が脅かされている人**として設計されています。

- マルコ（イタリア）：「32歳なのに親の家から出られない」→個人の生活の問題  
- ジョン（ロンドン）：「月の手取りが6万3千円しか残らない」→個人の生活の問題  
- クラウス（現状）：「工場の電気代が高い」→**企業の財務問題**

視聴者がクラウスに感情移入するためには、「電気代の高さ」がクラウス**自身の人生**を脅かしていることを、早い段階で見せる必要があります。  

また、クラウスの年齢が台本中に一度も明示されていません（1986年に3歳と書かれているので逆算すると43歳前後ですが、視聴者は計算しません）。年齢を明示することでキャラクターが一気に立体化します。

### 修正後イメージ

```
僕はクラウス。43歳。
ドイツ南部、ミュンヘン郊外の自動車部品工場の運営の仕事をしている。

正直に言う。最近、工場が閉まるかもしれないという話が出ている。
電気代が高すぎて、このままではドイツで作り続けられないというわけだ。
工場が止まれば、僕の仕事も消える。
43歳で、次にどこへ行けるのか。
毎月この封筒を開けるたびに、その番号が頭をよぎる。
```

※文体・細部はクラウスの口調に合わせて整えてください。  
ポイントは「電気代の高さ」が「工場の問題」ではなく「クラウス個人の未来」と直結していることを示すことです。

## 修正④：各請求書の間に「エスカレーション宣言」を入れる【🟡 高優先】

### 今の台本

```
"1枚目の請求書は「富」だった。"

でも、失われたのはお金だけじゃなかった。
2枚目の請求書。
それは環境だ。
```

```
請求書の2枚目は環境だった。

しかも、請求はまだ終わらない。
3枚目の請求書は命だ。
```

```
ドイツは原発事故のリスクを避けた。
その代わりに石炭の煙を選んだ。請求書の3枚目は命だった。
4枚目の請求書。それはこの国の電力システムだ。
```

### なぜ直すか

ドイツ不都合な真実の台本で最も視聴者を引き留めた技法は、  
「**でもね、お金の問題だけならまだマシなんだ**」という「エスカレーション宣言」でした。  

次の層が**前の層よりも深刻**であることを予告することで、視聴者は「え、まだあるの？」と先を読む理由を得ます。  

現状の台本は各請求書の着地と次の請求書への移行が平板で、「次に進む理由」が生まれていません。  
4枚の請求書を「単なるリスト」ではなく「エスカレーション」として見せてください。

### 修正後イメージ

**1枚目→2枚目の橋渡し：**
```
1枚目の請求書は「富」だった。
でもね、失われたのはお金だけじゃなかった。
もし富だけの請求書なら、まだ計算できる。問題は、次の請求書は計算が難しいんだ。
2枚目を開けよう。
```

**2枚目→3枚目の橋渡し：**
```
請求書の2枚目は「環境」だった。
でも正直に言う。環境の請求書より、次の3枚目の方が僕には重かった。
3枚目の請求書には「命」と書いてあった。
```

**3枚目→4枚目の橋渡し：**
```
3枚目は「命」だった。
数字を知った今、カールの咳を同じように聞けない。
ただ、請求はまだ終わっていない。4枚目だ。
```

## 修正⑤：「カール」のエピソードに直接話法と身体感覚を加える【🟡 高優先】

### 今の台本

```
私の家の近所に、カールという男が住んでいる。
長くミュンヘンに住んでいる。
最近、ぜんそくがひどいとこぼしていた。
子供の頃、ぜんそくがひどかった。
あれは、森を枯らした大気汚染のせいだと思う。
環境対策が進んだ後は収まっていたのに、最近、息が苦しいことが多くなった。
また、空気が汚れて来たんじゃないのかい。
私は、きれいな空気を吸い、安心な水を飲んで生活したいだけなのに。
直接の証明はできない。
でもこの数字を知った後では、同じようには聞けない。
```

### なぜ直すか

1万9200人という大きな数字の直後に「人間の顔」を見せる技法は正しいのですが、  
現状のカールのエピソードは**3つの要素が不足**しています。

サムライ経済学のルールでは、数字の後に出す人物には以下が必要です：
- ✅ 名前（カール）→ ある
- ❌ **本人の直接の台詞**→ ない
- ❌ **身体感覚が伝わる描写**→ 「息が苦しい」は告白ではなくクラウスの要約

さらに「直接の証明はできない」という留保が感情の勢いを止めています。  
因果関係を断定する必要はありませんが、留保を前に置くか、軽くするかで印象が大きく変わります。

### 修正後イメージ

```
私の家の近所に、カールという男が住んでいる。
65歳。長くミュンヘンに住んでいる。

先週、道で会った。
「最近また、夜中に目が覚めるんだよ」と彼は言った。
右手を胸に当てながら。「子供のころみたいに、息が詰まる感じがしてな」と。

彼はチェルノブイリの後、喘息がひどかった。
環境対策が進んで、一度は治まった。
なのに今また、夜中に起き上がっている。

直接の証明はできない。
でもこの1万9200人という数字を知った今、あの話を同じようには聞けなかった。
```

ポイント：
- カールの年齢を足すと人物が立体化します
- 「言った」という直接話法を入れることで、カールが「生きている人間」になります
- 「右手を胸に当てながら」という動作描写で身体感覚が伝わります
- 「直接の証明はできない」は残してよいですが、エピソードの**後**に置いてください（前に置くと感情が始まる前に止まります）

## 修正⑥：クラウスが「原発に賛成していた自分」への葛藤を語る箇所を強化する【🟡 高優先】

### 今の台本

```
僕だって、そうだ。
福島の惨状を見て、やはり原発には頼れないと思った。
だから今日の話は、誰かを責める話じゃない。請求書の中身を確認する話だ
```

### なぜ直すか

この箇所はすでに台本の誠実さを守る重要な役割を果たしています。残してください。  

ただし、1文加えることでクラウスが**自己矛盾を抱えた人間**として格段にリアルになります。  
視聴者が最も感情移入しやすいのは「正しいことをしようとしたが、その結果に苦しんでいる人」です。  

現状は「請求書を確認する客観的な語り手」ですが、「原発を止めることに賛成した当事者」として語ることで、クラウスの問いが「誰かへの批判」ではなく「自分への問い」になります。

### 修正後イメージ

```
僕だって、そうだ。
福島の惨状を見て、やはり原発には頼れないと思った。
正直に言えば、工場で働き始めた後も、止めるべきだと思っていた一人だ。

だから、この請求書を開けるのが怖かった。
誰かを責めたいわけじゃない。
ただ、正しいことをしようとした結果が何だったのか、
自分の目で確認したかった。

請求書を開けよう。1枚目。
```

## 修正⑦：日本セクションをクラウスの一人称に収まる量に絞る【🟢 中優先】

### 今の台本（日本セクション）

```
日本も2011年に、すべての原発を止めた。
福島の事故を目にして、身近に恐怖が迫っていることを感じたに違いない。
チェルノブイリの事故の後のドイツと同じように。
でも、日本は原発を再稼働させる道を選んだ。
万全の事故対策を尽くすとの決意とともに。
日本が再稼働した原発は既に14基。
2040年には電力の20%を原子力でまかなう計画だ。
ドイツの請求書を横目で見ながら、別の道を選んだということかもしれない。
```

### なぜ直すか

日本の原子力政策の詳細（14基・2040年・20%）を、ミュンヘンの工場運営者クラウスが正確に把握しているのは不自然です。これがクラウスの「一人称の語り」から「第三者のナレーション」に滑っている瞬間です。

また、量が多いと「日本の話が始まった」と感じられ、ドイツの物語の余韻が壊れます。  
サムライ経済学のルールでは日本ブリッジは「2〜4文・短く・押しつけない」です。

### 修正後イメージ

```
ドイツの請求書を、隣で見ていた国がある。
日本だ。
2011年に一度は全ての原発を止めた。
でも今、再稼働に踏み出している。
ドイツとは別の答えを選んだ、ということだ。
どちらが正しいのか、僕にはわからない。
ただ、同じ問いを抱えている人たちがいることは確かだ。
```

## 修正⑧：結末の「核心フレーズ」を1行追加する【🟢 中優先】

### 今の台本（結末）

```
幼い僕を放射線から守ろうと必死だった両親。
大気汚染でせき込むカール。
2人の姿と、高すぎる電気料金の請求書。
どれも事実だ。
それだけに、僕には何が正しい道なのか、いまだに答えは出ない。
```

### なぜ直すか

結末の情感は正しい方向で、「答えが出ない」という誠実な着地も台本のテーマに合っています。  
ただし、サムライ経済学の最高作品には必ず「動画を見た後に頭に残る1行」があります。

- ドイツ不都合な真実：「普通に働いて普通に暮らす。たったそれだけの夢が、この国ではあまりにも高価になりすぎた」
- イタリア：「来月33歳になる。そろそろ実家を出たいと思っている。でも見当もつかない」
- イラン：「革命の子供たちが、次の革命を不可能にした」

現状の「何が正しい道なのか、いまだに答えは出ない」は誠実ですが、SNSで引用されたり、見終わった後に頭に残る「核心フレーズ」としてはやや弱い。

**修正⑥で追加した「正しいことをしようとした結果が何だったのか」の問いを結末で回収**することで、リング構造が完成します。

### 修正後イメージ

```
幼い僕を放射線から守ろうと必死だった両親。
大気汚染で夜中に目を覚ますカール。
4枚の請求書。

どれも、正しいことをしようとした人たちが払った代金だ。

それだけに、僕には何が正しい道なのか、いまだに答えは出ない。
```

「正しいことをしようとした人たちが払った代金だ」の1行を入れることで：
- 台本全体のテーマが1行に凝縮されます
- 両親・カール・クラウス自身・ドイツ国民、全員が「悪人ではなかった」という台本の誠実さが確定します
- 視聴者が「自分の国ではどうだろう」と考えるブリッジになります
