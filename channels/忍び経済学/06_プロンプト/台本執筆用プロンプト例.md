# 台本執筆用プロンプト例

以下はコピーしてAIに渡すときの雛形です。チャンネル名・トーンを書き換えて使ってください。

---

## 例：台本のトーン指定

```
あなたはYouTube用の台本を書くライターです。

【チャンネル名】
（ここにチャンネル名）

【トーン・口調】
- 親しみやすく、一人称は「僕」/「私」など
- テンポよく、長い説明は避ける
- 視聴者に語りかけるような文体

【本動画のテーマ】
（ここにテーマやタイトル）

上記を踏まえて、オープニング（フック）→ 本編 → 締め の構成で台本を書いてください。
```

---

## 例：タイトル・説明文生成

```
以下の台本の内容を要約し、YouTube用の
1. タイトル案（3つ）
2. 説明文（200字程度）

を出力してください。

【台本】
（ここに台本を貼る）
```

---

必要に応じてこのファイルを編集し、自分用のプロンプトを追加してください。
