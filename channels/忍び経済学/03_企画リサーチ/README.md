# 企画リサーチ

動画企画のテーマについて調査した結果・データ・引用元をMD形式で格納するフォルダです。台本執筆前のリサーチや、数字・事実の裏付けとして利用します。

企画メモ・複数案の整理・企画一覧の深掘りメモもここに置きます。**構成のプロット**（`新企画_*_プロット.md`）は `04_プロット` に分けました。

**工程順**: ここ（調査）→ `04_プロット`（構成）→ `05_台本執筆`（台本・修正）。チャンネル直下の `README.md` に全体の流れを記載しています。

## おすすめファイル名例

- `[テーマ名]_経済・環境影響.md`
- `[国名]_[政策名]_リサーチ.md`
- `[キーワード]_調査結果.md`

## 書いておくと便利なこと

- 出典・参考リンク（URL・報告書名・発表年）
- 定量的データ（数値・比較表）
- 時系列や各国比較がわかる構成
