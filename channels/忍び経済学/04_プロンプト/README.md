# プロンプト（チャンネル用）

このチャンネル専用のAI用プロンプトを保存するフォルダです。

## 共通プロンプト（ルートの prompts-shared/）

- **台本分析**：伸びる理由・感情曲線・構成の分析
- **新企画プロット作成**：競合の分析結果＋文字起こしから、自チャン用の新企画の情報収集リストとプロットを作成

→ 新企画を立てるときは `prompts-shared/新企画プロット作成_競合分析ベース.md` を参照。

## このフォルダに置く例

- 台本のトーン・口調を指定するプロンプト
- このチャンネル用のタイトル・説明文生成プロンプト
- サムネ用の文言生成プロンプト
