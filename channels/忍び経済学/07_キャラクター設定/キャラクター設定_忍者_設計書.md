# キャラクター設定設計書：忍者（Ninja）キャラクター

**対象チャンネル**: サムライ経済学  
**用途**: 動画編集者向け・ビジュアル制作の指針  
**最終更新**: 2026-02-14  

---

## 1. 概要と目的

### 1.1 本ドキュメントの目的

本設計書は、サムライ経済学チャンネルの動画に登場する**忍者風オリジナルキャラクター**のビジュアル表現を統一するための指針です。動画編集者・イラストレーターが、チャンネル既存のスタイルを踏襲しつつ、このキャラクターを一貫して描く際の基準としてご利用ください。

### 1.2 キャラクターの役割

| 項目 | 内容 |
|------|------|
| **名前** | 忍者（ニンジャ）※名称は動画内で固定しなくても可 |
| **主な役割** | 経済データや「極秘情報」を解説する語り手／視覚的なアイコン |
| **登場シーン** | データ分析パート、裏側の構造解説、数字の深掘り、チャンネル紹介 |
| **トーン** | 親しみやすく、知的で、秘密を明かすワクワク感 |

## 2. サムライ経済学の既存スタイル（模倣の基準）

### 2.1 チャンネル全体のビジュアル特徴

| 項目 | サムライ経済学の特徴 | 忍者キャラへの適用 |
|------|---------------------|-------------------|
| **画風** | フラットでクリーンなアニメーション風 | 同様にフラット、過度な立体表現は避ける |
| **体型** | ちびキャラ（頭大・体小） | 同じプロポーションを維持 |
| **顔の描き方** | シンプル：目は点または大きく、口は最小限 | 目と口で感情を表現。詳細は控えめ |
| **色彩** | 落ち着いたトーン、一部にアクセントカラー | 黒×赤を基調にしつつ、背景とのコントラストを確保 |
| **背景** | 具体的な場面（スーパー、欧州街並み、図書館など） | 日本的な要素（雲、桜）＋経済要素（グラフ、データ） |

| 忍者（本キャラ） | 黒装束、赤マフラー、巻物 | 明るい・自信・発見 | 極秘データの提示者 |

## 3. 忍者キャラクターの基本スペック

### 3.1 体型とプロポーション

| 項目 | 指定 |
|------|------|
| **体型** | ちびキャラ（SD）／頭身比 約 2.5〜3 頭身 |
| **頭** | 丸く、やや大きめ。シンプルな輪郭 |
| **手足** | 短く、丸みを帯びる。指は省略または簡略化可 |
| **全体的印象** | 親しみやすく、動きやすそうなシルエット |

### 3.2 服装

| 部位 | 指定 | 補足 |
|------|------|------|
| **忍装束** | 黒色のシンプルな装束 | 袖・足元に布のシワ・結び目で動きを表現 |
| **フード** | 黒、頭上部を覆う | 目元から上を隠す。神秘的さと知性の両立 |
| **マフラー／帯** | 鮮やかな赤（#C41E3A 程度） | 左右に広がってなびく。動きのアクセント |
| **装飾** | 最小限 | チャンネルの「サムライ」要素はロゴで表現 |

### 3.3 顔のデザイン

| 部位 | 基本仕様 | 表情別バリエーション |
|------|---------|---------------------|
| **肌色** | 明るいベージュ／#FFF8E7 系 | — |
| **目** | 黒い点または簡略化した形 | 笑い：へ字型／驚き：丸／真剣：細め |
| **口** | 小さく、シンプル | 笑い：大きく開く／納得：横線／困惑：逆V |
| **頬** | 薄いピンクのチーク | 喜び・発見のシーンで強調可 |
| **鼻・耳** | 省略または極小 | サムライ経済学の既存スタイルに合わせる |

### 3.4 小道具

| 小道具 | 用途 | デザイン |
|--------|------|---------|
| **巻物** | 極秘データ・分析結果の提示 | クリーム色の和紙、木製の軸。内容はグラフ、¥・$、テキスト |
| **巻物の文言** | シーンに応じて変更可 | 例：「極秘データ」「74兆円の内訳」「金利差の正体」 |
| **その他** | 必要に応じて | 虫眼鏡、本、グラフ用紙など既存チャンネルの小道具と統一 |

## 4. 表情セット（必須パターン）

動画編集では、以下の表情を最低限用意することを推奨します。

| 表情名 | 目 | 口 | 眉 | 使用シーン例 |
|--------|----|----|-----|-------------|
| **標準（笑顔）** | へ字型、細め | 大きく開いたV字 | 自然 | オープニング、発見の報告 |
| **真剣** | 細い線または点 | 横線 | やや下げる | データ解説、構造説明 |
| **驚き** | 丸く大きく | 小さく開く | 上がる | 衝撃的な数字の提示 |
| **納得** | 半円または点 | 穏やかな弧 | 自然 | 結論・着地 |
| **困惑** | 点＋汗 | 逆V | 中央寄り | 複雑な状況の説明 |
| **ウインク** | 片目閉じ | 笑み | 自然 | 視聴者への呼びかけ |

## 5. ポーズとジェスチャー

### 5.1 基本ポーズ

| ポーズ名 | 説明 | 使用場面 |
|---------|------|---------|
| **巻物を広げる** | 両手で巻物を掲げ、内容を見せる | データ・分析結果の提示 |
| **指差し** | 巻物やグラフを指す | 重要な数字・箇所の強調 |
| **腕組み** | 胸の前で腕を組む | 解説のまとめ、結論 |
| **歩行** | 軽やかに歩く | シーン転換、導入 |
| **正座／しゃがみ** | 巻物を膝の上に | 落ち着いた解説、締め |

### 5.2 サムライ経済学スタイルでの注意点

- 既存キャラは「手を広げて困惑」「拳を突き上げる」など感情に合わせたジェスチャーを使う
- 忍者は「データ提示」「発見」「解説」寄りなので、過度にダイナミックよりは**知的で機敏**な動きを優先
- 赤いマフラー／帯のなびきで動きを補強

## 6. 背景・シーンとの関係

### 6.1 背景パターン

| シーン | 背景例 | 忍者の配置 |
|--------|--------|------------|
| **データ解説** | グラフ、数字、巻物の拡大 | 中央または左右。巻物が主役になるよう配置 |
| **日本的情緒** | 雲、桜、和風模様 | 前景に忍者、背景はシンプルに |
| **経済シーン** | スーパー、オフィス、街並み | 既存キャラ（佐藤さん等）と並置する場合はサイズ・トーンを統一 |
| **抽象表現** | シーソー、矢印、フローチャート | 既存動画の視覚メタファーと同系統で |

### 6.2 既存チャンネルとの統一

- **吹き出し**: 白いギザギザ吹き出しに短い文言（例：「消費税0%」「極秘データ」）
- **字幕**: 画面下部の黒帯に白文字
- **チャンネルロゴ**: 右下に赤×兜デザインを配置
- **配色**: キャラ・背景ともに彩度を抑え、アクセントに赤や黄を使用

## 8. アニメーション・エフェクトの指針

| 項目 | 推奨 |
|------|------|
| **登場** | 煙やキノコ雲とともに現れる、巻物を広げる動作 |
| **データ提示** | 巻物からグラフや数字が浮き上がる、拡大表示 |
| **強調** | 赤いマフラーがなびく、目がキラッと光る |
| **退場** | 煙とともに消える、あるいはそのままフェードアウト |
| **トランジション** | 巻物がパラパラと捲れる、ページめくり風 |

---

## 9. 禁止事項・注意点

| 項目 | 内容 |
|------|------|
| **キャラの役割** | 政治的主張や差別的表現には使わない。あくまで「解説者」「データの提示者」 |
| **過度な萌え表現** | 既存チャンネルのトーンに合わせ、控えめに |
| **他キャラとの整合** | 佐藤さん、エマ、ハンス等と並べる場合は、頭身・線の太さ・色のトーンを統一 |
| **フォント** | 吹き出しや巻物の文字は、既存動画で使っているフォントに合わせる |
