# 第六部：チェックリスト・禁止事項

## 20. 納品前チェック

- [ ] 国別髪色が指定通りか
- [ ] シーンと表情の対応が台本と合っているか
- [ ] 背景の建物・色調が国ごとの仕様と一致しているか

- [ ] 忍者と他キャラの頭身・線の太さが統一されているか
- [ ] チャンネルロゴ（赤×兜）が右下に配置されているか
