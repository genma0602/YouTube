# 第一部：キャラクター設定（国別・詳細仕様）

## 1. アートスタイル基本原則

| 項目 | 仕様 |
|------|------|
| スタイル | フラット、ミニマル、カートゥーン。ちびキャラ（2〜3頭身） |
| 線画 | シンプルで明確なアウトライン |
| 色彩 | パステル〜落ち着いたトーン。重要要素はコントラストで強調 |
| 表情 | 目・眉・口で感情を表現。汗・涙・怒りマーク等の記号も使用可 |
| 髪色 | 国・役割で色分け（下記国別仕様参照） |

# 第二部：編集の抽象原則（国を問わず適用）

## 3. 感情ビート→表情の選び方

**原則**：台本の「感情の向き」に合わせて表情を選ぶ。国別キャラの表情パターン（上記）から、最も近いものを当てる。

| 感情のビート | 表情の方向 | 目・眉・口の傾向 |
|-------------|-----------|-----------------|
| 日常・導入 | ぼんやり、自然 | 点、自然、横線 |
| 疑問・違和感 | 思案、困惑 | 点、ハの字、下がる |
| 衝撃・数字の提示 | 驚き、衝撃 | 丸く、釣り上がり、O型 |
| 構造解説・説明 | 真剣、指差し | 集中、自然、閉じる |
| 苦しさ・共感 | 苦しみ、共感 | 下向き、下げる、横線 |
| 怒り・告発 | 怒り、皮肉 | 鋭い、強く下げる、固く結ぶ |
| 絶望・悲しみ | 絶望、涙 | 虚ろ、涙、下がる |
| 決意・抵抗 | 決意、解放 | 鋭い、上がる、拳 |
| 結論・提案 | 穏やか、納得 | 穏やか、自然、弧 |

**使い方**：台本を読み、その瞬間の「感情ビート」を上表で特定し、該当する国キャラの表情を適用する。

---

## 4. 経済・お金の話→画像の抽象原則

**原則**：マクロの数字は「1人の人間の1日・1ヶ月」に翻訳して可視化する。

| 話題の種類 | 画像の方向性 | 例（考え方） |
|-----------|-------------|-------------|
| 賃金・給料 | 明細、額面vs手取り、残高 | 「〇〇円→△△円」の2段表示、引き算 |
| 物価・インフレ | 買い物、値札、日常品 | スーパー、棚、カゴ、レシート |
| 通貨・為替 | 2つの通貨の対比、換算 | A通貨→B通貨の矢印、秤 |
| 金利・借金 | 返済の内訳、時間経過 | 償還表、利息が膨らむ図、段階変化 |
| 家賃・住宅 | 需要と供給の不均衡 | シーソー、天秤、人の数vs家の数 |
| 資産・投資 | 持っている人vs持っていない人 | 2人の対比、スマホのアプリ |
| コスト・負担 | 大きい数字のスケール感 | 他との比較（予算、年間収入等） |

**背景**：該当国のキャラが登場するシーンでは、第一部の国別背景仕様を参照。解説パートはシンプルな図解でOK。

## 5. 政治・国際の話→画像の抽象原則

**原則**：抽象的な権力・体制は「建物・象徴・人々の動き」で具体化する。

| 話題の種類 | 画像の方向性 | 例（考え方） |
|-----------|-------------|-------------|
| 政府・政策 | 議会、中央銀行、象徴的建物 | 国会、日銀、その国の議事堂 |
| 選挙・政権 | 投票、政権交代、支持率 | 投票箱、旗、下がるグラフ |
| 制裁・紛争 | 封鎖、孤立、境界 | ×マーク、封鎖された港、地図 |
| 戦争・軍事 | 破壊、瓦礫、兵器 | 爆発、瓦礫、戦闘機シルエット |
| 革命・デモ | 群衆、拳、スローガン | 拳を突き上げる人々、閉じたシャッター |
| 弾圧・権力 | リスト、追放、沈黙 | 名簿、解雇通知、凍りつく場面 |
| 国際関係 | 国と国の矢印、利害 | 地図、矢印、資源・お金の流れ |

**背景**：該当国の象徴的建物・街並みを、第一部の国別背景仕様を参考に描く。新しい国は「その地域で一般的な建築・風景」をデフォルメして使用。

# 第三部：編集フロー（どの国でも共通）

## 8. シーン作成の手順

```
1. 台本を読み、そのブロックの「感情ビート」を特定する
   → 疑問？ 衝撃？ 解説？ 怒り？ 絶望？ 結論？

2. 登場キャラを決める
   → 国が指定されていれば第一部の該当キャラ
   → 解説パートなら日本キャラ or 忍者

3. 表情を選ぶ
   → 第二部「感情ビート→表情」の表を参照
   → その国キャラの表情マトリクスに当てはめる

4. 画像の種類を決める
   → 経済の話 → 第二部「経済・お金」の原則
   → 政治の話 → 第二部「政治・国際」の原則
   → 日常・導入 → 買い物、街、テレビなど生活シーン

5. 背景を決める
   → 第二部「背景・建物」＋第一部の国別背景仕様
   → その国の象徴的要素を1〜2個入れる

6. 共通要素を追加
   → ロゴ右下、吹き出し・字幕のスタイル
```

---

## 9. 新しい国を追加するとき

1. **キャラ**：髪色・服装・肌色をその国らしく設定。基本はちびキャラ＋丸み。
2. **背景**：その国で象徴的な建物・風景を1〜2個選び、デフォルメして描く。
3. **表情**：第二部の感情ビート表をそのまま使う。国ごとの固有表情が必要なら追記。
4. **画像**：第二部の経済・政治の抽象原則に従い、その国の具体例に置き換える。

---

**以上**

本マニュアルの第一部（キャラ設定）と第二部（抽象原則）を参照すれば、どの国のシーンでも編集・画像生成が可能です。
