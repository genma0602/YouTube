# 競合文字起こし（スコープ）

このフォルダに格納するのは、競合チャンネル **心理の栞** の動画台本の文字起こしのみとする。他チャンネルの台本はここに置かない。

## 再生数によるパフォーマンス区分（格納時点）

分析・メモ用の定義。いずれも **その時点の再生回数** で見る。

| 区分 | 条件（言い回し） |
|------|------------------|
| **伸びていない** | 1万再生 **以下** |
| **通常** | 1万再生 **以上** かつ 5万再生 **以内** |
| **伸びている** | 5万再生 **以上** |

※「以下」と「以上」が **ちょうど1万回・5万回** で重なるため、数値で照合するときは次の三分割にする：

- **～9,999回** → 伸びていない  
- **10,000～49,999回** → 通常  
- **50,000回～** → 伸びている
