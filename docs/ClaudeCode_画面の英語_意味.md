# Claude Code の画面でよく出る英語 → 日本語

## よく見る単語・フレーズ

| 英語 | 意味 |
|------|------|
| **Send** | 送信（メッセージを送る） |
| **New chat** / **New conversation** | 新しい会話を始める |
| **Past conversations** | 過去の会話 |
| **Open in New Tab** | 新しいタブで開く |
| **Attach** | ファイルなどを添付する |
| **@** (メンション) | ファイルやフォルダを参照して会話に含める |
| **Accept** | 承諾する（Claudeの提案をそのまま採用） |
| **Reject** | 却下する |
| **Apply** | 適用する（変更をコードに反映） |
| **Plan** | 計画（何をするか先に説明してから実行） |
| **Auto-accept** | 自動で承諾（いちいち確認せず変更を入れる） |
| **Permission** | 許可（ツール実行などの許可設定） |
| **Settings** / **Configure** | 設定 |
| **Sign in** | ログイン |
| **Usage** | 使用量（APIの利用状況など） |
| **Model** | モデル（使うAIの種類） |
| **Extended thinking** | 長時間の推論モード |
| **Compact** | コンテキストを圧縮する（会話を短くまとめる） |
| **/plugins** | プラグイン管理 |
| **/config** | 設定を開く |
| **/usage** | 使用量を表示 |

## メッセージ例

- **"Sign in to get started"**  
  → 始めるにはログインしてください

- **"Ask me anything"** / **"What would you like to do?"**  
  → 何でも聞いてください / 何をしたいですか？

- **"I'll need your permission to..."**  
  → …するにはあなたの許可が必要です

- **"Accept these changes?"**  
  → この変更を採用しますか？

- **"Review plan"**  
  → 計画を確認する

## Cursor を日本語にする（おすすめ）

Cursor 自体を日本語にすると、メニューやボタンがだいぶ分かりやすくなります。

1. **Ctrl+Shift+X** で拡張機能を開く
2. **「Japanese Language Pack」** で検索 → **Japanese Language Pack for Visual Studio Code** をインストール
3. **Ctrl+Shift+P** でコマンドパレットを開く
4. **「Configure Display Language」** と入力して選択
5. **「日本語 (ja)」** を選ぶ
6. **Restart**（再起動）を押して Cursor を再起動

※Claude Code の**チャットの中身**（AIの返答）は、プロンプトで「日本語で答えて」と書けば日本語になります。
